🎓 AI-РЕПЕТИТОР MVP 1.0
═════════════════════════════════════════════════════════════

Полнофункциональная система обучения на основе RAG (Retrieval-Augmented Generation)
с поддержкой математики, физики, химии и программирования.

═════════════════════════════════════════════════════════════

🚀 БЫСТРЫЙ СТАРТ (30 СЕКУНД)

1. Откройте терминал в этой папке
2. Выполните:
   
   python quickstart.py

3. Откройте браузер на http://localhost:5000
4. Начните задавать вопросы! 🎉

⚙️ ОСНОВНЫЕ КОМПОНЕНТЫ

Фреймворк: Flask 2.3 (Python веб-фреймворк)
ML: PyTorch 2.0 + Transformers 4.30
Embeddings: Sentence Transformers
LLM: GPT-2 (можно менять)
Frontend: HTML5 + CSS3 + Vanilla JavaScript

➕ ДОБАВЛЕНИЕ СВОИХ МАТЕРИАЛОВ

1. Создайте file.txt в папке data/documents/
2. Добавьте содержимое:
   
   НАЗВАНИЕ ПРЕДМЕТА - ТЕМА
   
   1. РАЗДЕЛ 1
   1.1 Подраздел
   Объяснение...

3. Перезагрузите: python run.py
4. Готово! Система автоматически индексирует

🌐 API ENDPOINTS

GET  /                   # Веб-интерфейс
GET  /api/status         # Статус системы
POST /api/ask            # Отправить вопрос
GET  /api/history        # История
GET  /api/session-info   # Информация о сессии
POST /api/clear-history  # Очистить историю
