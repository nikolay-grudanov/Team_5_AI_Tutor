✅ RAG-система (Retrieval-Augmented Generation)
   └─ Семантический поиск по документам
   └─ Генерация ответов на основе контекста

✅ Веб-интерфейс (HTML/CSS/JavaScript)
   └─ Чат с AI-репетитором
   └─ История вопросов
   └─ Информация об источниках

✅ REST API (Flask)
   └─ 6 основных эндпоинтов
   └─ JSON-based коммуникация

✅ ML-компоненты
   └─ Embeddings (Sentence Transformers)
   └─ LLM (GPT-2 + transformer models)
   └─ Векторный поиск (cosine similarity)

✅ Управление данными
   └─ Загрузка документов (.txt, .md)
   └─ Обработка и разбиение на части
   └─ Кэширование векторов
