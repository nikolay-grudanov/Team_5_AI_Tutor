ПРОГРАММИРОВАНИЕ - ОСНОВНЫЕ КОНЦЕПЦИИ

1. ВВЕДЕНИЕ В ПРОГРАММИРОВАНИЕ

1.1 Что такое программирование?
Программирование - это создание инструкций для компьютера в виде последовательности команд.

Основные компоненты программы:
- Переменные: хранилища для данных
- Операции: действия с данными
- Условия: принятие решений
- Циклы: повторение действий
- Функции: многократно используемые блоки кода

1.2 Языки программирования
Высокоуровневые языки (ближе к человеческому языку):
- Python: простой, универсальный
- Java: объектно-ориентированный
- C++: быстрый, мощный
- JavaScript: язык веб-браузеров

Низкоуровневые языки (ближе к машинному коду):
- Ассемблер: инструкции процессора
- Машинный код: бинарные команды

2. ОСНОВНЫЕ ТИПЫ ДАННЫХ

2.1 Числа
Целые числа (integers): -5, 0, 42
Действительные числа (floats): 3.14, -2.5, 0.001

Операции:
- Сложение: 5 + 3 = 8
- Вычитание: 10 - 4 = 6
- Умножение: 6 * 7 = 42
- Деление: 20 / 4 = 5
- Целочисленное деление: 20 // 3 = 6
- Остаток от деления: 20 % 3 = 2
- Возведение в степень: 2 ** 3 = 8

2.2 Строки (текст)
Строка - последовательность символов.
Примеры: "Hello", 'World', "123"

Операции со строками:
- Конкатенация: "Hello" + " " + "World" = "Hello World"
- Повтор: "a" * 3 = "aaa"
- Длина: len("Hello") = 5
- Индексирование: "Hello"[0] = 'H', "Hello"[-1] = 'o'
- Срезы: "Hello"[1:4] = "ell"

2.3 Логические значения (Boolean)
True (истина) и False (ложь)

Операции:
- And (И): True and True = True, True and False = False
- Or (ИЛИ): True or False = True, False or False = False
- Not (НЕ): not True = False, not False = True

2.4 Списки (массивы)
Упорядоченная коллекция элементов.
Пример: [1, 2, 3, 4, 5]

Операции со списками:
- Длина: len([1, 2, 3]) = 3
- Индексирование: [1, 2, 3][0] = 1
- Слайс: [1, 2, 3, 4][1:3] = [2, 3]
- Добавление: [1, 2].append(3) → [1, 2, 3]
- Удаление: [1, 2, 3].remove(2) → [1, 3]

2.5 Словари (хеш-таблицы)
Коллекция пар ключ-значение.
Пример: {"name": "John", "age": 25}

Доступ: dict["name"] = "John"

3. УСЛОВНЫЕ ОПЕРАТОРЫ

3.1 Условие if-else
if (условие):
    # выполнить, если условие истинно
else:
    # выполнить, если условие ложно

Пример:
if x > 0:
    print("x положительное")
elif x < 0:
    print("x отрицательное")
else:
    print("x равно нулю")

3.2 Операторы сравнения
- Равно: ==
- Не равно: !=
- Больше: >
- Меньше: <
- Больше или равно: >=
- Меньше или равно: <=

4. ЦИКЛЫ

4.1 Цикл while
while (условие):
    # выполнять, пока условие истинно

Пример:
i = 0
while i < 5:
    print(i)
    i = i + 1

4.2 Цикл for
for переменная in коллекция:
    # выполнить для каждого элемента

Пример:
for i in [1, 2, 3, 4, 5]:
    print(i)

for i in range(5):  # 0, 1, 2, 3, 4
    print(i)

4.3 Управление циклом
break: выход из цикла
continue: переход к следующей итерации

5. ФУНКЦИИ

5.1 Определение функции
def имя_функции(параметры):
    # тело функции
    return результат

Пример:
def add(a, b):
    return a + b

result = add(3, 5)  # result = 8

5.2 Параметры функции
Позиционные параметры:
def greet(first, last):
    return f"Hello {first} {last}"

greet("John", "Doe")  # "Hello John Doe"

Параметры по умолчанию:
def greet(name="Guest"):
    return f"Hello {name}"

greet()  # "Hello Guest"
greet("Alice")  # "Hello Alice"

Переменное количество параметров:
def sum_all(*args):
    return sum(args)

sum_all(1, 2, 3)  # 6

5.3 Область видимости переменных
Глобальная область: доступна везде
Локальная область: доступна только внутри функции

6. ОБЪЕКТНО-ОРИЕНТИРОВАННОЕ ПРОГРАММИРОВАНИЕ

6.1 Классы и объекты
Класс - план для создания объектов.

Пример:
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def greet(self):
        return f"Hello, I'm {self.name}"

person = Person("John", 30)
print(person.greet())  # "Hello, I'm John"

6.2 Наследование
Класс-наследник получает свойства класса-родителя.

class Student(Person):
    def __init__(self, name, age, grade):
        super().__init__(name, age)
        self.grade = grade

6.3 Инкапсуляция
Скрытие внутренних деталей класса от внешнего мира.

Приватные атрибуты (начинаются с _):
class Account:
    def __init__(self):
        self._balance = 0
    
    def deposit(self, amount):
        self._balance += amount

7. ОБРАБОТКА ОШИБОК

7.1 Try-except
try:
    # код, который может вызвать ошибку
except ТипОшибки:
    # обработка ошибки

Пример:
try:
    x = 1 / 0
except ZeroDivisionError:
    print("Деление на ноль!")

7.2 Finally
try:
    # код
except:
    # обработка ошибки
finally:
    # выполнится в любом случае

8. РАБОТА С ФАЙЛАМИ

8.1 Открытие и закрытие файлов
with open("file.txt", "r") as file:
    content = file.read()

Режимы:
- "r": чтение
- "w": запись (перезаписывает файл)
- "a": добавление в конец файла
- "rb": чтение в бинарном формате

8.2 Чтение файла
file = open("file.txt", "r")
lines = file.readlines()  # список строк
file.close()

8.3 Запись в файл
with open("file.txt", "w") as file:
    file.write("Hello, World!")

10. АЛГОРИТМЫ И СТРУКТУРЫ ДАННЫХ

10.1 Поиск
Линейный поиск: проверяет каждый элемент
Сложность: O(n)

Бинарный поиск: требует отсортированный массив
Сложность: O(log n)

10.2 Сортировка
Сортировка пузырьком: O(n²)
Сортировка слиянием: O(n log n)
Быстрая сортировка: O(n log n) в среднем

10.3 Структуры данных
Массив: быстрый доступ по индексу
Связный список: гибкое добавление/удаление
Стек (LIFO): последний вошел, первый вышел
Очередь (FIFO): первый вошел, первый вышел
Граф: связанные узлы
Дерево: иерархическая структура
