ОСНОВЫ ЯЗЫКА PYTHON

1. ТИПЫ ДАННЫХ
Python поддерживает несколько основных типов данных:

int (целое число): 42, -10, 0
float (дробное число): 3.14, -2.5, 1.0
str (строка): "Hello", 'Python', "123"
bool (логический): True, False

Проверка типа: type(переменная)

2. ПЕРЕМЕННЫЕ И ПРИСВАИВАНИЕ
Переменная - это имя для хранения значения.

x = 10          # целое число
name = "Alice"  # строка
pi = 3.14       # дробное число
is_active = True  # логическое значение

Правила имен:
- Начинаются с буквы или подчеркивания
- Содержат только буквы, цифры и подчеркивание
- Чувствительны к регистру (x и X - разные переменные)

3. ОПЕРАЦИИ
Арифметические: +, -, *, /, //, %, **
Сравнения: ==, !=, <, >, <=, >=
Логические: and, or, not

Примеры:
a = 10 + 5  # 15
b = 20 / 4  # 5.0
c = 3 ** 2  # 9 (степень)
d = 10 // 3  # 3 (целочисленное деление)
e = 10 % 3  # 1 (остаток)

4. СТРОКИ
Строка - последовательность символов.

s = "Hello, Python!"
Индексация: s[0] = 'H', s[1] = 'e'
Длина: len(s) = 14
Срезы: s[0:5] = "Hello"
Методы: s.upper(), s.lower(), s.split(), s.replace()

5. СПИСКИ (МАССИВЫ)
Список - упорядоченная коллекция элементов.

numbers = [1, 2, 3, 4, 5]
mixed = [1, "text", 3.14, True]

Операции:
- Добавление: numbers.append(6)
- Вставка: numbers.insert(2, 99)
- Удаление: numbers.remove(3) или del numbers[0]
- Доступ: numbers[0] = 1, numbers[-1] = 5
- Длина: len(numbers) = 5

Срезы: numbers[1:4] = [2, 3, 4]

6. СЛОВАРИ (АССОЦИАТИВНЫЕ МАССИВЫ)
Словарь - коллекция пар ключ-значение.

person = {
    "name": "Alice",
    "age": 30,
    "city": "Moscow"
}

Доступ: person["name"] = "Alice"
Добавление: person["email"] = "alice@mail.com"
Удаление: del person["city"]

7. УСЛОВНЫЕ ОПЕРАТОРЫ
if-else управляет ходом выполнения программы.

if x > 10:
    print("x больше 10")
elif x == 10:
    print("x равен 10")
else:
    print("x меньше 10")

8. ЦИКЛЫ
Цикл for для итерации по последовательности:
for i in range(5):
    print(i)  # выведет 0, 1, 2, 3, 4

Цикл while для повторения условия:
while x < 10:
    print(x)
    x += 1

9. ФУНКЦИИ
Функция - блок переиспользуемого кода.

def greet(name):
    return f"Hello, {name}!"

result = greet("Alice")  # "Hello, Alice!"

10. ВСТРОЕННЫЕ ФУНКЦИИ
print() - вывод на экран
input() - ввод с клавиатуры
len() - длина
sum() - сумма элементов
max(), min() - максимум, минимум
int(), float(), str() - преобразование типов
