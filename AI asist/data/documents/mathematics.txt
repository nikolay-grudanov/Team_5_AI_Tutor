МАТЕМАТИКА - ОСНОВНЫЕ КОНЦЕПЦИИ И МЕТОДЫ

1. АЛГЕБРА

1.1 Квадратные уравнения
Квадратное уравнение имеет форму: ax² + bx + c = 0, где a ≠ 0.

Решение через дискриминант:
D = b² - 4ac

Если D > 0: два различных корня
x₁ = (-b + √D) / 2a
x₂ = (-b - √D) / 2a

Если D = 0: один корень (двойной)
x = -b / 2a

Если D < 0: нет действительных корней

Пример: x² - 5x + 6 = 0
D = 25 - 24 = 1
x₁ = (5 + 1) / 2 = 3
x₂ = (5 - 1) / 2 = 2

1.2 Функции и графики
Линейная функция: y = kx + b
- k - коэффициент наклона (slope)
- b - точка пересечения с осью y

Квадратичная функция: y = ax² + bx + c
- Парабола, ветви вверх если a > 0, вниз если a < 0
- Вершина парабола: x = -b / 2a

Экспоненциальная функция: y = a^x
- Растет быстро при a > 1
- Убывает при 0 < a < 1

Логарифмическая функция: y = log_a(x)
- Обратна экспоненциальной функции
- Определена только для x > 0

2. ГЕОМЕТРИЯ

2.1 Треугольники
Теорема Пифагора: a² + b² = c² (для прямоугольного треугольника)

Площадь треугольника: S = (1/2) * основание * высота

Периметр: P = a + b + c

Синус угла: sin(A) = противоположная сторона / гипотенуза
Косинус угла: cos(A) = прилежащая сторона / гипотенуза
Тангенс угла: tan(A) = sin(A) / cos(A)

2.2 Окружность
Площадь окружности: S = πr²
Периметр (длина окружности): P = 2πr

Диаметр: d = 2r

Уравнение окружности: (x - x₀)² + (y - y₀)² = r²

3. МАТЕМАТИЧЕСКИЙ АНАЛИЗ

3.1 Производные
Производная функции показывает скорость изменения функции.

Правила дифференцирования:
- (c)' = 0, где c - константа
- (x^n)' = n*x^(n-1)
- (sin(x))' = cos(x)
- (cos(x))' = -sin(x)
- (e^x)' = e^x

Правило произведения: (u*v)' = u'*v + u*v'
Правило деления: (u/v)' = (u'*v - u*v') / v²

3.2 Интегралы
Интеграл - это операция, обратная производной.

Неопределённый интеграл: ∫f(x)dx

Основные интегралы:
∫x^n dx = x^(n+1)/(n+1) + C
∫sin(x) dx = -cos(x) + C
∫cos(x) dx = sin(x) + C
∫e^x dx = e^x + C

Определённый интеграл: ∫[a,b] f(x)dx

Геометрический смысл: площадь под кривой от a до b

4. ТЕОРИЯ ВЕРОЯТНОСТИ И СТАТИСТИКА

4.1 Вероятность
Вероятность события: P(A) = количество благоприятных исходов / количество всех исходов

Свойства вероятности:
- 0 ≤ P(A) ≤ 1
- P(A) + P(not A) = 1
- P(A и B) = P(A) * P(B|A) - условная вероятность

4.2 Статистика
Среднее арифметическое: x̄ = (x₁ + x₂ + ... + xₙ) / n

Дисперсия: σ² = Σ(xᵢ - x̄)² / n

Стандартное отклонение: σ = √(σ²)

Медиана: средний элемент упорядоченного набора данных

Мода: наиболее часто встречающийся элемент
