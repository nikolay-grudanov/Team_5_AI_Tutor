ОСНОВЫ АЛГЕБРЫ

1. ПОНЯТИЕ ПЕРЕМЕННОЙ И ВЫРАЖЕНИЯ
Переменная - это буква, которая обозначает неизвестное число.
Алгебраическое выражение - это комбинация чисел, переменных и операций.

Пример: 3x + 5 - это алгебраическое выражение, где x - переменная.

2. ЛИНЕЙНЫЕ УРАВНЕНИЯ
Линейное уравнение - это уравнение первой степени.
Общий вид: ax + b = 0, где a ≠ 0

Решение линейного уравнения:
- Шаг 1: Перенесите известные в одну сторону
- Шаг 2: Разделите обе части на коэффициент переменной
- Шаг 3: Проверьте ответ подстановкой

Пример: 2x + 5 = 11
2x = 11 - 5
2x = 6
x = 3

3. КВАДРАТНЫЕ УРАВНЕНИЯ
Квадратное уравнение - это уравнение второй степени.
Общий вид: ax² + bx + c = 0, где a ≠ 0

Формула дискриминанта: D = b² - 4ac
Корни: x = (-b ± √D) / (2a)

Случаи:
- Если D > 0: два различных корня
- Если D = 0: один корень
- Если D < 0: нет вещественных корней

4. СТЕПЕНИ И КОРНИ
Степень: aⁿ = a · a · a ... · a (n раз)
Свойства:
- aᵐ · aⁿ = aᵐ⁺ⁿ
- aᵐ / aⁿ = aᵐ⁻ⁿ
- (aᵐ)ⁿ = aᵐⁿ
- (ab)ⁿ = aⁿbⁿ

Квадратный корень: √a² = |a|
Кубический корень: ∛a³ = a

5. ФАКТОРИЗАЦИЯ
Факторизация - разложение на множители.

Формулы сокращённого умножения:
- a² - b² = (a - b)(a + b)
- (a + b)² = a² + 2ab + b²
- (a - b)² = a² - 2ab + b²
- a³ + b³ = (a + b)(a² - ab + b²)
- a³ - b³ = (a - b)(a² + ab + b²)
