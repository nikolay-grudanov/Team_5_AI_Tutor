ХИМИЯ - ОСНОВНЫЕ ПОНЯТИЯ И РЕАКЦИИ

1. АТОМЫ И МОЛЕКУЛЫ

1.1 Атомная структура
Атом состоит из:
- Ядра (протоны и нейтроны)
- Электронной оболочки (электроны)

Атомный номер = количество протонов
Массовое число = протоны + нейтроны

Электронная конфигурация:
Электроны расположены по энергетическим уровням:
1s² 2s² 2p⁶ 3s² 3p⁶ 3d¹⁰ ...

1.2 Ионы и химическая связь
Катион - атом, потерявший электроны (+)
Анион - атом, приобретший электроны (-)

Виды химических связей:
- Ковалентная связь: общая пара электронов
- Ионная связь: электростатическое притяжение
- Металлическая связь: море электронов
- Водородная связь: специальная ковалентная связь

2. СТЕХИОМЕТРИЯ

2.1 Молярные массы
Молярная масса = сумма атомных масс
Пример: H₂O = 2×1 + 16 = 18 г/моль

2.2 Уравнение химических реакций
Уравнение показывает типы и количества реагентов и продуктов.

Закон сохранения массы:
Масса реагентов = Масса продуктов

Пример уравнения:
2H₂ + O₂ → 2H₂O
- 2 молекулы водорода + 1 молекула кислорода = 2 молекулы воды

2.3 Расчеты по уравнениям реакций
Пример: сколько воды образуется из 4 г водорода?

2H₂ + O₂ → 2H₂O

Молярная масса H₂ = 2 г/моль
Молярная масса H₂O = 18 г/моль

Количество вещества H₂ = 4 г / 2 г/моль = 2 моль

По уравнению: 2 моль H₂ → 2 моль H₂O
Следовательно: 2 моль H₂O = 2 × 18 = 36 г

3. КИСЛОТНО-ЩЕЛОЧНЫЕ РЕАКЦИИ

3.1 pH и pOH
pH = -log[H⁺]
pOH = -log[OH⁻]
pH + pOH = 14 (при 25°C)

pH < 7: кислотная среда
pH = 7: нейтральная среда
pH > 7: щелочная среда

3.2 Сильные и слабые кислоты/основания
Сильные кислоты: HCl, HNO₃, H₂SO₄
Сильные основания: NaOH, KOH, Ca(OH)₂

Слабые кислоты: CH₃COOH, HF
Слабые основания: NH₃

3.3 Нейтрализация
Кислота + Основание → Соль + Вода
HCl + NaOH → NaCl + H₂O

4. ОКИСЛИТЕЛЬНО-ВОССТАНОВИТЕЛЬНЫЕ РЕАКЦИИ

4.1 Степени окисления
Степень окисления - число электронов, потерянных или приобретенных атомом.

Правила определения:
- В элементарном веществе: 0
- В ионе одноатомном: равна заряду иона
- O в соединениях: -2 (кроме пероксидов -1)
- H обычно: +1

4.2 Окисление и восстановление
Окисление: потеря электронов (степень окисления возрастает)
Восстановление: приобретение электронов (степень окисления уменьшается)

Окислитель: вещество, которое окисляется (принимает электроны)
Восстановитель: вещество, которое восстанавливается (отдает электроны)

Пример: 2Na + Cl₂ → 2NaCl
Na: 0 → +1 (окисляется, восстановитель)
Cl: 0 → -1 (восстанавливается, окислитель)

5. ОРГАНИЧЕСКАЯ ХИМИЯ

5.1 Углеводороды
Алканы (CₙH₂ₙ₊₂): содержат только C-C и C-H связи
Метан: CH₄
Этан: C₂H₆
Пропан: C₃H₈

Алкены (CₙH₂ₙ): содержат C=C двойную связь
Этен: C₂H₄
Пропен: C₃H₆

Алкины (CₙH₂ₙ₋₂): содержат C≡C тройную связь
Этин (ацетилен): C₂H₂

5.2 Кислородсодержащие соединения
Спирты (R-OH):
Этанол: C₂H₅OH
Пропанол: C₃H₇OH

Альдегиды (R-CHO):
Формальдегид: HCHO
Ацетальдегид: CH₃CHO

Карбоновые кислоты (R-COOH):
Муравьиная кислота: HCOOH
Уксусная кислота: CH₃COOH

Эфиры (R-O-R'):
Диэтиловый эфир: C₂H₅-O-C₂H₅

5.3 Важнейшие реакции органических соединений
Горение (при наличии O₂):
CₙHₘ + O₂ → CO₂ + H₂O

Присоединение (для алкенов и алкинов):
C₂H₄ + H₂ → C₂H₆ (гидрирование)
C₂H₄ + HCl → C₂H₅Cl

Замещение (для алканов):
CH₄ + Cl₂ → CH₃Cl + HCl

Эстерификация:
R-COOH + HO-R' → R-COO-R' + H₂O

6. ХИМИЯ РАСТВОРОВ

6.1 Концентрация растворов
Массовая доля: ω = mвещества / mраствора × 100%

Молярная концентрация (молярность): c = n/V (моль/л)

Моляльность: b = n/mрастворителя (моль/кг)

6.2 Растворимость
При 20°C типичные растворимости:
- NaCl: 36 г/100 мл воды
- CaCO₃: 0.0013 г/100 мл воды
- MgSO₄: 35 г/100 мл воды

Ненасыщенный раствор: содержит меньше растворенного вещества, чем при полном растворении
Насыщенный раствор: содержит максимальное количество растворенного вещества
Пересыщенный раствор: метастабильное состояние

7. ХИМИЧЕСКОЕ РАВНОВЕСИЕ

7.1 Константа равновесия
aA + bB ⇌ cC + dD

K = [C]^c × [D]^d / [A]^a × [B]^b

Если K >> 1: реакция практически завершена
Если K << 1: реакция практически не идет
Если K ≈ 1: достигается равновесие со значительными количествами реагентов и продуктов

7.2 Принцип Ле Шателье
При изменении условий (температура, давление, концентрация) равновесие смещается так, чтобы частично компенсировать это изменение.

Примеры:
- Увеличение температуры смещает равновесие в сторону эндотермической реакции
- Увеличение давления смещает равновесие в сторону образования вещества с меньшим объемом
