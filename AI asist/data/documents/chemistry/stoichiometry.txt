СТЕХИОМЕТРИЯ - РАСЧЕТЫ ПО УРАВНЕНИЯМ РЕАКЦИЙ

1. ОСНОВНЫЕ ПОНЯТИЯ
Стехиометрия - раздел химии, занимающийся расчетом количеств реагирующих веществ.

Молярная масса (M): сумма атомных масс элементов в веществе (г/моль)
Пример: H₂O: 2×1 + 16 = 18 г/моль

Количество вещества (n): n = m/M (моль)
где m - масса вещества (г)

Объем газа при н.у.: V = n × 22.4 (л)
где 22.4 л/моль - молярный объем газа

2. ХИМИЧЕСКИЕ УРАВНЕНИЯ
Химическое уравнение показывает качественный состав и количественное соотношение.

Пример: 2H₂ + O₂ = 2H₂O
Читается: 2 молекулы водорода + 1 молекула кислорода = 2 молекулы воды

Коэффициенты показывают количественное соотношение участвующих веществ.

3. РАСЧЕТЫ ПО МАССЕ
Задача: Сколько грамм CO₂ образуется при сгорании 12 г углерода?
2C + O₂ = 2CO₂

Решение:
1. M(C) = 12 г/моль, M(CO₂) = 44 г/моль
2. n(C) = 12 / 12 = 1 моль
3. По уравнению: 2 моль C → 2 моль CO₂
4. 1 моль C → 1 моль CO₂
5. m(CO₂) = 1 × 44 = 44 г

4. РАСЧЕТЫ ПО ОБЪЕМУ ГАЗОВ
При н.у. (0°C, 101.325 кПа):
V = n × 22.4 л/моль

Пример: N₂ + 3H₂ = 2NH₃
При сгорании 5.6 л N₂:
n(N₂) = 5.6 / 22.4 = 0.25 моль
n(NH₃) = 0.5 моль
V(NH₃) = 0.5 × 22.4 = 11.2 л

5. СТЕПЕНЬ ОКИСЛЕНИЯ
Степень окисления - условный заряд атома в соединении.

Правила:
- Для простого вещества: 0
- Для иона одноатомного: заряд иона
- O в соединениях: обычно -2 (кроме пероксидов и OF₂)
- H в соединениях: обычно +1 (кроме гидридов -1)
- Сумма степеней окисления = заряду

6. ОКИСЛИТЕЛЬНО-ВОССТАНОВИТЕЛЬНЫЕ РЕАКЦИИ
Окисление - потеря электронов (увеличение степени окисления)
Восстановление - присоединение электронов (уменьшение степени окисления)

Восстановитель - вещество, которое окисляется
Окислитель - вещество, которое восстанавливается

7. ПРЕДЕЛЬНЫЕ РЕАГЕНТЫ
Если вещества даны в неправильном количественном соотношении, то одно из них расходуется полностью (предельный реагент), а другое остается в избытке.

Метод решения:
1. Найдите количество каждого вещества
2. Разделите на коэффициент в уравнении
3. Меньший результат - это предельный реагент
4. Расчеты ведите по предельному реагенту
