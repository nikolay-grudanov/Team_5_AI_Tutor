   ├─ Загрузка файлов (.txt, .md, .pdf)
   ├─ Автоматическая очистка текста
   ├─ Разбиение на чанки с перекрытием
   ├─ 4 встроенных предмета (Математика, Физика, Химия, ПО)
   └─ Максимальная гибкость добавления своих материалов

2. ВЕКТОРНОЕ ХРАНИЛИЩЕ ✓
   ├─ Модель эмбеддингов: sentence-transformers
   ├─ Создание векторных представлений
   ├─ Быстрый поиск по косинусному сходству
   ├─ Сохранение и загрузка индексов
   └─ Оптимизированная батч-обработка

3. RAG-ПАЙПЛАЙН ✓
   ├─ Поиск релевантных документов (retrieve)
   ├─ Форматирование контекста
   ├─ Построение умных промптов
   ├─ Генерация ответов
   └─ Полная настраиваемость каждого шага

4. ГЕНЕРАЦИЯ ОТВЕТОВ ✓
   ├─ Интеграция с LLM (gpt-2 by default)
   ├─ Поддержка GPU/CUDA
   ├─ Фоллбэк механизм для простой генерации
   ├─ Улучшение и форматирование ответов
   └─ Добавление ссылок на источники

5. ВЕБ-ИНТЕРФЕЙС ✓
   ├─ Современный отзывчивый дизайн
   ├─ Реал-тайм чат с AI
   ├─ История всех вопросов и ответов
   ├─ Информационная панель с метаданными
   ├─ Отображение источников информации
   └─ Управление сессиями пользователей

6. REST API ✓
   ├─ GET / - Главная страница
   ├─ GET /api/status - Статус системы
   ├─ POST /api/ask - Обработка вопроса
   ├─ GET /api/history - История вопросов
   ├─ GET /api/session-info - Информация сессии
   └─ POST /api/clear-history - Очистка истории

АРХИТЕКТУРА:
  Фронтенд:   HTML5 + CSS3 + JavaScript (Vanilla)
  Бэкенд:     Python Flask 2.3
  ML:         PyTorch 2.0 + Transformers 4.30
  Эмбеддинги: Sentence Transformers 2.2
  Данные:     NumPy 1.24

ПРОИЗВОДИТЕЛЬНОСТЬ:
  Время запуска:         ~10-30 секунд
  Время первого ответа:  ~3-10 секунд (первая загрузка модели)
  Время ответа:         ~1-3 секунды
  GPU ускорение:        Автоматическое (если NVIDIA доступна)

ТРЕБОВАНИЯ:
  Минимум: Python 3.8+, 2GB RAM, 1GB диска
  Рекомендуется: Python 3.10+, 8GB RAM, GPU
