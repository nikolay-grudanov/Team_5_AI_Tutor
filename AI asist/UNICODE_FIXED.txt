ЕСЛИ ВСЕ ЕЩЕ ЕСТЬ ПРОБЛЕМЫ:

Способ 1 - Включить UTF-8 в консоли:
  
  chcp 65001
  python quickstart.py

Способ 2 - Использовать CMD вместо PowerShell:

  cmd /k "chcp 65001 && python quickstart.py"

Способ 3 - Обновить консоль Windows (если на старой версии Windows 10):

  - Откройте Microsoft Store
  - Найдите "Windows Terminal"
  - Установите (это новая консоль с лучшей поддержкой UTF-8)
