✨ Вариант 1: Автоматический запуск (РЕКОМЕНДУЕТСЯ)

  ✓ Проверит версию Python (нужна 3.8+)
  ✓ Создаст виртуальное окружение
  ✓ Установит все зависимости
  ✓ Запустит веб-сервер

Откройте в браузере: http://localhost:5000


✨ Вариант 2: Ручной запуск
───────────────────────────────────────────────────────────────
# Создать виртуальное окружение
python -m venv venv

# Активировать (Windows)
venv\Scripts\activate

# Установить зависимости
pip install -r requirements.txt

# Запустить
python run.py

📊 СТАТИСТИКА И МОНИТОРИНГ

Размер индекса:
  - Документов: 4 (встроенные)
  - Chunks: 150+ (автоматически создаются)
  - Размер embeddings.npy: ~1-5 MB

Производительность:
  - Время индексирования: 10-30 сек (первый старт)
  - Время поиска: 100-200 ms
  - Время генерации ответа: 1-3 сек

Модели:
  - Embedding: sentence-transformers/all-MiniLM-L6-v2
  - LLM: gpt-2 (по умолчанию)

🐛 РЕШЕНИЕ ПРОБЛЕМ

Проблема: "ModuleNotFoundError: No module named 'flask'"
Решение:
  $ pip install -r requirements.txt

Проблема: "Port 5000 is already in use"
Решение:
  Измените PORT в config.py или завершите процесс на 5000

Проблема: "No vector store found"
Решение:
  Перезагрузите приложение, он создаст индекс автоматически

Проблема: Система работает медленно
Решение:
  Может идти индексирование при первом запуске (30 сек)
  Проверьте RAM (нужно минимум 2 GB)

Проблема: Система не находит документы
Решение:
  Проверьте наличие файлов в data/documents/
  Пересоздайте индекс: rm -rf data/vector_store/

Сохранить историю разговора:
  Скопируйте из левой панели или API /api/history

💡 СОВЕТЫ И ТРЮКИ

1. Для лучших результатов используйте русский язык при 
   добавлении документов, если вопросы на русском

2. Структурируйте документы с четкими заголовками - это 
   улучшает поиск

3. Добавляйте примеры в текст - они помогают системе

4. Периодически проверяйте качество ответов и 
   добавляйте уточнения в документы

5. Используйте /api/status для мониторинга 
   состояния системы
