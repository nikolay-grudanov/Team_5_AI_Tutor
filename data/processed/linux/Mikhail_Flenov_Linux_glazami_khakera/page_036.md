В случае установки серверной версии все шаги будут выполняться в текстовом режиме и вот что интересно — клавиатуру придется определять старым способом, т. е. нажимая разные клавиши. Программа установки при этом определит, какая именно клавиатура используется на компьютере. В графическом режиме обычно дается выбор в зависимости от языка. Если пользователь выбрал русский язык, то для уточнения клавиатуры можно будет выбрать какую-либо из разных русских раскладок. Окно настройки оборудования при установке в текстовом режиме показано на рис. 2.1.

![Настройка оборудования при установке в текстовом режиме](https://i.imgur.com/3Q5z5QG.png)

Рис. 2.1. Настройка оборудования при установке в текстовом режиме

Может показаться, что при установке в текстовом режиме Linux задает слишком много вопросов. Это только кажется. Просто для каждого из параметров будет открыто самостоятельное окно: отдельно спросят имя компьютера, имя пользователя, потом запросят пароль и подтверждения пароля — это уже четыре окна с вопросами. В графическом режиме программа установки показывает только одно окно, в котором все собрано в одном месте и кажется, что установка проходит за меньшее количество шагов.
