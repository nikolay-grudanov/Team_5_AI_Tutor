Часть V. Шаблоны

Эта часть книги посвящена шаблонам, являющимся составной частью некоторых, одних из самых мощных команд Vim. Мы по-знакомимся с некоторыми хитростями, упрощающими составление регулярных выражений и поиск литерального текста. Изучим особенности работы самого механизма поиска и затем исследуем две мощные команды Ex: :substitute, позволяющую находить совпадения с шаблоном и заменять их чем-то другим, и :global, дающую возможность выполнить любую команду Ex для каждой строки, соответствующей указанному шаблону.
