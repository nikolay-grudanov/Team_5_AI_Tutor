Таблица 4.1. Стандартные команды Linux для работы с файлами

<table>
  <tr>
    <th>Команда</th>
    <th>Описание</th>
  </tr>
  <tr>
    <td>cat файл</td>
    <td>Выводит текстовый файл. Файлы могут быть довольно длинными, поэтому лучше использовать ее в паре с командой less, например, cat /etc/services | less</td>
  </tr>
  <tr>
    <td>tac файл</td>
    <td>Подобна команде cat, но выводит файл в обратном порядке. Данная команда удобна для чтения журналов, в которых самые свежие сообщения заносятся в конец файла, например, tac /var/log/messages | less</td>
  </tr>
  <tr>
    <td>cp файл1 файл2</td>
    <td>Копирует файл1 в файл2. Если второй файл существует, программа спросит вас, нужно ли его перезаписать</td>
  </tr>
  <tr>
    <td>mv файл1 файл2</td>
    <td>Используется для перемещения файла1 в файл2. Можно использовать для переименования файлов</td>
  </tr>
  <tr>
    <td>rm файл</td>
    <td>Удаляет файл</td>
  </tr>
  <tr>
    <td>touch файл</td>
    <td>Используется для создания нового пустого файла</td>
  </tr>
  <tr>
    <td>locate файл</td>
    <td>Быстрый поиск файла. Позже мы рассмотрим процесс поиска подробнее</td>
  </tr>
  <tr>
    <td>which исполняемый_файл</td>
    <td>Производит быстрый поиск программы (исполнимого файла). Если программа находится в пути PATH, то which выведет каталог, в котором находится программа.</td>
  </tr>
</table>

В таблице 4.1 представлены основные команды, которые используются для создания (touch), копирования (cp), перемещения (mv) и удаления (rm) файлов, а также несколько дополнительных команд.
