Таблица 2.1. Параметры команды shutdown

<table>
  <tr>
    <th>Параметр</th>
    <th>Описание</th>
  </tr>
  <tr>
    <td>-H (или --halt)</td>
    <td>Завершает работу системы, питание не отключается</td>
  </tr>
  <tr>
    <td>-P (или --power-off)</td>
    <td>Завершает работу системы, питание отключается</td>
  </tr>
  <tr>
    <td>-r (или --reboot)</td>
    <td>Перезагружает компьютер</td>
  </tr>
  <tr>
    <td>-h</td>
    <td>Аналогично --poweroff, то есть завершение работы с отключением питания</td>
  </tr>
  <tr>
    <td>-k</td>
    <td>Работа системы не завершается, просто каждый пользователь получит указанное сообщение.</td>
  </tr>
  <tr>
    <td>--no-wall</td>
    <td>При завершении работы системы (в том числе перезагрузке) пользователям не будут выводиться</td>
  </tr>
  <tr>
    <td>-c</td>
    <td>Отменяет отложенное завершение работы (если вы передумали завершать работу или перезагружать систему, но при условии, что процесс завершения работы еще не начат)</td>
  </tr>
</table>
