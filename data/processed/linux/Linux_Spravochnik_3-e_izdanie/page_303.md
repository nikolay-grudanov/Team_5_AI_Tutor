    <th>mkfs</th>
    <td>
      <b>mkfs [options][fs-options] filesys [blocks]</b><br>
      Команда системного администрирования. Создать файловую систему на устройстве (например, на разделе жесткого диска). Аргумент filesys является либо именем устройства, либо точкой монтирования. Команда, по существу, представляет собой интерфейс для вызова соответствующего варианта mkfs в зависимости от файловой системы, указанной в параметре -t. К примеру, для создания второй расширенной файловой системы Linux вызывается mkfs.ext2 (что идентично mke2fs); файловые системы MS-DOS используют mkfs.msdos. fs-options — это параметры создаваемой системы. blocks — размер файловой системы в блоках по 1024 байт.<br>
      <b>Параметры</b><br>
      <b>-V</b> Диагностика работы, включая все выполняемые для создания файловой системы команды.<br>
      <b>-t fs-type</b><br>
      Тип создаваемой файловой системы.<br>
      <b>Специфические параметры файловой системы</b><br>
      Эти параметры должны следовать строго после основных. В большинстве программ, создающих файловые системы, поддерживаются следующие параметры:<br>
      <b>-c</b> Перед созданием файловой системы произвести проверку на наличие сбойных блоков.<br>
      <b>-l file</b> Взять список существующих сбойных блоков из указанного файла.<br>
      <b>-v</b> Диагностика работы.
    </td>
