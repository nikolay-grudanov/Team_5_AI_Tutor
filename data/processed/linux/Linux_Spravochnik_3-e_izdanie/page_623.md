    <th>history</th>
    <td><b>-S file</b><br>
      Только для tcsh. Отобразить последовательность команд pushd и popd, воссоздающую стек, в файл <i>file</i>.
      <br>
      <b>Пример</b>
      <br>
      Сохранить и выполнить пять последних команд:
      <pre>
history -h 5 &gt; do_it
source do_it
      </pre>

    <th>hup</th>
    <td><b>hup [command]</b><br>
      Только для tcsh. Выполнить команду, но принудительно завершить задание при посылке сигнала hangup при завершении работы интерпретатора. Команда без параметра может использоваться внутри сценария, который должен быть принудительно завершен при выходе из интерпретатора.

    <th>if</th>
    <td><b>if</b><br>
      Начало условного оператора. Простейший формат оператора:
      <pre>
if (expr) cmd
      </pre>
      Применяются еще три формата, которые приведены ниже:
      <pre>
if (expr) then   if (expr) then   if (expr) then
  cmds           cmds1           cmds1
endif            else             endif
                cmds2             else if (expr) then
endif            endif            cmds2
                        else
                        cmds3
                        endif
      </pre>
      В первой форме оператора команда <i>cmd</i> выполняется, если выражение <i>expr</i> истинно, иначе ничего не происходит (кроме перенаправления, что является ошибкой). Во всех прочих вариантах выполняется одна или более команд. Если выражение истинно, выполняются команды, расположенные непосредственно после <i>then</i>; если <i>expr</i> ложно, происходит выполнение команд после <i>else</i> (либо переход к <i>else if</i> и следующей проверке). Дополнительные примеры см. ранее в этой главе (в разделе «Выражения»), а также в описаниях <i>while</i> и <i>shift</i>.
