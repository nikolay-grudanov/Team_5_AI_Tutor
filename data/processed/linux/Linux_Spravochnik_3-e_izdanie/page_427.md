    <th>tunelp</th>
    <td><b>tunelp device [options]</b><br>
      Команда системного администрирования. Настройка параметров работы принтера. Команда без параметров выведет информацию по устройству (устройствам).

      <b>Параметры</b>
      <ul>
        <li><b>-a [on | off]</b><br>
          Позволяет указать, надо завершать работу при ошибке принтера или нет. По умолчанию работа не завершается.
        </li>
        <li><b>-c n</b><br>
          Количество попыток повторной посылки символа в устройство. (По умолчанию равно 250). После обнуления счетчика сделать паузу перед повторением попыток.
        </li>
        <li><b>-i irq</b><br>
          Назначить указанное прерывание <i>irq</i> для работы с параллельным портом. Игнорируются параметры <b>-t</b> и <b>-c</b>. При значении 0 восстанавливается непрерываемая работа (polling) с устройством.
        </li>
        <li><b>-o [on | off]</b><br>
          Переключение завершения работы, если устройство недоступно или нет бумаги в устройстве.
        </li>
        <li><b>-q [on | off]</b><br>
          Переключение отображения текущих настроек IRQ.
        </li>
        <li><b>-r</b><br>
          Сброс порта.
        </li>
        <li><b>-s</b><br>
          Отображение текущего состояния принтера.
        </li>
        <li><b>-t time</b><br>
          Длительность паузы (в jiffies) между повторными посылками символа в устройство. Мгновение (jiffy) определяется либо как один квант системных часов, либо один цикл времени АС. Приблизительно 0,01 секунды.
        </li>
        <li><b>-w time</b><br>
          Указать время задержки (в мгновениях) перед повторной посылкой тактового сигнала.
        </li>
        <li><b>-C [on | off]</b><br>
          Включение или отключение повышенной «внимательности» демона печати при выявлении ошибок печатающего устройства.
        </li>
      </ul>
