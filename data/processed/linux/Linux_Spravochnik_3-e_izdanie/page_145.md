    <th>fetchmail</th>
    <td>
      <b>-u name, --username name</b><br>
      Указать имя пользователя для авторизации на почтовом сервере.<br>
      <b>-V, --version</b><br>
      Вывести информацию о номере версии fetchmail и набор параметров для каждого из почтовых серверов. Получение сообщений не производится.<br>
      <b>-v, --verbose</b><br>
      Выводить все сообщения о состоянии в процессе получения почты. Имеет больший приоритет, чем <b>-q</b>.<br>
      <b>-Z nnn, --antispm nnn</b><br>
      Указать номер SMTP-ошибки, <i>nnn</i>, который будет использован для предупреждения о спам-блоке от клиента. Если <i>nnn = -1</i>, параметр не имеет влияния.

    <th>fgrep</th>
    <td>
      <b>fgrep [options] pattern [files]</b><br>
      Поиск в перечисленных файлах <i>files</i> строк, соответствующих заданному шаблону <i>pattern</i>. Коды завершения: 0 — если найдена хотя бы одна строка, удовлетворяющая критерию поиска, 1 — если не найдено таких строк, 2 — если произошла ошибка.<br>
      Полный перечень параметров приведен в описании команды grep. См. также egrep.<br>
      Примеры<br>
      Вывести строки файла <i>file</i>, не содержащие пробелы:<br>
      <pre>fgrep -v ' ' file</pre>
      Вывести строки файла <i>file</i>, содержащие слова из spell_list:<br>
      <pre>fgrep -f spell_list file</pre>
    </td>
  </tr>
  <tr>
    <th>file</th>
    <td>
      <b>file [options] files</b><br>
      Классификация файлов по типу данных, содержащихся в них. Программа file обращается к «волшебному файлу» (как правило, <i>/usr/share/magic</i>), чтобы опознать некоторые типы файлов.<br>
      Параметры<br>
      <b>-b</b> Режим сокращенного отображения — не печатать имена файлов в начале строк вывода.
    </td>
