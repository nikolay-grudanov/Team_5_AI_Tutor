Импортирование целого каталога в репозиторий в качестве нового модуля. Используется для добавления в проект кода из внешних источников либо произвольного кода, который изначально создавался вне системы контроля версий CVS. Может быть указано несколько идентификаторов изданий (версий) release_tag; в таком случае для начальной версии будет создано несколько меток.

Параметр –k имеет тот же смысл, что и обычно. Дополнительные параметры перечислены в табл. 14.28.

Параметр –k будет действовать только для файлов, импортируемых в текущем сеансе. Режимы подстановки ключевых слов для файлов, уже находящихся в репозитории, не меняются.

При указании параметра –W аргумент spec имеет такой же формат, как и записи в административном файле cvswrappers (см. предшествующий раздел «Файл cvswrappers»).
