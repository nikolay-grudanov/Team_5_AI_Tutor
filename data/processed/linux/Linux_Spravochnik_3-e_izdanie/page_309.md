    <th>mount</th>
    <td>
      <b>nodev</b><br>
      Не интерпретировать специальные устройства, существующие в системе.<br>
      <b>noexec</b><br>
      Запретить исполнение файлов.<br>
      <b>nosuid</b><br>
      Не принимать во внимание биты <b>suid</b> и <b>sgid</b>.<br>
      <b>nouser</b><br>
      Разрешить доступ к монтируемой файловой системе только привилегированному пользователю.<br>
      <b>remount</b><br>
      Считать, что система уже подмонтирована, и перезмонтировать ее.<br>
      <b>ro</b> Разрешить доступ только для чтения.<br>
      <b>rw</b> Разрешить доступ для чтения и записи.<br>
      <b>suid</b><br>
      Принимать во внимание биты <b>suid</b> и <b>sgid</b>.<br>
      <b>sync</b><br>
      Синхронный ввод/вывод для устройства.<br>
      <b>user</b><br>
      Разрешить непривилегированным пользователям монтировать файловую систему. Обратите внимание, что значениями по умолчанию для такой системы будут nodev, noexec и nosuid, если не указано обратное.<br>
      <b>check=relaxed | normal | strict</b><br>
      Позволяет указать, насколько строго должна регулироваться интеграция файловой системы MS-DOS.<br>
      <b>conv=binary | text | auto</b><br>
      Определить метод преобразования файлов на файловых системах MS-DOS и ISO 9660.<br>
      <b>debug</b><br>
      Включить отладку для файловых систем MS-DOS и ext2fs.<br>
      <b>errors=continue | remount | ro|panic</b><br>
      Действие, совершаемое при возникновении ошибки. Только для файловых систем ext2fs.<br>
      <b>-r</b> Монтирование в режиме только для чтения.
