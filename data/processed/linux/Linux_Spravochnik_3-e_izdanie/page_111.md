      <b>conv=flags</b><br>
      Производить преобразование исходного файла в соответствии с перечисленными (через запятую) флагами. Первые пять флагов <i>flags</i> взаимно исключают друг друга.<br>
      <b>ascii</b><br>
      Преобразование из системы EBCDIC в ASCII.<br>
      <b>ebcdic</b><br>
      Преобразование из системы ASCII в EBCDIC.<br>
      <b>ibm</b><br>
      Преобразование из системы ASCII в EBCDIC с учетом стандартов IBM.<br>
      <b>block</b><br>
      Преобразование записей переменной длины (т. е. тех, которые завершаются символом новой строки) в записи фиксированной длины.<br>
      <b>unblock</b><br>
      Обратно предыдущему параметру.<br>
      <b>lcase</b><br>
      Преобразование прописных букв в строчные.<br>
      <b>ucase</b><br>
      Преобразование строчных букв в прописные.<br>
      <b>noerror</b><br>
      Продолжать обработку при возникновении ошибки чтения.<br>
      <b>notrunc</b><br>
      Не усекать конечный файл.<br>
      <b>swab</b><br>
      Обменивать значения пар байтов.<br>
      <b>sync</b><br>
      При чтении дополнять нулями блоки до размера, заданного параметром <i>ibs</i>.<br>
      <b>count=n</b><br>
      Копировать первые <i>n</i> прочитанных блоков.<br>
      <b>ibs=n</b><br>
      Установить размер блока для чтения в <i>n</i> байт (512 по умолчанию).<br>
      <b>if=file</b><br>
      Читать данные из файла <i>file</i> (по умолчанию выполняется чтение со стандартного ввода).
