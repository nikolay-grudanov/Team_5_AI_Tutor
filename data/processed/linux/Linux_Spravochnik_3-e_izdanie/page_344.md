Параметры

-р Вывести полученные данные.
-s Синхронизировать системное время с полученным. Действие доступно только привилегированному пользователю.

rdev [options][image[value[offset]]]
Команда системного администрирования. Команда без аргументов отображает строку в синтаксисе /etc/mtab, описывающую корневую файловую систему. В противном случае изменяет значения байт в образе ядра, которые определяют размер RAM-диска (по умолчанию располагаются по смещению 504), режим VGA (по умолчанию — 506) и корневое устройство (по умолчанию — 508). Необходимо указать изменяемый образ ядра. Можно указать значение value и иное смещение offset.

Параметры
-o offset
    То же, что и задание смещения в качестве аргумента. Смещение задается в десятичном виде.
-r Эмуляция ramsize.
-s Эмуляция swapdev.
-v Эмуляция vidmode.
-R Эмуляция rootflags.

rdist [options][names]
Команда системного администрирования. Клиентская часть программы удаленного распределения файлов. rdist обеспечивает синхронизацию версий файлов на множественных узлах. Управление обновлением файлов и/или каталогов производится командами из файла distfile. Альтернативный distfile может быть указан параметром -f или -c.

Параметры
-a nmt
    Не обновлять файловую систему, если на ней менее nmt свободных байт.
