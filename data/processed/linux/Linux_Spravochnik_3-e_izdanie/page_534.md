<table>
  <tr>
    <th>Синтаксис</th>
    <th>Действие</th>
  </tr>
  <tr>
    <td>(cmd1 ; cmd2)</td>
    <td>«Подчиненный интерпретатор»; считать команды cmd1 и cmd2 группой команд</td>
  </tr>
  <tr>
    <td>cmd1 | cmd2</td>
    <td>Конвейер; использовать вывод команды cmd1 как ввод для команды cmd2</td>
  </tr>
  <tr>
    <td>cmd1 `cmd2`</td>
    <td>Подстановка команд; результат работы команды cmd2 передается в качестве аргументов команде cmd1</td>
  </tr>
  <tr>
    <td>cmd1 $(cmd2)</td>
    <td>Подстановка команд с допустимой вложенностью</td>
  </tr>
  <tr>
    <td>cmd1 && cmd2</td>
    <td>Логическое И; команда cmd2 выполняется только в том случае, если успешно выполнена команда cmd1</td>
  </tr>
  <tr>
    <td>cmd1 || cmd2</td>
    <td>Логическое ИЛИ; команда cmd2 выполняется только в том случае, если cmd1 выполнена неудачно</td>
  </tr>
  <tr>
    <td>{ cmd1 ; cmd2 }</td>
    <td>Выполнять команды в текущем интерпретаторе</td>
  </tr>
</table>

Примеры

$ nroff file &        Фоновое форматирование текста
$ cd; ls              Последовательное выполнение команд
$ (date; who; pwd) > logfile   Перенаправление всего вывода в файл
$ sort file | pr -3 | lp      Сортировка файла, разбивка результата на страницы, печать
$ vi 'grep -l ifdef *.c'     Редактировать файлы, найденные grep
$ egrep '(yes|no)' 'cat list'   Определить список файлов для поиска
$ egrep '(yes|no)' $(cat list)   bash-эквивалент предыдущей команды
$ egrep '(yes|no)' $(<list)     Действует так же, как и предыдущая команда, но быстрее выполняется
$ grep XX file && lp file     Печать файла, содержащего заданный текст
$ grep XX file || echo "XX not found"   Вывод сообщения об ошибке, если текст XX не найден
