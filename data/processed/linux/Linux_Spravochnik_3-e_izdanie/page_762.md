Каталоги CVS

Каждый каталог в рабочем каталоге содержит подкаталог CVS. Файлы в этом каталоге (табл. 14.15) содержат метаданные, используемые CVS для определения расположения репозитория и отслеживания копирования версий файлов в рабочий каталог.

Поскольку в каждом рабочем каталоге есть только один файл CVS/Root, один рабочий каталог соответствует только одному репозиторию. Невозможно извлечь в один рабочий каталог какие-либо файлы из одного репозитория и какие-либо — из другого.
