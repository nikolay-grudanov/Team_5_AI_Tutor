Специальные файлы

<table>
  <tr>
    <th>Имя файла</th>
    <th>Описание</th>
  </tr>
  <tr>
    <td>~/.cshrc или ~/.tcshrc</td>
    <td>Выполняется при каждом запуске интерпретатора</td>
  </tr>
  <tr>
    <td>~/.login</td>
    <td>Выполняется интерпретатором после .cshrc при входе в систему</td>
  </tr>
  <tr>
    <td>~/.cshdirs</td>
    <td>Выполняется интерпретатором после .login при входе в систему (tcsh)</td>
  </tr>
  <tr>
    <td>~/.logout</td>
    <td>Выполняется при завершении сеанса работы с интерпретатором</td>
  </tr>
  <tr>
    <td>/etc/passwd</td>
    <td>Информация о домашних каталогах пользователей для расширения ~name</td>
  </tr>
</table>

Специальные символы в именах файлов

<table>
  <tr>
    <th>Символы</th>
    <th>Значение</th>
  </tr>
  <tr>
    <td>*</td>
    <td>Любая строка, пустая или состоящая из произвольного количества символов</td>
  </tr>
  <tr>
    <td>?</td>
    <td>Один произвольный символ</td>
  </tr>
  <tr>
    <td>[abc...]</td>
    <td>Любой из перечисленных в скобках символов; для задания интервала может использоваться дефис (например, a–z, A–Z, 0–9)</td>
  </tr>
  <tr>
    <td>{abc, xxx,...}</td>
    <td>Расширение каждой из перечисленных строк</td>
  </tr>
  <tr>
    <td>~</td>
    <td>Домашний каталог текущего пользователя</td>
  </tr>
  <tr>
    <td>~name</td>
    <td>Домашний каталог пользователя с регистрационным именем name</td>
  </tr>
</table>

Примеры

% ls new*        Вывести список файлов, имена которых начинаются на new.
% cat ch?         Соответствует файлам, имена которых имеют вид ch9,
                  но не ch10
% vi [D-R]*       Редактировать файлы, имена которых начинаются на любую букву от прописной D до R
% ls {ch,app}?     Расширить список и вывести список файлов ch1, ch2, app1, app2
% cd ~tom         Перейти в домашний каталог пользователя tom
