    <th>popd</th>
    <td><b>popd [options]</b><br>
Работа со стеком каталогов. По умолчанию удалить из вершины стека каталог и перейти (<b>cd</b>) в него.<br>
<b>Параметры</b><br>
<code>+n</code> Удалить из стека каталог с номером <i>n</i> (первый имеет порядковый номер 0), считая от вершины стека.<br>
<code>-n</code> Удалить из стека каталог с номером <i>n</i>, считая от основания стека (последний элемент имеет порядковый номер 0).
    </td>

    <th>printf</th>
    <td><b>printf string [arguments]</b><br>
Функция форматирования строки, аналогичная библиотечной функции <b>printf</b> языка С. В форматной строке распознаются стандартные ключи с символом процента, например <code>%i</code>. Escape-последовательности, такие как <code>\n</code>, могут включаться в форматную строку и распознаются автоматически. Если вы намереваетесь использовать их в аргументах, укажите в качестве строки <i>string</i> <code>%b</code>. Символы в аргументах можно экранировать (с целью использования результатов в качестве ввода других команд), используя в качестве строки <i>string</i> <code>%q</code>.
<br>
<b>Примеры</b>
<pre>
$ printf "Previous command: %i\n" "$(($HISTCMD-1))"
Previous command: 534
$ echo $PAGER
less -E
$ printf "%q\n" "\t$PAGER"
\tless\ -E
</pre>
Последняя команда может быть использована для записи настройки в файл, из которого она затем может быть прочитана и присвоена другому сценарию интерпретатора.
    </td>
  </tr>
