remove

remove
[ -f ]
[ -l | -R ]
[ file ... ]

Указать CVS, что файлы должны быть удалены из репозитория. Пока изменения не внесены в репозиторий, файлы реально не удаляются. Если вы передумали, используйте cvs add для восстановления файлов, которые были удалены таким образом.

Параметры -l и -R имеют тот же смысл, что и обычно. С командой remove может использоваться только один дополнительный параметр: -f. При его использовании файл удаляется сначала из рабочего каталога.
