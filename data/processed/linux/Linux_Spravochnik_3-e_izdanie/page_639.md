Метасимволы

Следующие символы имеют специальное значение только в шаблонах поиска:

<table>
  <tr>
    <th>Символ</th>
    <th>Значение</th>
  </tr>
  <tr>
    <td>.</td>
    <td>Соответствует любому одиночному символу, кроме символа новой строки (newline)</td>
  </tr>
  <tr>
    <td>*</td>
    <td>Соответствует любому количеству вхождений (или ни одного) непосредственно предшествующего символа. Предшествующий символ также может быть регулярным выражением, например, поскольку «.» (точка) означает любой символ, выражение .* означает найти любое количество любых символов (кроме символов конца строки)</td>
  </tr>
  <tr>
    <td>^</td>
    <td>Найти соответствие следующему регулярному выражению в начале строки</td>
  </tr>
  <tr>
    <td>$</td>
    <td>Найти соответствие следующему регулярному выражению в конце строки</td>
  </tr>
  <tr>
    <td>[ ]</td>
    <td>Найти любой из перечисленных символов. Диапазон символов можно задавать с помощью символа - (дефис). Диакритический символ ^, будучи первым символом среди перечисленных, меняет смысл выражения на противоположный — выполняется поиск символа, не входящего в указанный список. Дефис или закрывающая скобка (]) в качестве первого элемента списка интерпретируются как обычные символы. Все прочие специальные символы также считаются элементами списка</td>
  </tr>
  <tr>
    <td>[^ ]</td>
    <td>Найти любой символ, не относящийся к перечисленным</td>
  </tr>
  <tr>
    <td>\{n, m\}</td>
    <td>Найти определенное количество вхождений непосредственно предшествующего символа. Предшествующий символ может быть регулярным выражением. По выражению \{n\} выполняется поиск ровно n вхождений символа, по выражению \{n, \} — поиск по крайней мере n вхождений символа, выражение \{n, m\} задает диапазон для количества вхождений от n до m</td>
  </tr>
  <tr>
    <td>\{n, m\}</td>
    <td>Аналогично \{n, m\}. Метасимвол доступен по умолчанию в grep, а также в gawk при использовании ключа —Wre-interval</td>
  </tr>
</table>
