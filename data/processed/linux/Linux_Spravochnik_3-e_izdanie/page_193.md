    <td>
      <b>-v</b> Выдавать версии программ, запускаемых groff.<br>
      <b>-V</b> Не исполнять конвейерную последовательность команд, а выдать ее на стандартный вывод.<br>
      <b>-wname</b><br>
      Включить предупреждения вида name. Может быть задано произвольное число параметров -w. Виды предупреждений перечислены в руководстве по команде troff.<br>
      <b>-Wname</b><br>
      Отключить предупреждения вида name. Может быть задано произвольное число параметров -W. Виды предупреждений перечислены в руководстве по команде troff.<br>
      <b>-z</b> Подавить выдачу информации troff (кроме сообщений об ошибках).<br>
      <b>-Z</b> Не выполнять дополнительную обработку результатов, выдаваемых troff. По умолчанию соответствующая пост-обработка производится groff автоматически.
