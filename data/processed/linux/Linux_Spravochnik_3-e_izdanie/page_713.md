  </tr>
  <tr>
    <td></td>
    <td>Заменить соответствие шаблону pattern на replacement во всех указанных строках. Если используются шаблонные адреса, шаблон «//» представляет собой последний заданный адрес. Могут быть указаны следующие ключи:</td>
  </tr>
  <tr>
    <td>n</td>
    <td>Заменить n-ное вхождение соответствия шаблону /pattern/ в каждой из строк диапазона. n может быть любым числом от 1 до 512. По умолчанию n = 1.</td>
  </tr>
  <tr>
    <td>g</td>
    <td>Заменить все вхождения соответствий шаблону /pattern/ в каждой из строк диапазона, а не только первое из вхождений.</td>
  </tr>
  <tr>
    <td>p</td>
    <td>Отобразить строку, если произведена успешная замена. Если произведено несколько таких замен, будет отображено соответствующее количество копий строки.</td>
  </tr>
  <tr>
    <td>w file</td>
    <td>Записать строку в файл file, если была произведена успешная замена.</td>
