    <th>strfile</th>
    <td>
      <b>strfile [options] input_file [output_file]</b><br>
      <b>unstr [-c delimiter] input_file[.ext] [output_file]</b><br>
      <b>strfile</b> создает файл произвольного доступа для хранения строк. Исходный файл должен содержать группы строк, разделенные строкой, которая содержит единственный символ % (или любой другой заданный символ-разделитель). <b>strfile</b> создает конечный файл, содержащий структуру-заголовок и таблицу смещений каждой из групп строк, которые и позволяют организовать произвольный доступ к строкам. Конечный файл, если он не задан, получает имя <i>sourcefile.dat</i>. <b>unstr</b> обращает действие <b>strfile</b>, отображая строки, содержащиеся в исходном файле в том порядке, в каком они перечислены в заголовке файла данных. Конечным файлом по умолчанию считается стандартный вывод. Команда <b>unstr</b> также может применяться для глобальной замены символа-разделителя в исходном файле.<br>
      <b>Параметры</b><br>
      Из нижеперечисленных параметров только <b>-c</b> может использоваться с <b>unstr</b>. Все прочие параметры относятся только к <b>strfile</b>.
    </td>
