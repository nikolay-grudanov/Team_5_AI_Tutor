Параметры

-а, —append
Добавлять теги в уже существующий список.

-d, —defines
Включать строку для каждой из определяющих директив препроцессора.

-i <i>file</i>, —include=<i>file</i>
Добавить к файлу тегов примечание о том, что в дополнение к исходному файлу необходимо прочитать файл <i>file</i>.

-l <i>language</i>, —language=<i>language</i>
Файлы, перечисленные после этого параметра, содержат исходные тексты на языке <i>language</i>. При помощи параметра —h можно получить перечень узнаваемых языков и стандартных расширений файлов с текстами на этих языках.

-o <i>file</i>, —output=<i>file</i>
Запись результата в файл <i>file</i>.

-r <i>regexp</i>, —regex=<i>regexp</i>
Включать тег для каждой из строк, содержащей соответствие регулярному выражению <i>regexp</i>, из всех файлов, следующих после параметра.

-C, —c++
Считать, что файлы .c и .h содержат текст на языке C++, а не C.
