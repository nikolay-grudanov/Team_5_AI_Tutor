      <b>dir [remote-directory] [local-file]</b><br>
      Отобразить содержимое каталога (<i>remote-directory</i>) с удаленной машины и при необходимости поместить результат в локальный файл (<i>local-file</i>).       <b>form format</b><br>

      <b>get remote-file [local-file]</b><br>
      Получить файл (<i>remote-file</i>) с удаленной машины и сохранить его на локальной машине. Если не задано новое имя для сохраняемого файла (<i>local-file</i>), имя файла не меняется, подвергаясь лишь изменениям, предусмотренным командами case, ntrans и nmap. Если вместо локального файла задан параметр «—», то файл выводится на терминал.<br>
      <b>glob</b><br>
      Переключение режима расширения имен для команд mdelete, mget и mput. Если расширение отключено, имена файлов в аргументах команд воспринимаются буквально и не расширяются.<br>
      <b>hash</b><br>
      Переключение режима отображения символа # после каждого успешно переданного блока данных.<br>
      <b>help [command]</b><br>
      Вывести справку по использованию команды (<i>command</i>). help без параметров выводит список команд ftp.<br>
      <b>idle [seconds]</b><br>
      Получить (или установить) значение (<i>seconds</i>) таймера бездействия на удаленной машине. Команда без параметров отображает текущее значение таймера.<br>
