    <th>bison</th>
    <td>bison [options] file

Построение таблиц для последующего грамматического разбора на основе файла (file), содержащего описание контекстно-независимой грамматики. Результат записывается в файл file.c. Этот инструмент был назван по аналогии с yacc и очень во многом совместим с ним. Имена всех исходных файлов должны содержать суффикс .y, а имена получаемых файлов — исходный префикс имени. Все длинные имена параметров (предваряемые обычно символами «—») могут предваряться символом «+».

Параметры

<b>-b prefix, --file-prefix=prefix</b>
Для всех получаемых файлов использовать префикс prefix.

<b>-d, --defines</b>
Создать файл file.h, содержащий директивы #define, связывающие коды лексем bison с именами лексем, определяемыми пользователем.</td>
