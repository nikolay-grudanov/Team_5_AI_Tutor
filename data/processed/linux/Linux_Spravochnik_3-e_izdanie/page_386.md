Параметры

-а, --all
    Перечислить значения всех настроек.
-g   Вывести значения в шестнадцатеричной системе счисления.

Режимы управления

n    Установить скорость передачи терминала в n бод (например, 2400).

[-]clocal
    [Включение] отключение управления модемом.

[-]cread
    [Отключение] включение приемника.

csbits
    Установить размер символа равным bits, причем аргумент может принимать только значения 5, 6, 7 или 8.

[-]cstopb
    [Один] два стоповых бита на символ.

[-]hup
    [Не] обрывать соединение при последнем закрытии.

[-]hupcl
    То же самое, что и [-]hup.

ispeed n
    Установить скорость ввода терминала в n бод.

ospeed n
    Установить скорость вывода терминала в n бод.

[-]parenb
    [Отключить] включить генерацию четности и контроль по четности.

[-]parodd
    Использовать контроль по [четности] нечетности.

[-]crtsccts*
    [Отключить] включить квитирование по RTS/CTS.
