    <th>more</th>
    <td>
      <b>d, ^D</b><br>
      Прокрутить <i>lines</i> строк текста и переопределить размер прокрутки в <i>lines</i> строк. По умолчанию происходит прокрутка на одну строку.<br>
      <b>q, Q, INTERRUPT</b><br>
      Завершить работу.<br>
      <b>s</b><br>
      Пропустить одну строку текста.<br>
      <b>f</b><br>
      Пропустить одну экранную страницу текста.<br>
      <b>b, ^B</b><br>
      Прокрутка на одну экранную страницу назад.<br>
      <b>'</b><br>
      Вернуться к точке, из которой начался последний поиск.<br>
      <b>=</b><br>
      Отобразить номер текущей строки.<br>
      <i>/pattern</i><br>
      Поиск по шаблону <i>pattern</i>, переход к вхождению с номером <i>nit</i>, если задан аргумент.<br>
      <b>n</b><br>
      Повторить последний поиск, совершить переход к вхождению с номером <i>nit</i>, если задан аргумент.<br>
      <b>!cmd, ::!cmd</b><br>
      Выполнить команду <i>cmd</i> в интерпретаторе.<br>
      <b>v</b><br>
      Вызвать редактор <i>vi</i> для редактирования текущего файла, открыть файл на текущей строке.<br>
      <b>^L</b><br>
      Обновить экран.<br>
      <b>:n</b><br>
      Перейти к просмотру следующего файла.<br>
      <b>:p</b><br>
      Перейти к просмотру предыдущего файла.<br>
      <b>:f</b><br>
      Вывести имя текущего файла и номер строки.<br>
      <b>.</b><br>
      Повторно выполнить последнюю команду.<br>
      <b>Примеры</b><br>
      Постранично просматривать файл <i>file</i> в «чистом» режиме и отображать приглашения:<br>
      <code>more -cd file</code><br>
      Удалять подчеркивания из файла <i>doc</i> при отображении:<br>
      <code>nroff doc | more -u</code><br>
      Просмотр страниц руководства для команды <i>grep</i>; начать со слова «BUGS», удалять лишние пробелы:<br>
      <code>man grep | more +/BUGS -s</code>
