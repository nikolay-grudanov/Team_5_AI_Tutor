    <th>ispell</th>
    <td><b>ispell [options] [files]</b><br>
      Сравнение слов из указанных файлов с набором слов из словаря. Неопознанные слова отображаются в верхней части экрана в сопровождении возможных вариантов корректного написания; кроме того, возможно редактирование с помощью ряда команд.<br><br>
      <b>Параметры</b><br>
      <b>-b</b> Создать резервную копию исходного файла с именем <i>filename.bak</i>.<br>
      <b>-d file</b><br>
      Использовать файл <i>file</i> в качестве стандартного словаря.<br>
      <b>-m</b> Предлагать различные комбинации корня/аффикса.<br>
      <b>-n</b> Исходный файл является результатом работы nroff или troff.<br>
      <b>-p file</b><br>
      Файл <i>file</i> является дополнительным пользовательским словарем.<br>
      <b>-t</b> Исходный файл имеет формат TeX или LaTeX.<br>
      <b>-w chars</b><br>
      Добавить символы (<i>chars</i>) к набору допустимых в слове в дополнение к a–z, A–Z.<br>
      <b>-x</b> Не создавать резервную копию исходного файла.<br>
      <b>-B</b> Помимо обычных ошибок найти пропущенные пробелы («склейки» слов).<br>
      <b>-C</b> Не выдавать сообщение об ошибке при встрече склейки слов.<br>
      <b>-L number</b><br>
      Показывать указанное количество (<i>number</i>) строк контекста.<br>
      <b>-M</b> Вывести список диалоговых команд в нижней части экрана.<br>
      <b>-N</b> Подавлять вывод диалоговых команд.<br>
      <b>-P</b> Предлагать минимальное количество вариантов корня/аффикса.<br>
      <b>-S</b> Сортировать предлагаемые замены по их вероятной предпочтительности.
