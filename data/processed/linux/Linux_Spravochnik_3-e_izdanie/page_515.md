    <th>dselect</th>
    <td>
      <b>dselect [options][action]</b><br>
      Экранно-ориентированный интерфейс к dpkg. Основной пользовательский интерфейс для установки и управления пакетами. Информация по созданию пакетов приведена в описаниях команд dpkg и dpkg-deb.<br><br>
      <b>Действия</b><br>
      Если при запуске dselect в командной строке не было указано действие, отображается следующее меню:
      <ul>
        <li>* 0. [A]ccess Выбор метода установки.</li>
        <li>1. [U]pdate Обновление списка доступных пакетов, если возможно.</li>
        <li>2. [S]elect Выбор набора пакетов для установки, обновления или удаления.</li>
        <li>3. [I]nstall Установка и обновление пакетов.</li>
        <li>4. [C]onfig Настройка ненастроенных пакетов.</li>
        <li>5. [R]emove Удаление ненужных пакетов.</li>
        <li>6. [Q]uit Выход.</li>
      </ul>
      Звездочка (отображаемая здесь в первой строке) указывает на выбранный в данный момент пункт меню. Любой из пунктов меню может быть указан в командной строке в качестве действия (access, update, select, install, config, remove, quit), которое будет немедленно выполнено при старте. Например:
      <pre>% dselect access</pre>
      Если выбрать действие quit, то dselect завершит работу немедленно, не совершая никаких операций. Дополнительным действием, доступным в командной строке, является menu. Это действие эквивалентно выполнению команды без параметров.<br><br>
      <b>Параметры</b><br>
      <b>--admindir dir</b><br>
      Изменить каталог, в котором содержатся файлы с внутренними данными, на dir. По умолчанию это /var/lib/dpkg.
