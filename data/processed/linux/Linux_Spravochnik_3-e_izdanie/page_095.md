    <td>
      <b>--version</b><br>
      Отобразить информацию о версии программы и завершить работу.<br>
      <b>Примеры</b><br>
      При помощи команды find создать список файлов, имена которых заканчиваются на .old. Передать список срио:
      <pre>find . -name "*.old" - print | cpio -ocBv > /dev/rst8</pre>
      Извлечь из архива с ленты все файлы, имена которых содержат save (подкаталоги создаются по необходимости):
      <pre>cpio -icdv "*save*" < /dev/rst8</pre>
      Переместить дерево каталогов:
      <pre>find . -depth -print | cpio -padm /mydir</pre>

    <th>cpp</th>
    <td>
      <b>cpp [options] [ifile [ ofile ]]</b><br>
      GNU-версия препроцессора для языка C. Вызывается при первом проходе компиляции любой C-программы с помощью gcc. Результат работы является вводом для следующего прохода компилятора C, и обычно cpp вызывает gcc, закончив обработку. ifile и ofile являются, соответственно, именами исходного и получаемого файла для препроцессора; по умолчанию этими файлами являются стандартный ввод и стандартный вывод.<br>
      <b>Параметры</b><br>
      <b>-S</b> Символ $ в идентификаторах запрещен.<br>
      <b>-dM</b><br>
      Подавление нормального вывода. Отображаются последовательности макроопределений #define, использованных в исходном файле.<br>
      <b>-dD</b><br>
      Аналогично параметру -dM, но исключаются заранее заданные макроопределения и включаются результаты работы препроцессора.<br>
      <b>-idirafter dir</b><br>
      Искать заголовочные файлы в каталоге dir, если файл не найден ни в одном из каталогов с включаемыми файлами.
