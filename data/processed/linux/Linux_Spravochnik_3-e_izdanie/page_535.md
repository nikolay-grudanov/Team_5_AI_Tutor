Таблица 7.15. Символы перенаправления ввода/вывода

<table>
  <tr>
    <th>Символ</th>
    <th>Действие</th>
  </tr>
  <tr>
    <td>&gt; file</td>
    <td>Перенаправление стандартного потока вывода в файл file</td>
  </tr>
  <tr>
    <td>&lt; file</td>
    <td>Получение стандартного потока ввода из файла file</td>
  </tr>
  <tr>
    <td>cmd1 | cmd2</td>
    <td>Конвейер. Стандартный поток вывода команды cmd1 перенаправляется в стандартный поток ввода команды cmd2</td>
  </tr>
  <tr>
    <td>&gt;&gt;file</td>
    <td>Перенаправление стандартного потока вывода в файл file с добавлением данных к концу файла, если таковой уже существует</td>
  </tr>
  <tr>
    <td>&gt;| file</td>
    <td>Принудительное перенаправление стандартного потока вывода в файл file, даже в режиме работы noclobber</td>
  </tr>
  <tr>
    <td>n&gt;| file</td>
    <td>Перенаправление вывода файлового дескриптора n в файл file, даже в режиме работы noclobber</td>
  </tr>
  <tr>
    <td>&lt;&gt; file</td>
    <td>Использовать файл file и как стандартный поток ввода, и как стандартный поток вывода</td>
  </tr>
  <tr>
    <td>&lt;&lt; text</td>
    <td>Выполнять чтение из стандартного ввода до появления строки, идентичной строке text (строка может храниться в переменной окружения). Ввод обычно осуществляется с клавиатуры. Программы, с которыми чаще всего используется этот синтаксис, — cat, echo, ex и sed. Если вводимый текст text заключается в кавычки, то в нем не подставляются значения переменных, результаты работы команд и т. п.</td>
  </tr>
  <tr>
    <td>n&gt;file</td>
    <td>Перенаправить файловый дескриптор n в file</td>
  </tr>
  <tr>
    <td>n&lt;file</td>
    <td>Присвоить файлу file файловый дескриптор n</td>
  </tr>
  <tr>
    <td>&gt;&n</td>
    <td>Дублировать стандартный поток вывода в файловый дескриптор n</td>
  </tr>
  <tr>
    <td>&lt;&n</td>
    <td>Дублировать стандартный поток ввода из файлового дескриптора n</td>
  </tr>
  <tr>
    <td>&gt;&gt;file</td>
    <td>Перенаправить стандартный поток вывода и стандартный поток ошибок в файл file</td>
  </tr>
  <tr>
    <td>&lt;&-&</td>
    <td>Закрыть стандартный поток ввода</td>
  </tr>
  <tr>
    <td>&gt;&-&</td>
    <td>Закрыть стандартный поток вывода</td>
  </tr>
  <tr>
    <td>n&gt;&-&</td>
    <td>Закрыть вывод из файлового дескриптора n</td>
  </tr>
  <tr>
    <td>n&lt;&-&</td>
    <td>Закрыть ввод из файлового дескриптора n</td>
  </tr>
</table>
