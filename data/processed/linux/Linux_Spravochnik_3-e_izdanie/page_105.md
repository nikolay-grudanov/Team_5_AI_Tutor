    <th>ctags</th>
    <td>
      <b>-r regexp, --regex=regexp</b><br>
      Включать теги для строк в тексте, соответствующих регулярному выражению <i>regexp</i>. Применять ко всем файлам, следующим за параметром.<br>
      <b>-R, --no-regex</b><br>
      Не включать теги для строк в тексте, соответствующих регулярному выражению. Применять ко всем файлам, следующим за параметром.<br>
      <b>-t, --typedefs</b><br>
      Включать строки для определений типов с помощью typedefs.<br>
      <b>-u, --update</b><br>
      Обновить файл тегов, чтобы учесть новое местоположение функций (например, если функции перемещены в другой исходный файл). Старые теги удаляются, новые добавляются в конец файла.<br>
      <b>-v, --vgrind</b><br>
      Выдать на стандартный вывод распечатку (указатель) для каждой функции, исходный файл и номер страницы (одна страница содержит 64 строки).<br>
      <b>-w, --no-warn</b><br>
      Не выводить предупреждения.<br>
      <b>-x, --cxref</b><br>
      Выводить распечатку каждой функции, номер строки, исходный файл и контекст.<br>
      <b>-B, --backward-search</b><br>
      Поиск тегов в файлах от конца к началу.<br>
      <b>-C, --c++</b><br>
      Считать, что файлы .c и .h содержат текст на языке C++, а не C.<br>
      <b>-H, -h, --help</b><br>
      Вывести справку по использованию и завершить работу.<br>
      <b>-S, --ignore-indentation</b><br>
      Обычно ctags использует отступы в тексте для разбора файла тегов; данный параметр делает работу менее зависимой от отступов.<br>
      <b>-T, --typedefs-and-c++</b><br>
      Включать строки тегов для определений typedef, struct, enum, union и функций-членов C++.
