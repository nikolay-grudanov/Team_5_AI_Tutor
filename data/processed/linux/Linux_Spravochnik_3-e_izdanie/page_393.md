    <th>sum</th>
    <td>
      <b>sum [options] file</b><br>
      Подсчет и отображение контрольной суммы файла и количества блоков (по 1 Кбайт). Команда полезна для проверки целостности при передаче данных.<br>
      <b>Параметры</b><br>
      <b>-r</b> Режим по умолчанию. Использовать BSD-версию алгоритма подсчета контрольной суммы.<br>
      <b>-s, --sysv</b><br>
      Применить альтернативный алгоритм подсчета контрольной суммы (из System V). Размер блока — 512 байт.<br>
      <b>--help</b><br>
      Отобразить информацию по использованию sum и завершить работу.<br>
      <b>--version</b><br>
      Вывести номер версии sum и завершить работу.
    </td>

    <th>swapdev</th>
    <td>
      <b>swapdev [option][image[swapdevice[offset]]]</b><br>
      Команда системного администрирования. Команда без аргументов отображает информацию по использованию устройства подкачки. Если задан только образ ядра (image), то отображается информация из этого образа. Для изменения информации необходимо указать новое устройство подкачки (swapdevice). Также возможно задание смещения (offset) для изменения информации в образе ядра. Обратите внимание, что rdev -s является синонимом swapdev.<br>
      <b>Параметр</b><br>
      <b>-o offset</b><br>
      Синоним задания смещения в качестве аргумента.
