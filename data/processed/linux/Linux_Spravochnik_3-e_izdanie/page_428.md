<ul>
<li><b>ul [options] [names]</b>
<p>Преобразовать символы подчеркивания в подчеркивания символов. Корректная обработка операции зависит от конкретного типа терминала. Некоторые из терминалов не поддерживают подчеркивание символов.</p>
<h3>Параметры</h3>
<ul>
<li><b>-i</b> Преобразовать символ «-» (дефис) в отдельной строке в подчеркивание символов вместо преобразования символа _ (символ подчеркивания).</li>
<li><b>-t terminal-type</b>
<p>Указать тип терминала. По умолчанию тип терминала — TERM.</p>
</li>
</ul>
</li>
<li><b>umount [options] [special-device/directory]</b>
<p>Команда системного администрирования. Размонтировать файловую систему. umount уведомляет систему, что удаляемая файловая структура, которая была подмонтирована на устройстве <i>special-device</i>, будет впредь недоступна. В качестве аргумента umount может быть указана точка монтирования. Процессы ввода/вывода для файловой системы завершаются, и файловая структура отмечается как корректная.</p>
<h3>Параметры</h3>
<ul>
<li><b>-a</b> Размонтировать все файловые системы, перечисленные в /etc/mtab.</li>
<li><b>-n</b> Размонтировать, но не записывать изменения в /etc/mtab.</li>
<li><b>-t type</b>
<p>Размонтировать только файловые системы указанного типа.</p>
</li>
</ul>
</li>
<li><b>uname [options]</b>
<p>Вывод информации об операционной системе и архитектуре машины. Команда без параметров отображает только имя операционной системы (Linux).</p>
</li>
</ul>
