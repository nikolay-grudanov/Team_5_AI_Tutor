Сохранение и завершение работы

Сохранение файла — это запись последних изменений и обновление времени изменения файла.

<table>
  <tr>
    <th>Команда</th>
    <th>Действие</th>
  </tr>
  <tr>
    <td>ZZ</td>
    <td>Завершить работу с vi, сохранив файл, если он был изменен в ходе сеанса работы</td>
  </tr>
  <tr>
    <td>:x</td>
    <td>Идентично ZZ</td>
  </tr>
  <tr>
    <td>:wq</td>
    <td>Сохранить файл и закончить его редактирование</td>
  </tr>
  <tr>
    <td>:w</td>
    <td>Сохранить файл</td>
  </tr>
  <tr>
    <td>:w file</td>
    <td>Сохранить копию в файл file</td>
  </tr>
  <tr>
    <td>:n1,n2w file</td>
    <td>Записать строки с n1 по n2 в файл file</td>
  </tr>
  <tr>
    <td>:n1,n2w >> file</td>
    <td>Добавить строки с n1 по n2 в существующий файл file</td>
  </tr>
  <tr>
    <td>:w!</td>
    <td>Принудительно записать файл</td>
  </tr>
  <tr>
    <td>:w! file</td>
    <td>Перезаписать файл содержимым текущего буфера редактирования</td>
  </tr>
  <tr>
    <td>:w % .new</td>
    <td>Записать текущий буфер с именем file под именем file.new</td>
  </tr>
  <tr>
    <td>:q</td>
    <td>Завершить редактирование файла</td>
  </tr>
  <tr>
    <td>:q!</td>
    <td>Завершить редактирование файла (без сохранения изменений)</td>
  </tr>
  <tr>
    <td>Q</td>
    <td>Завершить работу с vi и вызвать ex</td>
  </tr>
  <tr>
    <td>:vi</td>
    <td>Вернуться к работе с vi после ввода команды Q</td>
  </tr>
  <tr>
    <td>%</td>
    <td>Имя текущего файла</td>
  </tr>
  <tr>
    <td>#</td>
    <td>Альтернативное имя файла</td>
  </tr>
