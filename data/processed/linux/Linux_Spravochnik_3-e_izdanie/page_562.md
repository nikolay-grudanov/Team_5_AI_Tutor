Встроенные команды

Параметры

+n Прокрутка элементов стека таким образом, чтобы на вершине оказался элемент с номером n (первый элемент имеет порядковый номер 0).

-n Прокрутка элементов стека таким образом, чтобы на вершине оказался n-ный элемент от основания стека (последний элемент имеет порядковый номер 0).

pwd [-P]

Отобразить абсолютный путь текущего рабочего каталога. Если установлен встроенный режим -P, путь не будет содержать символические ссылки.

read [options] variable1 [variable2...]

Прочитать одну строку со стандартного потока ввода и присвоить слова строки (определяемые по IFS) соответствующим переменным, а все лишние слова добавить к последней переменной. Если задана только одна переменная, она станет равной всей прочитанной строке. Код возврата — 0, если не встретился символ EOF; это различие полезно для программы циклов для файлов ввода. Если имена переменных опущены, вся строка читается в переменную окружения REPLY.

Параметры

-a var
Прочитать все слова в переменную var, которая является массивом.

-d char
Прекратить чтение по указанному символу, а не по символу новой строки.

-e Построчное редактирование и журнал команд активны во время ввода.

-n nmt
Читать только nmt символов строки.

-p string
Отображать диалоговое приглашение в виде строки string перед чтением каждой из строк ввода.
