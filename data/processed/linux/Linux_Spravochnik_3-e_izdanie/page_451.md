    <td>
      <b>ypmatch [options] key ... mname</b><br>
      Команда NFS/NIS. Вывести значения перечисленных ключей из NIS-карты, заданной именем <i>mname</i> (имя или псевдоним карты).
      <br>
      <b>Параметры</b><br>
      <b>-d domain</b><br>
      Указать домен для использования вместо домена по умолчанию.<br>
      <b>-k</b> Для каждого значения Вывести собственно ключ с символом двоеточия (:).<br>
      <b>-t</b> Не переводить псевдоним в имя карты.<br>
      <b>-x</b> Отобразить таблицу псевдонимов, в которой перечисляются известные псевдонимы (<i>mnames</i>) и имена связанных карт. Аргумент <i>mname</i> при этом не требуется.
