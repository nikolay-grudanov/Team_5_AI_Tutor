    <th>umask</th>
    <td>
      <b>umask [nnn]</b><br>
      <b>umask [–p][–S]</b><br>
      Отобразить или установить значение маски прав доступа для создаваемого файла (в восьмеричной системе счисления). Маска определяет, какие права доступа отсутствуют. (Так, результатом выполнения команды umask 002 будут права доступа rw–rw–r––).
      <b>Параметры</b>
      <ul>
        <li><b>–p</b> Отобразить значение маски в команде umask, чтобы пользователь мог прочитать ее и выполнить команду.</li>
        <li><b>–S</b> Отобразить значение umask в символьном виде, а не в виде восьмеричного числа.</li>
      </ul>
    </td>
