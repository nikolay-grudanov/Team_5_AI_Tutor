    <th>rlogind</th>
    <td>
      <b>rlogind [options]</b><br>
      Команда TCP/IP. Сервер программы <b>rlogin</b>, обеспечивающий возможность удаленного входа в систему с определением подлинности пользователей на основе выделенного номера порта и списка узлов, которым разрешен доступ. <b>rlogind</b> выполняется сервером <b>inetd</b> по запросу на удаленный вход в систему и работает по следующей схеме:
      <ul>
        <li>Сервер проверяет исходный порт клиента. Если порт лежит вне диапазона 0–023, сервер завершает соединение.</li>
        <li>Сервер проверяет исходный адрес клиента и выясняет соответствующее имя узла. Если имя узла не может быть определено, используется IP-адрес узла в записи через точку.</li>
      </ul>
      Процесс регистрации (login) передает скорость терминала клиента и тип терминала из переменной окружения <b>TERM</b>.

      <b>Параметры</b>
      <ul>
        <li><b>-a</b> Проверять имя узла.</li>
        <li><b>-l</b> Не авторизовать узлы по файлу <i>rhosts</i>, не принадлежащему пользователю root.</li>
        <li><b>-n</b> Подавлять сообщения keep-alive (сохранение соединения).</li>
      </ul>
    </td>
