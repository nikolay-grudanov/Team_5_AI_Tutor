    <th>gcc</th>
    <td>
      <b>-traditional</b><br>
      Поведение, максимально близкое к поведению традиционного компилятора C.<br>
      <b>-traditional-cpp</b><br>
      Поведение препроцессора, соответствующее традиционному языку C.<br>
      <b>-trigraphs</b><br>
      Включить поддержку триграфов.<br>
      <b>-u symbol</b><br>
      Предписание редактору связей искать имя <i>symbol</i> в библиотеках и связывать с библиотекой программу, если результат поиска положительный.<br>
      <b>-undef</b><br>
      Определить только константы, требуемые стандартом языка, а не системно-зависимые константы вроде unix.<br>
      <b>-v</b><br>
      Режим подробной диагностики. Команды отображаются по мере выполнения. Также отображается номер версии gcc и номер версии препроцессора.<br>
      <b>-w</b><br>
      Подавить вывод предупреждений.<br>
      <b>-x language</b><br>
      Считать, что исходные файлы содержат текст на языке <i>language</i>. Аргумент может быть равен c, objective-c, c-header, c++, cpp-output, assembler или assembler-with-cpp. Значение none является предписанием для автоматического распознавания.<br>
      <b>-Aquestion(answer)</b><br>
      Препроцессор, встретив условную директиву <i>question</i>, например #if <i>question</i>, добавляет в ответ указание <i>answer</i>. Отключение стандартных указаний производится параметром <b>-A-</b>.<br>
      <b>-Bpath</b><br>
      Указать каталог <i>path</i>, в котором расположены файлы компилятора.<br>
      <b>-C</b><br>
      Указание препроцессору сохранять комментарии. Имеет смысл использовать только с параметром <b>-E</b>.<br>
      <b>-Dname[=def]</b><br>
      Определить имя <i>name</i> как имеющее значение <i>def</i> (по аналогии с директивой #define). Если значение (=<i>def</i>) опущено, <i>name</i> получает значение 1. <b>-D</b> имеет меньший приоритет, чем <b>-U</b>.
