Встроенные команды csh и tcsh

Ограничения
Ограничение — это число, за которым может следовать символ (определятель единиц).
Для ресурса cputime: \( nh \) (n часов)
\( nm \) (n минут)
\( mm:ss \) (минуты и секунды)
Для прочих ресурсов: \( nk \) (n Кбайт, по умолчанию)
\( nm \) (n Мбайт)

log
Только для tcsh. Список пользователей, за которыми ведется наблюдение, определяется переменной watch. Отобразить список пользователей, работающих с системой в настоящий момент. Если в качестве параметра указан символ «-», сбросить значения переменных окружения, как если бы пользователь только что зарегистрировался в системе (в новой группе).

login [user|-p]
Для пользователя user заменить интерпретатор по умолчанию на /bin/login. Параметр -p указывается для сохранения параметров окружения.

logout
Завершить сеанс работы с интерпретатором.

ls-F [options] [files]
Только для tcsh. Быстрая альтернатива ls -F. При запуске с любыми дополнительными параметрами выполняется ls.

newgrp [-][group]
Только для tcsh. Изменить идентификатор группы пользователя на указанный либо, если идентификатор не задан, на идентификатор исходной группы. Если в качестве параметра указан символ «-», сбросить значения переменных окружения, как если бы пользователь только что зарегистрировался в системе (в новой группе).
