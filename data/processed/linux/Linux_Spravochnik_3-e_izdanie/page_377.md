    <th>sort</th>
    <td><b>Примеры</b><br>
Перечислить файлы в порядке убывания числа строк в них:<br>
<code>wc -l * | sort -r</code><br>
Расположить список слов в алфавитном порядке, удалив дубликаты, и отобразить количество вхождений каждого слова:<br>
<code>sort -fd wordlist | uniq -c</code><br>
Отсортировать файл с паролями по значению в третьем поле (идентификатор пользователя):<br>
<code>sort +2n -t: /etc/passwd</code>

    <th>split</th>
    <td><b>split [option][infile][outfile]</b><br>
Разбить файл <i>infile</i> на сегменты равного размера. <i>infile</i> не изменяется, а результаты записываются в файл <i>outfileaa</i>, <i>outfileab</i> и т. д. (По умолчанию — хаа, хаб и т. д.). Если имя исходного файла опущено или указан символ «—» (дефис), происходит чтение со стандартного ввода. См. также csplit.<br>
<b>Параметры</b><br>
<b>-n, -l n, --lines=n</b><br>
Разделить файл <i>infile</i> на сегменты по <i>n</i> строк (по умолчанию <i>n=1000</i>).<br>
<b>-b n[bkm], --bytes=n[bkm]</b><br>
Разбить файл <i>infile</i> на сегменты из <i>n</i> байт. Могут быть заданы альтернативные размеры блоков:<br>
<b>b</b> 512 байт.<br>
<b>k</b> 1 Кбайт.<br>
<b>m</b> 1 Мбайт.<br>
<b>-C bytes[bkm], --line-bytes=bytes[bkm]</b><br>
Ограничить размер файла указанным числом байт; заполнять файл полными строками.<br>
<b>--help</b><br>
Отобразить справку по использованию и завершить работу.<br>
<b>--verbose</b><br>
Режим диагностики; вывести сообщение при создании секции.
