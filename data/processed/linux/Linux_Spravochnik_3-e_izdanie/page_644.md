Введение

Несмотря на то что Emacs не является частью Linux, этот текстовый редактор присутствует во многих Unix-системах в качестве распространенной альтернативы редактору vi. Существует несколько вариантов Emacs. Здесь рассматривается GNU Emacs, предоставляемый фондом FSF (Free Software Foundation) (Кембридж, Массачусетс). За более подробной информацией обращайтесь к выпущенной издательством O’Reilly книге «Learning GNU Emacs» (Изучаем GNU Emacs) Дебры Кэмерон (Debra Cameron), Билла Розенблатта (Bill Rosenblatt) и Эрика Рэймонда (Eric Raymond).

Для того чтобы начать работу с Emacs, наберите:

emacs [file]
