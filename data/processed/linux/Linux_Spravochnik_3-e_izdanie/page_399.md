    <th>tar</th>
    <td>параметры с длинными именами (например, —modification-time). В этом случае точный синтаксис выглядит следующим образом:

    tar --long-option -function-options files

    К примеру:

    tar --modification-time -xvf tarfile.tar

    <b>Функциональные параметры</b>

    Можно использовать только один из этих параметров, и он обязательно должен предшествовать всем остальным:

    <b>-c, --create</b>
    Создать новый архив.

    <b>-d, --compare</b>
    Сравнить файлы, хранимые в архиве tarfile, с файлами other-files. Сообщать обо всех различиях: отсутствующих файлах, различных размерах, различных атрибутах файла (таких как права доступа и время изменения).

    <b>-r, --append</b>
    Добавить файлы other-files в конец существующего архива.

    <b>-t, --list</b>
    Перечислить имена файлов other-files, если они содержатся в архиве (при отсутствии аргумента other-files отображаются имена всех файлов архива).

    <b>-u, --update</b>
    Добавить файлы, которых нет в архиве, или измененные файлы.

    <b>-x, --extract, --get</b>
    Извлечь из архива файлы other-files (либо все, если аргумент опущен).

    <b>-A, --catenate, --concatenate</b>
    Добавить второй tar-файл в конец первого.

    <b>Параметры</b>
    n Выбор устройства n, причем n лежит в диапазоне 0–9999. Значение по умолчанию находится в файле /etc/default/tar.</td>
