Пользовательские команды

Программа-клиент CVS предоставляет пользователю команды, приведенные в табл. 14.20.

Таблица 14.20. Пользовательские команды

<table>
  <tr>
    <th>Команда</th>
    <th>Описание</th>
  </tr>
  <tr>
    <td>ad<br>add<br>new</td>
    <td>Указать CVS, что файлы/каталоги должны быть добавлены к репозиторию</td>
  </tr>
  <tr>
    <td>ann<br>annotate</td>
    <td>Вывести содержимое основной версии файла с номером версии, именем пользователя и датой последнего изменения для каждой строки</td>
  </tr>
  <tr>
    <td>checkout<br>co<br>get</td>
    <td>Создать рабочий каталог для модуля</td>
  </tr>
  <tr>
    <td>ci<br>com<br>commit</td>
    <td>Поместить изменения рабочего каталога в репозиторий</td>
  </tr>
  <tr>
    <td>di<br>dif<br>diff</td>
    <td>Определить различия между версиями файла</td>
  </tr>
  <tr>
    <td>edit</td>
    <td>Подготовка к редактированию файлов. Используется для улучшения координации усилий разработчиков</td>
  </tr>
  <tr>
    <td>editors</td>
    <td>Вывести список пользователей, работающих с файлами. Используется для улучшения координации усилий разработчиков</td>
  </tr>
  <tr>
    <td>ex<br>exp<br>export</td>
    <td>Извлечь модуль, но не создавать рабочую копию</td>
  </tr>
  <tr>
    <td>help</td>
    <td>Получить справку</td>
  </tr>
  <tr>
    <td>hi<br>his<br>history</td>
    <td>Отобразить информацию для файлов из журнала</td>
  </tr>
  <tr>
    <td>im<br>imp<br>import</td>
    <td>Импортировать новые модули в репозиторий</td>
  </tr>
  <tr>
    <td>lgn<br>login<br>logon</td>
    <td>Зарегистрироваться (закэшировать пароль) для работы на удаленном сервере CVS</td>
  </tr>
</table>
