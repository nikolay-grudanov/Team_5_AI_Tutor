    <td>
      <b>-T template</b><br>
      Имя основного файла шаблонов, используемого срр. Этот файл, как правило, находится в каталоге, который задан параметром -I. Имя файла по умолчанию — Imake.tpl.<br>
      <b>-v</b> Вывести командную строку срр, по которой будет создан Makefile.<br>
      <b>Инструменты</b><br>
      Далее следует список инструментов, используемых вместе с imake:<br>
      <b>makedepend [options] files</b><br>
      Создать зависимости для заголовочных файлов в файлах сборки Makefiles. makedepend выполняет последовательное чтение исходных файлов и разбор директив #include, #define, #undef, #ifdef, #ifndef, #endif, #if и #else, для того чтобы определить, какие директивы #include будут использованы при компиляции. makedepend определяет зависимости и записывает их в Makefile. Таким образом make узнает, какие из объектных файлов изменились и должны быть перекомпилированы. Параметры makedepend:
      <ul>
        <li><b>-- options --</b><br>
          Игнорировать все неопознанные параметры, следующие за двойным дефисом. Второй двойной дефис отменяет действие предыдущего. Опознанные параметры между дефисами обрабатываются как обычно.
        </li>
        <li><b>-a</b> Добавлять новые зависимости к уже существующим, а не заменять их.</li>
        <li><b>-f filename</b><br>
          Записывать зависимости в файл filename, а не в Makefile.
        </li>
        <li><b>-m</b> Вывести предупреждение о множественном включении файла.</li>
        <li><b>-s string</b><br>
          Использовать строку string вместо строки «#DO NOT DELETE THIS LINE – make depend depends on it» в качестве разделителя внутри файла.
        </li>
        <li><b>-v</b> Диагностика. Перечислить все файлы, включаемые в основной исходный файл.</li>
      </ul>
