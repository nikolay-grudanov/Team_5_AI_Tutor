    <th>apt-cdrom</th>
    <td>
      <b>-a, --thorough</b><br>
      Тщательный поиск пакетов. Может требоваться для некоторых старых компакт-дисков с Debian.<br>
      <b>-c, --config-file</b><br>
      Указать файл настройки, который будет прочитан после стандартного файла настройки.<br>
      <b>-d, --cdrom</b><br>
      Указать точку монтирования компакт-диска, которая должна присутствовать в файле /etc/fstab. Параметр настройки — Acqiv::cdrom::mount.<br>
      <b>-f, --fast</b><br>
      Быстрое копирование. Считать, что все файлы корректны и не требуют проверки. Используйте этот параметр только в том случае, если ранее не наблюдалось ошибок при работе с этим диском. Параметр настройки — APT::CDROM::Fast.<br>
      <b>-h, --help</b><br>
      Отобразить справку и завершить работу.<br>
      <b>-m, --no-mount</b><br>
      Не производить монтирование и размонтирование устройства. Параметр настройки — APT::CDROM::NoMount.<br>
      <b>-n, --just-print, --recon, --no-act</b><br>
      Проверить все, но не вносить никаких изменений. Параметр настройки — APT::CDROM::NoAct.<br>
      <b>-o, --option</b><br>
      Установить параметр настройки. Формат команды: -o group::tool=option (например APT::CDROM=Fast).<br>
      <b>-r, --rename</b><br>
      Запрашивать новую метку для переименования диска. Параметр настройки — APT::CDROM::Rename.<br>
      <b>-v, --version</b><br>
      Отобразить информацию о версии и завершить работу.
    </td>
