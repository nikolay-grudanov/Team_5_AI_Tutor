    <th>diff</th>
    <td>
      <b>diff [options] [diroptions] file1 file2</b><br>
      Сравнение двух текстовых файлов. <b>diff</b> отображает информацию о несовпадающих строках в файлах <i>file1</i> и <i>file2</i>. Результат состоит из соответствующих строк пары файлов, причем строки файла <i>file1</i> маркируются символом <>, а строки файла <i>file2</i> — символом <>. Строки предваряются командой редактора ed (a, c или d), которая необходима для преобразования файла <i>file1</i> в <i>file2</i>. Если задан параметр «-» вместо одного из имен файлов, то происходит чтение со стандартного ввода. Если одно из имен является именем каталога, diff обращается к тому файлу в заданном каталоге, который соответствует второму параметру (например, <b>diff my_dir junk</b> эквивалентно <b>diff my_dir/junk junk</b>). Если оба аргумента являются именами каталогов, diff обрабатывает попарно файлы, имеющие совпадающие имена (например, <i>olddir/program</i> и <i>newdir/program</i>). Кроме того, <b>diff</b> перечисляет уникальные имена для каждого каталога, а также подкаталоги, общие для обоих каталогов. См. также команду <b>cmp</b>.
