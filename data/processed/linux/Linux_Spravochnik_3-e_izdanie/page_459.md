    <th>zic</th>
    <td><b>SAVE</b><br>
      Количество времени, добавляемое к местному стандартному. Форматируется так же, как AT, но без суффиксов.<br>
      <b>LETTERS</b><br>
      Буквы, используемые для сокращенного указания часового пояса (например, S для EST). Если сокращение отсутствует, используйте символ «—».<br>
      <b>Поля строк часовых поясов</b><br>
      Формат строки часового пояса следующий:<br>
      Zone NAME GMTOFF RULES/SAVE FORMAT [UNTIL]<br>
      <b>NAME</b><br>
      Имя часового пояса.<br>
      <b>GMTOFF</b><br>
      Разница в часах с поясом GMT (среднее время по Гринвичу). Форматируется так же, как AT. Отрицательное время вычитается из GMT; по умолчанию время добавляется.<br>
      <b>RULES/SAVE</b><br>
      Либо имя правила для данного часового пояса, либо количество времени, добавляемого к местному стандартному. Символ «—» является указанием на часовой пояс для местного стандартного времени.<br>
      <b>FORMAT</b><br>
      Формат сокращений для часовых поясов. Переменная часть определяется параметром %s.<br>
      <b>UNTIL</b><br>
      Дата, при наступлении которой изменяется правило для часового пояса. В следующей строке должна определяться новая информация часового пояса, т. е. строка «Zone» и поле NAME должны быть опущены.<br>
      <b>Поля строк связей</b><br>
      Формат строки связей:<br>
      Link LINK-FROM LINK-TO<br>
      <b>LINK-FROM</b><br>
      Имя связываемого часового пояса.<br>
      <b>LINK-TO</b><br>
      Альтернативное имя для часового пояса, указанного в LINK-FROM.</td>
