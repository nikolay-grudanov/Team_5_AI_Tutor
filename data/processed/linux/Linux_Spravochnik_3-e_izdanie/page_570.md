Встроенные команды

-Н file
    Файл был изменен со времени последнего доступа к нему.

-О file
    Файл существует и его владелец является эффективным владельцем процесса.

-S file
    Файл является сокетом (socket).

f1 -ef f2
    Файлы f1 и f2 связаны (ссылаются на один и тот же файл).

f1 -nt f2
    Файл f1 новее файла f2.

f1 -ot f2
    Файл f1 старше файла f2.

Строковые условия

-n s1
    Строка s1 имеет ненулевую длину.

-o s1
    Установлен режим интерпретатора s1. Режимы интерпретатора приведены в описании встроенной команды set.

-z s1
    Строка s1 имеет нулевую длину.

s1 = s2
    Строки s1 и s2 идентичны.

s1 == s2
    Строки s1 и s2 идентичны.

s1 != s2
    Строки s1 и s2 не идентичны.

s1 < s2
    Строка s1 меньше строки s2 по алфавиту (либо по другому активному критерию сортировки). По умолчанию производится посимвольное сравнение в рамках таблицы символов ASCII.

s1 > s2
    Строка s1 больше строки s2 по алфавиту (либо по другому активному критерию сортировки).
