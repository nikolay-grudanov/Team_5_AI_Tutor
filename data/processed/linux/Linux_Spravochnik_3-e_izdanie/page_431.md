    <th> uniq </th>
    <td>
      <b>-w n, --check-chars=n</b><br>
      Сравнивать только n первых символов строк (отсчет начинается после пропуска указанного количества полей и символов).
      <br>
      <b>--help</b><br>
      Вывести справку по использованию и завершить работу.
      <br>
      <b>--version</b><br>
      Отобразить информацию о номере версии и завершить работу.
      <br>
      <b>Примеры</b><br>
      Записать по копии каждой из строк файла list в файл list.new:
      <br>
      uniq list list.new
      <br>
      Показать имена, имеющие более одного вхождения:
      <br>
      sort names | uniq -d
    </td>

    <th> unshar </th>
    <td>
      <b>unshar [options] [files]</b><br>
      Распаковать архив интерпретатора (shar-файл). unshar сканирует почтовые сообщения в поисках начала shar-файла. Затем архив передается в копию интерпретатора для распаковки. Команда принимает множественные имена файлов. Если файлы не указаны, выполняется чтение со стандартного ввода.
      <br>
      <b>Параметры</b><br>
      <b>-c, --overwrite</b><br>
      Разрешить перезапись существующих файлов.
      <br>
      <b>-d directory, --directory=directory</b><br>
      Перейти в указанный каталог, прежде чем распаковывать файлы.
      <br>
      <b>-e, --exit-0</b><br>
      Последовательно распаковать множественные архивы, содержащиеся в одном файле; для различения архивов используется тот факт, что многие архивы shar завершаются командой exit 0 в начале строки. (Параметр -e по существу эквивалентен -E «exit 0».)
    </td>
