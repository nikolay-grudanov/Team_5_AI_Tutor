Формы команд

<table>
  <tr>
    <th>Команда</th>
    <th>Действие</th>
  </tr>
  <tr>
    <td>cmd &</td>
    <td>Выполнить команду в фоновом режиме</td>
  </tr>
  <tr>
    <td>cmd1 ; cmd2</td>
    <td>Последовательность команд; выполнить команды, перечисленные в строке</td>
  </tr>
  <tr>
    <td>(cmd1 ; cmd2)</td>
    <td>«Дополнительный интерпретатор»; считать команды cmd1 и cmd2 одной группой команд</td>
  </tr>
  <tr>
    <td>cmd1 | cmd2</td>
    <td>Конвейер; использовать вывод команды cmd1 в качестве ввода команды cmd2</td>
  </tr>
  <tr>
    <td>cmd1 `cmd2`</td>
    <td>Подстановка команд; результат работы команды cmd2 передается в качестве аргументов команде cmd1</td>
  </tr>
  <tr>
    <td>cmd1 || cmd2</td>
    <td>Логическое ИЛИ; команда cmd2 выполняется только тогда, когда выполнение cmd1 завершилось неудачей</td>
  </tr>
  <tr>
    <td>cmd1 && cmd2</td>
    <td>Логическое И; команда cmd2 выполняется только тогда, когда успешно выполнилась команда cmd1</td>
  </tr>
</table>
