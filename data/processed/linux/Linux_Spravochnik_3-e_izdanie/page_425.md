    <td>
      <b>-p port</b><br>
      Установить основной адрес UDP-порта, который используется для отправки пробных пакетов. По умолчанию равно 33434 (десятичное).
      <br>
      <b>-q n</b><br>
      Установить количество пробных пакетов для каждого значения time-to-live в n. По умолчанию равно 3.
      <br>
      <b>-r</b><br>
      Посылать пакеты напрямую адресату в смежной сети в обход обычных таблиц маршрутизации.
      <br>
      <b>-s src_addr</b><br>
      Использовать указанный адрес в качестве исходного IP-адреса, добавляемого во все исходящие пробные пакеты.
      <br>
      <b>-t tos</b><br>
      Установить значение типа сервиса (type-of-service) в исходящих пакетах (по умолчанию — 0). Значение должно быть десятичным целым из диапазона 0—255.
      <br>
      <b>-v</b><br>
      Диагностика — перечислить получаемые ICMP-пакеты (кроме TIME_EXCEEDED и PORT_UNREACHABLE).
      <br>
      <b>-w wait</b><br>
      Установить время ожидания реакции на отправку пробного пакета в wait секунд (по умолчанию равно 3).
