Пример использования условного оператора:

```rust
let x = 5;

if x > 10 {
    println!("x больше 10");
} else {
    println!("x меньше или равно 10");
}
```

Пример использования цикла:

```rust
let mut i = 0;

loop {
    println!("i = {}", i);
    i += 1;
    if i >= 5 {
        break;
    }
}
```
