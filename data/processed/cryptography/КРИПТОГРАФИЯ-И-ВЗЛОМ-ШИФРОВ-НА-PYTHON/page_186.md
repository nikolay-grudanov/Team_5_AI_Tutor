В этом примере функция os.path.exists() подтверждает, что файл calc.exe существует в Windows. Разумеется, вы получите показанные результаты только в том случае, если Python запущен в Windows. Не забудьте экранировать символ обратной косой черты в обозначении пути к файлу в Windows, поместив перед ним еще один такой же символ. Если полный путь к файлу не задан (1), Python будет искать его в текущем каталоге. В случае интерактивной оболочки IDLE это будет папка, в которой установлен Python.
