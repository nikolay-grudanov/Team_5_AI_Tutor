2. Так же легко доказать, что \( 2^n = C_n^0 + C_n^1 + C_n^2 + \ldots + C_n^n \), достаточно положить в формуле (21) \( x = 1 \).

3. Если положить в равенстве (21) \( x = -1 \), то получим, что
\[
0 = C_n^0 - C_n^1 + C_n^2 - C_n^3 + \ldots + (-1)^k C_n^k + \ldots + (-1)^n C_n^n.
\]
Иными словами, сумма значений \( C_n^k \) с четными \( k \) равна сумме значений \( C_n^k \) с нечетными \( k \):
\[
C_n^0 + C_n^2 + C_n^4 + \ldots + C_n^{2m} + \ldots = C_n^1 + C_n^3 + \ldots + C_n^{2m+1} + \ldots
\]
Обе суммы конечны и обрываются, когда \( 2m \) (соответственно \( 2m + 1 \)) станет больше \( n \).

4. Любопытный результат получится, если в равенстве (21) положить \( x = i \) (мнимая единица, \( i^2 = -1 \)), \( n = 4m \).
Простой подсчет показывает, что \( (i + 1)^4 = (-1 + 2i + 1)^2 = 4i^2 = -4 \). Поэтому \( (i + 1)^{4m} = (-4)^m \). Мы получаем, таким образом, равенство
\[
(-4)^m = C_{4m}^0 + C_{4m}^1 i + C_{4m}^2 i^2 + C_{4m}^3 i^3 + C_{4m}^4 i^4 + \ldots + C_{4m}^{4m} =
= C_{4m}^0 + C_{4m}^1 i - C_{4m}^2 - C_{4m}^3 i + C_{4m}^4 + \ldots + C_{4m}^{4m}.
\]
Отделяя в этом равенстве действительную и мнимые части, приходим к тождествам
\[
C_{4m}^1 - C_{4m}^3 + C_{4m}^5 - C_{4m}^7 + \ldots - C_{4m}^{4m-1} = 0,
\]
\[
C_{4m}^0 - C_{4m}^2 + C_{4m}^4 - \ldots + C_{4m}^{4m} = (-4)^m.
\]
Выясните сами, какие тождества получатся, если положить \( n = 4m + 1, n = 4m + 2, n = 4m + 3 \).
