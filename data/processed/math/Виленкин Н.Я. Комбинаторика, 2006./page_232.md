В Западной Европе задолго до Ньютона ее знал Блез Паскаль. Заслуга же Ньютона была в ином — ему удалось обобщить формулу для \((a + x)^n\) на случай нецелых показателей. Именно, он доказал, что если \(a\) — положительное число и \(|x| < a\), то для любого действительного значения \(r\) имеет место равенство

\[
(a + x)^r = a^r + r a^{r-1} x +
+ \frac{r(r-1)}{1 \cdot 2} a^{r-2} x^2 + ... + \frac{r(r-1) ... (r-k+1)}{1 \cdot 2 ... k} a^{r-k} x^k + ... =
= a^r + C_r^1 a^{r-1} x + C_r^2 a^{r-2} x^2 + ... + C_r^k a^{r-k} x^k + ..., \tag{24}
\]

где \(C_r^k\) — введенные выше формулой (23) числа. Только теперь получилось не конечное число слагаемых, а бесконечный ряд. В случае, когда \(r\) — натуральное число, \(r = n\), начиная с некоторого места во все выражения для \(C_r^k\) входит скобка \((r-n)\), обращающая эти коэффициенты в нуль. Именно поэтому при натуральных значениях \(r\) вместо бесконечного ряда получается конечная сумма.

Доказательство формулы (24) в общем виде требует применения дифференциального исчисления. Мы не будем доказывать формулу (24) для всех значений \(r\) и рассмотрим лишь случай, когда \(r\) — целое отрицательное число, \(r = -n\). В этом случае подлежащая доказательству формула принимает следующий вид:

\[
(a + x)^{-n} = a^{-n} - C_n^1 a^{-n-1} x + C_{n+1}^2 a^{-n-2} x^2 - ... + (-1)^k C_{n+k-1}^k a^{-n-k} x^k + ...
\]

Иначе это равенство можно переписать так:

\[
\left(1 + \frac{x}{a}\right)^{-n} = 1 - C_n^1 \left(\frac{x}{a}\right) + C_{n+1}^2 \left(\frac{x}{a}\right)^2 - ... + (-1)^k C_{n+k-1}^k \left(\frac{x}{a}\right)^k + ...
\]

а заменив \(\frac{x}{a}\) на \(-t\) (тогда \(|t| < 1\)), получим следующее равенство:

\[
(1 - t)^{-n} = 1 + C_n^1 t + C_{n+1}^2 t^2 + C_{n+2}^3 t^3 + ... + C_{n+k-1}^k t^k + ... \tag{25}
\]

Его и будем доказывать (при \(|t| < 1\)) с помощью индукции по \(n\).

При \(n = 1\) имеем \(C_{n+k-1}^k = C_k^k = 1\) и потому доказываемое соотношение принимает следующий вид:

\[
\frac{1}{1 - t} = 1 + t + t^2 + t^3 + ... + t^k + ...
\] \tag{26}

Но это — уже известная нам формула (4).
