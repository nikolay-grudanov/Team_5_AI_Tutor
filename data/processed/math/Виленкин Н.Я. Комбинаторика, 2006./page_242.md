Но по рекуррентному соотношению (39)

\[
T_0 T_{n-1} + T_1 T_{n-2} + \ldots + T_{n-1} T_0 = T_n.
\]

Значит, \( F^2(x) = T_1 x^2 + T_2 x^3 + \ldots + T_n x^{n+1} + \ldots \)

Полученный ряд есть не что иное, как \( F(x) - T_0 x \); поскольку \( T_0 = 1 \), он равен \( F(x) - x \). Итак,

\[
F^2(x) = F(x) - x. \tag{41}
\]

Для функции \( F(x) \) получилось квадратное уравнение (41).

Решая его, находим, что \( F(x) = \frac{1 \pm \sqrt{1 - 4x}}{2} \). Поскольку из (40) \( F(0) = 0 \), выбираем перед корнем знак минус. По формуле (31) мы имеем

\[
\sqrt{1 - 4x} = (1 - 4x)^{\frac{1}{2}} =
= 1 - 2x - \frac{2}{2} C_2^1 x^2 - \frac{2}{3} C_4^2 x^3 - \ldots - \frac{2}{n + 1} C_{2n}^n x^{n+1} - \ldots
\]

Значит,

\[
F(x) = \frac{1}{2} \left[ 1 - \left( 1 - 2x - \frac{2}{2} C_2^1 x^2 - \ldots - \frac{2}{n + 1} C_{2n}^n x^{n+1} - \ldots \right) \right] =
= x + C_2^1 x^2 + \ldots + \frac{1}{n + 1} C_{2n}^n x^{n+1} - \ldots \tag{42}
\]

Сравнивая формулы (40) и (42), получаем, что \( T_n = \frac{1}{n + 1} C_{2n}^n \).

Это полностью соответствует решению, полученному ранее комбинаторным методом (п. 82).

117. Полиномиальная формула

\[
(x + y + z)^4 = [(x + y) + z]^4 =
= (x + y)^4 + C_4^1 (x + y)^3 z + C_4^2 (x + y)^2 z^2 + C_4^3 (x + y) z^3 + C_4^4 z^4.
\]

Разложим теперь \( (x + y)^4 \), \( (x + y)^3 \), \( (x + y)^2 \) по формуле бинома Ньютона и приведем подобные члены. Мы получим, что

\[
(x + y + z)^4 = x^4 + y^4 + z^4 + 4x^3 y + 4x^3 z + 4xy^3 + 4y^3 z + 4xz^3 +
+ 4yz^3 + 6x^2 y^2 + 6x^2 z^2 + 6y^2 z^2 + 12x^2 yz + 12xy^2 z + 12xyz^2. \tag{43}
