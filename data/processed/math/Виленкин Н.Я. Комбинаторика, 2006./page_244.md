Напомним, что

\[
P(k_1, k_2, ..., k_m) = \frac{(k_1 + k_2 + ... + k_m)!}{k_1! k_2! ... k_m!}.
\]

Ясно, что если набор \((s_1, s_2, ..., s_m)\) получается перестановкой из набора \((k_1, k_2, ..., k_m)\), то \(P(s_1, s_2, ..., s_m) = P(k_1, k_2, ..., k_m)\). Поэтому, например, в разложении (43) коэффициенты при \(x^2yz\) и \(xu^2z^2\) одинаковы. Это замечание облегчает выписывание членов разложения (45). Достаточно найти коэффициенты для таких разбиений \(n = k_1 + k_2 + ... + k_m\), что \(k_1 \geq k_2 \geq ... \geq k_m\), а потом переставлять показатели всеми возможными способами.

Например, вычислим \((x + y + z)^5\). Если не учитывать порядок слагаемых (например, записывая их в порядке убывания), то число 5 можно разбить на 3 слагаемых пятью способами:

\[
5 = 5 + 0 + 0,\ 5 = 4 + 1 + 0,\ 5 = 3 + 2 + 0,
\]
\[
5 = 3 + 1 + 1,\ 5 = 2 + 2 + 1.
\]

Но \(P(5, 0, 0) = 1,\ P(4, 1, 0) = 5,\ P(3, 2, 0) = 10,\ P(3, 1, 1) = 20,\ P(2, 2, 1) = 30\). Поэтому

\[
(x + y + z)^5 = x^5 + y^5 + z^5 + 5x^4y + 5xy^4 + 5x^4z + 5xz^4 + 5y^4z +
+ 5yz^4 + 10x^3y^2 + 10x^3z^2 + 10x^2y^3 + 10x^2z^3 + 10y^3z^2 + 10y^2z^3 +
+ 20x^3yz + 20xy^3z + 20xyz^3 + 30x^2y^2z + 30x^2yz^2 + 30xy^2z^2.
\]

Формула (45) позволяет легко доказать некоторые свойства чисел \(P(k_1, k_2, ..., k_m)\). Например, если положить в этой формуле \(x_1 = x_2 = ... = x_m\), то получим, что

\[
m^n = \sum_{k_1 + k_2 + ... + k_m = n} P(k_1, k_2, ..., k_m).
\] (46)

Здесь сумма распространена на все разбиения числа \(n\) на \(m\) неотрицательных целых слагаемых: \(n = k_1 + k_2 + ... + k_m\), и порядок слагаемых учитывается.

Далее, если умножить обе части равенства (45) на \(x_1 + x_2 + ... + x_m\), применить к левой части аналогичное разложение, а в правой раскрыть скобки, то для \(P(k_1, k_2, ..., k_m)\) получим следующее рекуррентное соотношение:

\[
P(k_1, k_2, ..., k_m) = P(k_1 - 1, k_2, ..., k_m) +
+ P(k_1, k_2 - 1, ..., k_m) + ... + P(k_1, k_2, ..., k_m - 1).\] (47)
