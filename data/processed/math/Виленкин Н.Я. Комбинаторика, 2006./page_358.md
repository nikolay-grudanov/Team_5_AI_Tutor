150. Нам надо доказать неравенство \( C_{2n+r}^n C_{2n-r}^n \leq (C_{2n}^n)^2 \). Его можно переписать в виде

\[
\frac{(2n + r)(2n + r - 1) ... (2n + 1)}{(n + r)(n + r - 1) ... (n + 1)} \leq \frac{2n(2n - 1) ... (2n - r + 1)}{n(n - 1) ... (n - r + 1)}
\]

Это неравенство вытекает из легко доказываемого при \( 1 \leq k < n \) неравенства \( \frac{2n + k}{n + k} < \frac{2n - k}{n - k} \).
