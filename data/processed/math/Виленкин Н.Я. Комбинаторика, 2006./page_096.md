360 = 2^3 \cdot 3^2 \cdot 5^1. При разложении числа N на два сомножителя, N = N_1 \cdot N_2, простые сомножители распределяются между N_1 и N_2. Если в N_1 сомножитель p_j войдет m_j раз, j = 1, ..., k, то разложение имеет вид

\[
N = N_1 \cdot N_2 = (p_1^{m_1} p_2^{m_2} ... p_k^{m_k}) (p_1^{n_1-m_1} p_2^{n_2-m_2} ... p_k^{n_k-m_k}).
\]

Таким образом, разложение N на два сомножителя сводится к разделу n_1 элементов одного вида, n_2 элементов второго вида, ..., n_k элементов k-го вида на две части. А формула (2) показывает, что это можно сделать (n_1 + 1)(n_2 + 1) ... (n_k + 1) способами. Значит, это и есть число делителей у натурального числа N = p_1^{n_1} p_2^{n_2} ... p_k^{n_k}. Это число обозначают \( \tau(N) \):

\[
\tau(N) = (n_1 + 1)(n_2 + 1) ... (n_k + 1).
\]
