Но при любом натуральном \( n \geq 2 \)

\[
\frac{1}{n+1} + \frac{1}{(n+1)(n+2)} + \frac{1}{(n+1)(n+2)(n+3)} + ... <
\]
\[
< \frac{1}{n+1} + \frac{1}{(n+1)^2} + \frac{1}{(n+1)^3} + ... = \frac{1}{n} \leq \frac{1}{2}.
\]

Этим наше утверждение доказано.
б) Общее число предметов во всех размещениях равно

\[
nA_n^n + (n-1)A_n^{n-1} + ... + A_n^1 = n! \left[ n + \frac{n-1}{1!} + ... + \frac{1}{(n-1)!} \right] =
\]
\[
= (n-1)n! \left[ \left( 1 + 1 + \frac{1}{2!} + ... + \frac{1}{(n-1)!} \right) + \frac{1}{n-1} \left( 1 - \frac{1}{2!} - \frac{2}{3!} - ... - \frac{n-2}{(n-1)!} \right) \right].
\]

Легко проверить, что \( 1 - \frac{1}{2!} - \frac{2}{3!} - ... - \frac{n-2}{(n-1)!} = \frac{1}{(n-1)!} \). Так как все предметы входят одно и то же число раз, то каждый из них войдет
\[
N = (n-1)(n-1)! \left[ 1 + 1 + \frac{1}{2!} + ... + \frac{1}{(n-1)!} \right] + 1 \text{ раз. С другой стороны,}
\]
\[(n-1)(n-1)!e =\]
\[
= (n-1)(n-1)! \left[ 1 + 1 + \frac{1}{2!} + ... + \frac{1}{(n-1)!} + \frac{1}{n!} + \frac{1}{(n+1)!} + ... \right] =
\]
\[
= (n-1)(n-1)! \left[ 1 + 1 + \frac{1}{2!} + ... + \frac{1}{(n-1)!} \right] + (n-1) \left[ \frac{1}{n} + \frac{1}{n(n+1)} + ... \right]
\]
и потому
\[
\Delta = N - (n-1)(n-1)!e = 1 - (n-1) \left[ \frac{1}{n} + \frac{1}{n(n+1)} + ... \right] > 1 - \frac{n-1}{n-1} = 0,
\]
но \( \Delta = \frac{1}{n} \left[ 1 - \frac{n-1}{n+1} - \frac{n-1}{(n+1)(n+2)} - ... \right] < \frac{1}{2} \) при \( n \geq 2 \).

Значит, \( N \) — ближайшее целое число к \( (n-1)(n-1)!e \).
