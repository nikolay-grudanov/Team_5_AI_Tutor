Поэтому в нашем случае \( N(\overline{a}_1 \overline{a}_2 \overline{a}_3 ... \overline{a}_n) = 0 \), \( N_k = C_{n-k}^{m-k} \). Подставляя эти значения в формулу (31), приходим к тождеству (33).

Точно так же доказываются соотношения

\[
C_n^0 C_{n+m-1}^m - C_n^1 C_{n+m-2}^{m-1} + C_n^2 C_{n+m-3}^{m-2} - ... + (-1)^n C_n^n C_{m-1}^{m-n} = 0,
\]
если \( m \geq n \);

\[
C_n^0 C_{n+m-1}^m - C_n^1 C_{n+m-2}^{m-1} + C_n^2 C_{n+m-3}^{m-2} - ... + (-1)^m C_n^m C_{n-1}^0 = 0,
\]
если \( m < n \).

Именно, рассмотрим сочетания с повторениями из \( n \) элементов \( a_1, a_2, ..., a_n \) по \( m \) и обозначим через \( \langle a_k \rangle, 1 \leq k \leq n \), свойство сочетания, заключающееся в том, что среди его элементов есть \( a_k \) (и, быть может, другие элементы). Тогда \( N(a_1 ... a_k) \) — число сочетаний, в которые заведомо входят элементы \( a_1, ..., a_k \).

Из каждого такого сочетания можно удалить по одному элементу \( a_1, ..., a_k \). В результате получится некоторое сочетание с повторениями из \( n \) элементов \( a_1, ..., a_n \) по \( m - k \). При этом, обратно, добавляя к сочетанию с повторениями из элементов \( a_1, ..., a_n \), содержащему \( m - k \) элементов, по одному элементу \( a_1, ..., a_k \), получим сочетание, содержащее \( m \) элементов, в котором заведомо представлены \( a_1, ..., a_k \). Отсюда вытекает, что число \( N(a_1 ... a_k) \) равно числу сочетаний с повторениями из \( n \) элементов по \( m - k \), т. е. \( N(a_1 ... a_k) = C_{n+m-k-1}^{m-k} \).

Далее, общее число сочетаний с повторениями из \( n \) элементов \( a_1, ..., a_n \) по \( m \) равно \( C_{n+m-1}^m \) (это \( N_0 \)), а сочетаний, не обладающих ни одним из свойств \( \langle a_k \rangle, 1 \leq k \leq n \), не существует. Подставляя найденные значения \( N(\overline{a}_1 \overline{a}_2 \overline{a}_3 ... \overline{a}_n) = 0 \), \( N_k = C_{n+m-k-1}^{m-k} \) в формулу (31), приходим к тождествам (34).
