Если же перемножить левые и правые части разложений

\[
(x_1 + x_2 + ... + x_m)^n = \sum_{k_1 + k_2 + ... + k_m = n} P(k_1, k_2, ..., k_m) x_1^{k_1} x_2^{k_2} ... x_m^{k_m},
\]

\[
(x_1 + x_2 + ... + x_m)^s = \sum_{l_1 + l_2 + ... + l_m = s} P(l_1, l_2, ..., l_m) x_1^{l_1} x_2^{l_2} ... x_m^{l_m}
\]

и сравнить коэффициенты в обеих частях при \( x_1^{r_1} x_2^{r_2} ... x_m^{r_m} \), то получим тождество

\[
P(r_1, r_2, ..., r_m) = \sum_{k_i + l_i = r_i} P(k_1, k_2, ..., k_m) P(l_1, l_2, ..., l_m).
\] (48)

Здесь суммирование распространено на все наборы целых неотрицательных чисел \((k_1, k_2, ..., k_m)\) и \((l_1, l_2, ..., l_m)\) таких, что \(k_1 + k_2 + ... + k_m = n,\ l_1 + l_2 + ... + l_m = s\) и \(k_1 + l_1 = r_1,\ k_2 + l_2 = r_2,\ ..., k_m + l_m = r_m\). Мы предоставляем читателю подробно провести соответствующие выкладки.

Разумеется, формулы (46) – (48) можно было бы получить и не пользуясь производящей функцией (45). Но в этом случае пришлось бы проводить непростые рассуждения. Применение же производящей функции позволяет получить эти тождества автоматически, выполняя лишь несложные алгебраические преобразования.
