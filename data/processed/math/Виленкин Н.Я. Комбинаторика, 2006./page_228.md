Нас будут интересовать производящие функции для последовательностей \(a_0, a_1, ..., a_n, ...\), так или иначе связанных с комбинаторными задачами. **110. Производящие функции и биномиальные коэффициенты**

Найдем производящую функцию для конечной последовательности биномиальных коэффициентов \(C_n^0, C_n^1, ..., C_n^n\).

Полагая в биноме Ньютона (п. 24) \(a = 1\) и располагая слагаемые в порядке возрастания степеней \(x\), получаем

\[
(1 + x)^n = C_n^0 + C_n^1 x + C_n^2 x^2 + ... + C_n^k x^k + ... + C_n^n x^n.
\] (21)

Мы видим, что \((1 + x)^n\) является производящей функцией для чисел \(C_n^k, k = 0, 1, ..., n\). С помощью этой производящей функции можно сравнительно просто доказать многие свойства чисел \(C_n^k\), которые были получены ранее с помощью довольно хитрых рассуждений.

\[
(1 + x)^{n+1} = (C_n^0 + C_n^1 x + ... + C_n^{k-1} x^{k-1} + C_n^k x^k + ... + C_n^n x^n)(1 + x).
