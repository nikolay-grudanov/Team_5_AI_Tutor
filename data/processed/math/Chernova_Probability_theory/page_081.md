§6. Функции от двух случайных величин

Для случайных величин с дискретными или с абсолютно непрерывными распределениями эквивалентные определения независимости выглядят так.

Определение 20. Случайные величины \(1, \ldots, n\) с дискретным распределением независимы, если для любых чисел \(a_1, \ldots, a_n\) имеет место равенство:
\[
\mathrm{P}(1 = a_1, \ldots, n = a_n) = \mathrm{P}(1 = a_1) \cdots \mathrm{P}(n = a_n).
\]

Определение 21. Случайные величины \(1, \ldots, n\) с абсолютно непрерывным совместным распределением независимы, если плотность совместного распределения распадается в произведение плотностей, т.е. для любых \(x_1, \ldots, x_n\) имеет место равенство:
\[
f(x_1, \ldots, x_n) = f_1(x_1) \cdots f_n(x_n).
\]
