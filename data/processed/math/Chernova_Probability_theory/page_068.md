Плотность распределения случайной величины \( = g( ) \) существует далеко не при любых функциях \( g \). Так, если функция \( g \) кусочно-постоянна, то имеет дискретное распределение, и плотность её распределения не существует. Например, если функция \( g(x) = 1 \) при всех \( x \), то величина \( = g( ) = 1 \) имеет вырожденное распределение \( \mathbf{I}_1 \).

Теорема 22. Пусть случайная величина имеет плотность распределения \( f(x) \), и постоянная \( a \) отлична от нуля. Тогда \( = a + b \) имеет плотность распределения

\[
f (x) = \frac{1}{|a|} f \left( \frac{x - b}{a} \right).
\]

Доказательство. Пусть сначала \( a > 0 \).

\[
F (x) = \mathrm{P}(a + b < x) = \mathrm{P} \left( < \frac{x - b}{a} \right) = F \left( \frac{x - b}{a} \right).
\]

Продифференцируем это выражение по \( x \) как сложную функцию:

\[
f (x) = (F (x))' = \frac{d}{dx} F \left( \frac{x - b}{a} \right) = \left( \frac{x - b}{a} \right)' \cdot f \left( \frac{x - b}{a} \right) = \frac{1}{a} f \left( \frac{x - b}{a} \right).
\]

Пусть теперь \( a < 0 \).

\[
F (x) = \mathrm{P}(a + b < x) = \mathrm{P} \left( > \frac{x - b}{a} \right) = 1 - F \left( \frac{x - b}{a} \right).
\]

Продифференцируем это выражение по \( x \):

\[
f (x) = (F (x))' = \frac{d}{dx} \left( 1 - F \left( \frac{x - b}{a} \right) \right) = -\frac{1}{a} f \left( \frac{x - b}{a} \right) = \frac{1}{|a|} f \left( \frac{x - b}{a} \right).
\]

Убедиться в том, что производная \( f (x) \) действительно является плотностью распределения, можно, например, проверив для неё свойство (f2) плотности.

Следствие 2. Если \( \sim \mathrm{N}_{0,1} \), то \( = + a \sim \mathrm{N}_{a, 2} \).

Доказательство. Действительно,

\[
f (x) = \frac{1}{\sqrt{2\pi}} \cdot f \left( \frac{x - a}{\sqrt{2}} \right) = \frac{1}{\sqrt{2\pi}} e^{-\frac{(x-a)^2}{2}}.
\]

Следствие 3. Если \( \sim \mathrm{N}_{a, 2} \), то её линейное преобразование \( c + d \) при \( c \neq 0 \) снова имеет нормальное распределение.

Этот факт следует запомнить: линейное преобразование оставляет распределение нормальным. Полезно также убедиться, что \( c + d \sim \mathrm{N}_{ca+d, c^2 2} \).

Следствие 4. Если \( \sim \mathrm{U}_{a, b} \), то её линейное преобразование \( c + d \) при \( c \neq 0 \) снова имеет равномерное распределение.

Границы интервала, на котором сосредоточено распределение \( c + d \), получаются линейным преобразованием границ исходного интервала \((a, b)\). Например, \( c + d \sim \mathrm{U}_{ca+d, cb+d} \) при \( c > 0 \).
