§ 5. Примеры абсолютно непрерывных распределений

Показательное распределение E . Случайная величина имеет показательное (экспоненциальное) распределение с параметром \( > 0 \), если имеет следующие плотность и функцию распределения:

\[
f(x) = \begin{cases}
0, & \text{если } x < 0, \\
e^{-x}, & \text{если } x \geqslant 0;
\end{cases} \qquad F(x) = \begin{cases}
0, & \text{если } x < 0, \\
1 - e^{-x}, & \text{если } x \geqslant 0.
\end{cases}
\]

![Графики плотности и функции распределения показательного распределения](https://i.imgur.com/3Q5z5QG.png)

Рис. 13. Плотность и функция распределения E

Плотность показательного распределения равна нулю на отрицательной полуоси, поэтому вероятность события \( \{ < 0 \} \) нулевая — случайная величина с показательным распределением не может быть отрицательна. К тому же плотность отлична от нуля на всей положительной полуоси, поэтому случайная величина с показательным распределением может принимать сколь угодно большие положительные значения: для всякого \( x \) вероятность события \( \{ > x \} \) не равна нулю.

Показательное распределение обладает свойством отсутствия последействия. Мы встречались с этим свойством у геометрического распределения.

Теорема 21. Пусть \( \sim \mathrm{E} \). Тогда для любых \( x, y > 0 \)

\[
\mathrm{P}( \geqslant x + y \mid \geqslant x ) = \mathrm{P}( \geqslant y ).
\] (12)

Доказательство. Заметим, что для любого \( t > 0 \)

\[
\mathrm{P}( \geqslant t ) = 1 - \mathrm{P}( < t ) = 1 - F(t) = e^{-t}.
\]

Поэтому

\[
\mathrm{P}( \geqslant x + y \mid \geqslant x ) = \frac{\mathrm{P}( \geqslant x + y, \geqslant x )}{\mathrm{P}( \geqslant x )} = \frac{\mathrm{P}( \geqslant x + y )}{\mathrm{P}( \geqslant x )} =
= \frac{e^{-(x+y)}}{e^{-x}} = e^{-y} = \mathrm{P}( \geqslant y ). \hspace{1cm} \square
\]
