§ 7. Контрольные вопросы

Следствие 5. Если \( \sim \mathrm{E} \), то \( \sim \mathrm{E}_1 \).

Пример 48. Случайная величина имеет стандартное нормальное распределение. Найти плотность распределения величины \( ^2 \).

Решение. Для \( x \leqslant 0 \) функция распределения величины \( = ^2 \) равна
\[
F(x) = \mathrm{P}(^2 < x) = 0, \text{ поэтому } f(x) = 0.
\]
Для \( x > 0 \) имеем: \( F(x) = \mathrm{P}(^2 < x) = \mathrm{P}(|| < \sqrt{x}) = 1 - 2\Phi(-\sqrt{x}) \).
Производная функции \( \Phi(x) \) равна плотности стандартного нормального распределения \( (x) \), поэтому при \( x > 0 \)
\[
f(x) = (F(x))' = -2\left(-\sqrt{x}\right) \cdot \left(-\sqrt{x}\right)' = \frac{1}{\sqrt{x}}\left(-\sqrt{x}\right) = \frac{1}{\sqrt{2\neq x}} e^{-x/2}.
\]
Мы получили следующую плотность:
\[
f(x) = \begin{cases}
0, & \text{если } x \leqslant 0, \\
\frac{1}{\sqrt{2\neq x}} e^{-x/2}, & \text{если } x > 0.
\end{cases}
\]
Это плотность гамма-распределения с параметрами \( = \frac{1}{2}, = \frac{1}{2} \). Тем самым мы доказали, что \( \sim \Gamma_{1/2, 1/2} \). Другое название этого распределения — распределение хи-квадрат с одной степенью свободы.

Пример 49. Случайная величина имеет равномерное распределение на отрезке \([0, 1]\). Найти распределение величины \( -\ln \).

Решение. Функция распределения величины \( = -\ln \) равна
\[
F(x) = \mathrm{P}(-\ln < x) = \mathrm{P}(> e^{-x}) = 1 - \mathrm{P}(\leqslant e^{-x}) = 1 - F(e^{-x}).
\]
Подставим \( e^{-x} > 0 \) вместо \( x \) в функцию распределения \( U_{0, 1} \):
\[
F(x) = \begin{cases}
0, & \text{если } x < 0, \\
x, & \text{если } 0 \leqslant x \leqslant 1, \\
1, & \text{если } x > 1,
\end{cases}
\]
\[
F(e^{-x}) = \begin{cases}
e^{-x}, & \text{если } 0 < e^{-x} \leqslant 1, \\
1, & \text{если } e^{-x} > 1.
\end{cases}
\]
Окончательно получаем:
\[
F(x) = 1 - F(e^{-x}) = \begin{cases}
1 - e^{-x}, & \text{если } x \geqslant 0, \\
0, & \text{если } x < 0,
\end{cases}
\]
Итак, \( \sim \mathrm{E}_1 \). Мы получили замечательный факт: применение функции \( g(x) = -\ln x \) к равномерно распределённой на отрезке \([0, 1]\) случайной величине приводит к показательному распределению с параметром 1. Если же мы хотим получить показательное распределение с произвольным параметром , следует воспользоваться следствием 5: \( = -\frac{1}{\alpha} \ln \sim \mathrm{E} \).
