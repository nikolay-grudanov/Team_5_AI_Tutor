Распределение Коши \( C_a \). Случайная величина имеет распределение Коши с параметрами \( a \in (-\infty, \infty), \neq > 0 \), если имеет плотность распределения

\[
f(x) = \frac{1}{\neq^2 + (x - a)^2}.
\]

Плотность распределения Коши симметрична относительно прямой \( x = a \) и похожа на плотность нормального распределения, но имеет более толстые «хвосты» на \( \pm \infty \). Функция распределения случайной величины \( \sim C_a \), равна

\[
F(x) = \frac{1}{2} + \frac{1}{\neq} \arctg \left( \frac{x - a}{\neq} \right).
\]

Распределение Коши имеет простой физический смысл (см. рис. 17). Если из точки плоскости с координатами \( (a, ) \) испустить луч в направлении оси \( OX \) под случайным углом , выбранным наудачу из интервала \( (0, \neq) \), то абсцисса точки пересечения луча с осью \( OX \) будет иметь распределение Коши \( C_a \).

Действительно,

\[
= \operatorname{ctg} + a = \operatorname{tg} \left( \frac{\neq}{2} - \right) + a,
\]

\[
\mathrm{P}( < x) = \mathrm{P} \left( \frac{\neq}{2} - < \arctg \left( \frac{x - a}{\neq} \right) \right) = \mathrm{P} \left( > \frac{\neq}{2} - \arctg \left( \frac{x - a}{\neq} \right) \right).
\]

Поскольку выбирается наудачу на интервале \( (0, \neq) \), вероятность есть отношение длин соответствующих интервалов:

\[
\mathrm{P} \left( > \frac{\neq}{2} - \arctg \left( \frac{x - a}{\neq} \right) \right) = \frac{\neq - \left( \frac{\neq}{2} - \arctg \left( \frac{x - a}{\neq} \right) \right)}{\neq} = \frac{1}{2} + \frac{1}{\neq} \arctg \left( \frac{x - a}{\neq} \right).
\]

Распределение Парето. Случайная величина имеет распределение Парето с параметром \( > 0 \), если имеет следующие плотность и функцию распределения:

\[
f(x) = \begin{cases}
\frac{1}{x^{+1}}, & \text{если } x \geqslant 1, \\
0, & \text{если } x < 1;
\end{cases}
\]
\[
F(x) = \begin{cases}
1 - \frac{1}{x}, & \text{если } x \geqslant 1, \\
0, & \text{если } x < 1.
\end{cases}
\]
