Доказательство. Рассмотрим такую же матрицу \( DCD^{-1} \), как и при доказательстве теоремы 37.5.3, но только теперь положим

\[
\rho = \max_{1 \leq j \leq n} \left| \frac{a_{n-j}}{\lambda_j} \right|^{1/j}.
\]

В результате снова получим неравенства \( |z| \leq \rho \) и

\[
|z| \leq |a_{n-1}| + \frac{|a_{n-2}|}{\rho} + \ldots + \frac{|a_0|}{\rho^{n-1}}.
\]

Из неравенства \( |a_{n-j}| \leq \rho^j \lambda_j \) следует неравенство \( \frac{|a_{n-j}|}{\rho^{j-1}} \leq \rho \lambda_j \). Поэтому

\[
|a_{n-1}| + \frac{|a_{n-2}|}{\rho} + \ldots + \frac{|a_0|}{\rho^{n-1}} \leq \rho (\lambda_1 + \ldots + \lambda_n) = \rho. \quad \Box
\]

Теорема 37.5.5. Корни многочлена (1) лежат в круге

\[
\left| z + \frac{a_{n-1}}{2} \right| \leq \frac{|a_{n-1}|}{2} + |a_{n-2}|^{1/2} + |a_{n-3}|^{1/3} + \ldots + |a_0|^{1/n}. \tag{4}
\]

Доказательство. Будем действовать так же, как и при доказательстве теоремы 37.5.4, только теперь положим \( \rho = \max_{2 \leq j \leq n} |a_{n-j}|^{1/j} \). В результате получим неравенства \( |z| \leq \rho \) и

\[
|z + a_{n-1}| \leq \frac{|a_{n-2}|}{\rho} + \ldots + \frac{|a_0|}{\rho^{n-1}} \leq \\
\leq \frac{|a_{n-2}|}{|a_{n-2}|^{1/2}} + \ldots + \frac{|a_0|}{|a_0|^{(n-1)/n}} \leq |a_{n-2}|^{1/2} + |a_{n-3}|^{1/3} + \ldots + |a_0|^{1/n}.
\]

Из каждого из этих неравенств следует неравенство (4), поскольку

\[
\left| z + \frac{a_{n-1}}{2} \right| \leq |z + a_{n-1}| + \frac{|a_{n-1}|}{2}, \quad \left| z + \frac{a_{n-1}}{2} \right| \leq |z| + \frac{|a_{n-1}|}{2}
\]

и \( \rho \leq |a_{n-2}|^{1/2} + |a_{n-3}|^{1/3} + \ldots + |a_0|^{1/n} \). \( \Box \)

Круги Гершгорина можно использовать аналогичным образом и для оценок снизу абсолютных величин корней многочленов.

Теорема 37.5.6. Фиксируем положительное число \( t \). Пусть \( M = \max_{1 \leq i \leq n} |a_i| t^i \).

Тогда корни многочлена (1) по абсолютной величине не меньше \( \frac{|a_0| t}{|a_0| + M} \).
