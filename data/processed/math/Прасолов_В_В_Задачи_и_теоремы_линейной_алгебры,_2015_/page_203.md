Следовательно, \( \|x + y\| + \|z + x\| \leq \|y\| + \|z\| \). Остаётся заметить, что \( \|y + z\| \leq \|x\| + \|x + y + z\| \).

**Случай 2.** \( \mu \geq \lambda \geq 0 > \nu \).
В этом случае \( z = \alpha x + \beta y \), где \( \alpha = -\lambda/\nu \geq 0 \) и \( \beta = -\mu/\nu \geq 0 \) (при этом \( \alpha \leq \beta \)). Положим
\[
l = \frac{1}{1 + \beta} \quad \text{и} \quad k = \frac{\beta - \alpha}{1 + \beta}.
\]
Тогда \( 0 \leq k, l \leq 1 \) и выполняются равенства \( kx + l(x + y + z) = x + y \) и \( (1 - k)x + (1 - l)(x + y + z) = x + z \). Поэтому
\[
\|x + y\| + \|z + x\| \leq \|x\| + \|x + y + z\|.
\]
Остаётся заметить, что \( \|y + z\| \leq \|y\| + \|z\| \).

**12.4.** Пусть \( x' = x - z, y' = y - z, w' = w - z \). Тогда \( d(w, z) = d(w', 0), d(w, x) = d(w', x') \) и \( d(w, y) = d(w', y') \). Поэтому достаточно рассмотреть случай, когда \( z = 0 \).

Рассмотрим сначала случай, когда \( a = 1/2 \), т. е. для любого \( w \) имеет место неравенство \( 2d(w, 0) \leq d(w, x) + d(w, y) \). Пусть \( T \) — множество всех целых неотрицательных \( t \), для которых
\[
(2m + t)d(x + y, 0) \leq d(mx + my, x) + d(mx + my, y)
\]
для всех натуральных \( m \). Докажем, что \( T \) содержит все целые неотрицательные числа \( t \). При \( t = 0 \) достаточно применить исходное неравенство для \( w = mx + my \). Остаётся доказать, что если \( t \in T \), то \( t + 1 \in T \). Ясно, что если число \( m \) натуральное, то число \( 2m \) тоже натуральное, поэтому
\[
(4m + t)d(x + y, 0) \leq d(2mx + 2my, x) + d(2mx + 2my, y).
\]
Подставим в исходное неравенство \( w = (2m - 1)x + 2my \). После несложных преобразований оно перепишется в виде
\[
d(2mx + 2my, x) \leq d(mx + my, x) + \left(m - \frac{1}{2}\right)d(x + y, 0).
\]
Аналогично
\[
d(2mx + 2my, x) \leq d(mx + my, y) + \left(m - \frac{1}{2}\right)d(x + y, 0).
\]
Подставляя неравенства (3) и (4) в неравенство (2), получаем
\[
(4m + t)d(x + y, 0) \leq d(mx + my, x) + d(mx + my, y) + (2m - 1)d(x + y, 0),
\]
т. е.
\[
(2m + t + 1)d(x + y, 0) \leq d(mx + my, x) + d(mx + my, y).
\]
Таким образом, \( t + 1 \in T \).
При \( m = 1 \) неравенство (1) принимает вид
\[
(2 + t)d(x + y, 0) \leq d(y, 0) + d(x, 0).
