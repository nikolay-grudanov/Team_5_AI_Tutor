Доказательство. Ясно, что \((S - U)^*(S - U) = S^2 - (U + U^*)S + I\) и \((S - I)^*(S - I) = S^2 - 2S + I\). Поэтому \(\|S - U\|^2_e - \|S - I\|^2_e = 2\operatorname{tr} S - \operatorname{tr}((U + U^*)S)\). Так как \(|\operatorname{tr}(US)| \leq \operatorname{tr} S\) и \(|\operatorname{tr}(U^*S)| \leq \operatorname{tr} S\) (см. задачу 37.1), то \(\operatorname{tr}((U + U^*)S) \leq |\operatorname{tr}(US)| + |\operatorname{tr}(U^*S)| \leq 2\operatorname{tr} S\).

Если \(S > 0\), то равенство может достигаться, только если \(U = e^{i\varphi}I\). В этом случае \(\operatorname{tr}((U + U^*)S) = (e^{i\varphi} + e^{-i\varphi})\operatorname{tr} S\). Ясно, что если \(e^{i\varphi} + e^{-i\varphi} = 2\), то \(e^{i\varphi} = 1\).

Теорема 38.3.3. Пусть \(A = US\) — полярное разложение матрицы \(A\). Тогда для любой унитарной матрицы \(X\) справедливо неравенство \(\|A - U\|_e \leq \leq \|A - X\|_e\), причём для невырожденной матрицы \(A\) равенство достигается только при \(X = U\).

Доказательство. Ясно, что \(\|A - X\|_e = \|US - X\|_e = \|U(S - U^{-1}X)\|_e = \|S - U^{-1}X\|_e\). Согласно теореме 38.3.2 \(\|S - U^{-1}X\|_e \geq \|S - I\|_e = \|A - U\|_e\), причём если \(S > 0\), то равенство достигается, только если \(U^{-1}X = I\), т. е. \(X = U\).

38.4. Наилучшее приближение невырожденной матрицы вырожденной матрицей

Теорема 38.4.1. Пусть \(A\) — невырожденная матрица, \(X\) — вырожденная матрица. Тогда \(\|A - X\|_s \geq \|A^{-1}\|_s^{-1}\), причём если \(\|A^{-1}\|_s = \rho(A^{-1})\), то существует вырожденная матрица \(X\), для которой \(\|A - X\|_s = \|A^{-1}\|_s^{-1}\).

Доказательство [Fr2]. Возьмём такой вектор \(v\), что \(Xv = 0\) и \(v \neq 0\). Тогда
\[
\|A - X\|_s \geq \frac{\|(A - X)v\|}{\|v\|} = \frac{\|Av\|}{\|v\|} \geq \min_x \frac{\|Ax\|}{\|x\|} = \min_y \frac{\|y\|}{\|A^{-1}y\|} = \|A^{-1}\|_s^{-1}.
\]
Предположим теперь, что \(\|A^{-1}\|_s = |\lambda^{-1}|\) и \(A^{-1}y = \lambda^{-1}y\), т. е. \(Ay = \lambda y\). Тогда \(\|A^{-1}\|_s^{-1} = |\lambda| = \|Ay\|/\|y\|\). Матрица \(X = A - \lambda I\) вырожденная и \(\|A - X\|_s = \|\lambda I\|_s = |\lambda| = \|A^{-1}\|_s^{-1}\).
