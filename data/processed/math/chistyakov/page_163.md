Приведенное здесь определение цепи Маркова является более узким по сравнению с определением § 1 гл. 3. Здесь числа \( p_{ij} \) (1.3) (\( i, j = 1, 2, \ldots, N \)), являющиеся вероятностями перехода из состояния \( i \) в состояние \( j \), не зависят от номера испытания, как в гл. 3. Такие цепи Маркова называют однородными. Утверждение теоремы 1.2 верно только для однородных цепей Маркова.

Условие (3.1.6) было проверено в гл. 3 для произвольной последовательности испытаний. Матрицы вида (1.3), для которых выполнено (1.4), называются стационарными. Цепи Маркова часто используются для описания различных систем, которые могут находиться в одном из \( N \) состояний и в дискретные моменты времени меняют состояние. В этом случае номер испытания \( t \) естественно интерпретировать как время, а символ \( l_t \) — как состояние системы в момент времени \( t \). С целью Маркова естественно связать последовательность случайных величин \( \xi_t = \xi_t(l_0 l_1 \ldots l_T) = l_t, t = 0, 1, \ldots, T \), являющихся номером состояния, в котором находится система в момент времени \( t \).

Из (1.2) следует, что

\[
P(\xi_0 = l_0, \xi_1 = l_1, \ldots, \xi_T = l_T) = q_{l_0} p_{l_0 l_1} p_{l_1 l_2} \cdots p_{l_{T-1} l_T}
\]

Так как

\[
(\xi_0 = L_0, \xi_1 = L_1, \ldots, \xi_t = L_t) =
\]
\[
= \{ \omega : l_0 = L_0, \ldots, l_t = L_t \} =
\]
\[
= \{ L_0 L_1 \ldots L_t l_{t+1} \ldots l_T \} = A_{L_0 \ldots L_t},
\]

то

\[
P(\xi_0 = L_0, \xi_1 = L_1, \ldots, \xi_t = L_t) =
\]
\[
= \sum_{\omega \in A_{L_0 \ldots L_t}} q_{L_0} p_{L_0 L_1} \cdots p_{L_{t-1} L_t} p_{L_t l_{t+1}} \cdots p_{l_{T-1} l_T} =
\]
\[
= q_{L_0} p_{L_0 L_1} \cdots p_{L_{t-1} L_t} \sum_{l_{t+1}, \ldots, l_T}^{N} p_{L_t l_{t+1}} p_{l_{t+1} l_{t+2}} \cdots p_{l_{T-1} l_T}.
\]

Из свойства (1.4) следует, что сумма в последнем равенстве равна 1. Таким образом,

\[
P(\xi_0 = l_0, \xi_1 = l_1, \ldots, \xi_t = l_t) = q_{l_0} p_{l_0 l_1} \cdots p_{l_{t-1} l_t} \tag{1.5}
\]

при любых \( l_0, l_1, \ldots, l_t = 1, 2, \ldots, N \) и любом \( t = 0, 1, 2, \ldots, T \).
