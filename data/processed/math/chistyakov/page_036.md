(l < a). Найти вероятность того, что игла пересечет какую-нибудь прямую.

Решение. Дадим сначала определение подходящего пространства элементарных событий. Пусть u — расстояние от центра иглы до ближайшей прямой, а φ — угол, составленный игрой с этой прямой. Пара чисел (φ, u) задает положение игры с точностью до выбора конкретной прямой. Поскольку нас интересует только взаимное расположение игры с ближайшей прямой, то можно в качестве Ω выбрать прямоугольник

\[
\Omega = \{(\varphi, u): 0 \leq \varphi \leq \pi, 0 \leq u \leq a\}.
\]

Пересечение игры с прямой происходит в том и только в том случае, когда (рис. 2)

\[
u \leq l \sin \varphi.
\]

Таким образом, интересующее нас событие

\[
A = \{(\varphi, u): u \leq l \sin \varphi\}
\]

является множеством, заштрихованным на рис. 3. Так как

\[
\text{пл. } A = \int_0^{\pi} l \sin \varphi \, d\varphi = 2l, \quad \text{пл. } \Omega = a\pi,
\]
