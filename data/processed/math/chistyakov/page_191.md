§ 2. Пуассоновский процесс

Случайный процесс \( \xi_t \) с непрерывным временем называется процессом с независимыми приращениями, если при любых \( 0 \leq t_1 < t_2 < ... < t_n, t_1, ..., t_n \in [0, \infty), n = 1, 2, ..., \) случайные величины

\[
\xi_{t_1}, \xi_{t_2} - \xi_{t_1}, ..., \xi_{t_n} - \xi_{t_{n-1}}
\]

независимы. Пуассоновским процессом называется процесс \( \xi_t \), удовлетворяющий следующим условиям:

\( 1^o. \) \( \xi_t \) — процесс с независимыми приращениями.

\( 2^o. \) При любых \( t_1 < t_2, \) с приращения \( \xi_{t_2} - \xi_{t_1}, \xi_{t_2+s} - \xi_{t_1+s} \) одинаково распределены (однородность по времени).

\( 3^o. \) \( \xi_0 (\omega) = 0, \omega \in \Omega. \)

\( 4^o. \) При \( h \to 0 \)
\[
P(\xi_h = 0) = 1 - \lambda h + o(h), \quad P(\xi_h \geq 2) = o(h),
\]
\[
P(\xi_h = 1) = \lambda h + o(h), \quad 0 < \lambda < \infty.
\]

Пуассоновский процесс \( \xi_t \) можно задать на вероятностном пространстве \( (\Omega, \mathcal{F}, P) \), где множество \( \Omega \) совпадает с множеством ступенчатых функций, у которых имеются только единичные скачки, а моменты времени, соответствующие скачкам, не имеют точек накопления.

Теорема 2.1. Если \( \xi_t \) — пуассоновский процесс, то
\[
P(\xi_t = k) = \frac{(\lambda t)^k}{k!} e^{-\lambda t}, \quad k = 0, 1, 2, ...
\]
