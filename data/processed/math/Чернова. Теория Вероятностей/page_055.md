Раздел 9. Случайные вектора и их распределения

Определение 30. Если случайные величины \( \xi_1, \ldots, \xi_n \) заданы на одном вероятностном пространстве, то вектор \( (\xi_1, \ldots, \xi_n) \) мы будем называть случайным вектором.

Определение 31. Функция \( F_{\xi_1, \ldots, \xi_n}(x_1, \ldots, x_n) = \mathbf{P}(\xi_1 < x_1, \ldots, \xi_n < x_n) \) называется функцией распределения случайного вектора \( (\xi_1, \ldots, \xi_n) \) или функцией совместного распределения случайных величин \( \xi_1, \ldots, \xi_n \).

9.1 Свойства функции совместного распределения

Для простоты обозначений все дальнейшие рассуждения и формулировки приводятся в случае \( n = 2 \) для случайного вектора \( (\xi_1, \xi_2) \).

F0) \( 0 \leq F_{\xi_1, \xi_2}(x_1, x_2) \leq 1 \).

F1) \( F_{\xi_1, \xi_2}(x_1, x_2) \) не убывает по каждой координате вектора \( (x_1, x_2) \).

F2) Для любого \( i = 1, 2 \) существует \( \lim_{x_i \to -\infty} F_{\xi_1, \xi_2}(x_1, x_2) = 0 \);
Для любого \( i = 1, 2 \) существует \( \lim_{x_i \to \infty} F_{\xi_1, \xi_2}(x_1, x_2) \). При этом
\[
F_{\xi_1, \xi_2}(\infty, x_2) \overset{\text{def}}{=} \lim_{x_1 \to \infty} F_{\xi_1, \xi_2}(x_1, x_2) = F_{\xi_2}(x_2), \qquad F_{\xi_1, \xi_2}(x_1, \infty) \overset{\text{def}}{=} \lim_{x_2 \to \infty} F_{\xi_1, \xi_2}(x_1, x_2) = F_{\xi_1}(x_1).
\]

F3) Функция \( F_{\xi_1, \xi_2}(x_1, x_2) \) по каждой координате вектора \( (x_1, x_2) \) непрерывна слева.

Доказательство этих свойств совершенно аналогично одномерному случаю.
Только теперь этих свойств оказывается недостаточно для описания класса функций совместного распределения. Иначе говоря, выполнение этих свойств для некоторой функции \( F : \mathbb{R}^2 \to \mathbb{R} \) вовсе не гарантирует, что эта функция является функцией распределения некоторого случайного вектора.

Упражнение 12. Доказать, что функция
\[
F(x_1, x_2) = \begin{cases}
0, & x_1 \leq 0 \text{ или } x_2 \leq 0 \text{ или } x_1 + x_2 \leq 1; \\
1, & \text{иначе, то есть когда одновременно } x_1 > 0, x_2 > 0, x_1 + x_2 > 1.
\end{cases}
\]
а) удовлетворяет всем свойствам (F0)-(F3);
б) не является функцией распределения никакого вектора \( (\xi_1, \xi_2) \) хотя бы потому, что, найдись такой вектор, найдется и прямоугольник \([a_1, b_1] \times [a_2, b_2]\), вероятность попасть в который (вычисленная с помощью этой «функции распределения») отрицательна: \( \mathbf{P}(a_1 \leq \xi_1 < b_1, a_2 \leq \xi_2 < b_2) < 0! \)
