По замечанию **14**, для доказательства \( \lim_{x \to -\infty} F_\xi(x) = 0 \) достаточно доказать, что \( F_\xi(-n) \to 0 \) при \( n \to \infty \).

Представим событие \( \{ \xi < -17 \} \) (например) как счетное объединение событий:

\[
\{ \xi < -17 \} = \ldots \cup \{ -20 \leq \xi < -19 \} \cup \{ -19 \leq \xi < -18 \} \cup \{ -18 \leq \xi < -17 \} = \bigcup_{i=17}^{\infty} \{ -i-1 \leq \xi < -i \}.
\]

Используя \( \sigma \)-аддитивность вероятности, и помня, что \( \mathbf{P}\{ \xi < -17 \} \leq 1 \), получим:

\[
\mathbf{P}\{ \xi < -17 \} = \sum_{i=17}^{\infty} \mathbf{P}\{ -i-1 \leq \xi < -i \} \leq 1, \quad \text{и, по замечанию 12, } \sum_{i=n}^{\infty} \mathbf{P}\{ -i-1 \leq \xi < -i \} \to 0.
\]

Но \( \sum_{i=n}^{\infty} \mathbf{P}\{ -i-1 \leq \xi < -i \} = \mathbf{P}\{ \xi < -n \} = F_\xi(-n) \), и сходимость \( F_\xi(x) \) к нулю при \( x \to -\infty \) доказана.

**Итого:** есть ряд, составленный из вероятностей, сумма которого тоже есть вероятность и, следовательно, конечна. А из того, что ряд сходится, по замечанию 12 вытекает сходимость «хвоста» ряда к нулю. Осталось посмотреть на этот хвост и убедиться, что он равен как раз той вероятности, сходимость к нулю которой нам нужно доказать. Точно так же докажем и остальные свойства.

По замечанию **14**, для доказательства \( \lim_{x \to \infty} F_\xi(x) = 1 \) достаточно доказать, что \( F_\xi(n) \to 1 \) при \( n \to \infty \), или что \( 1 - F_\xi(n) = \mathbf{P}(\xi \geq n) \to 0 \).

Представим событие \( \{ \xi \geq 11 \} \) (например :-) как счетное объединение событий:

\[
\{ \xi \geq 11 \} = \{ 11 \leq \xi < 12 \} \cup \{ 12 \leq \xi < 13 \} \cup \{ 13 \leq \xi < 14 \} \cup \ldots = \bigcup_{i=11}^{\infty} \{ i \leq \xi < i+1 \}.
\]

В силу \( \sigma \)-аддитивности вероятности,

\[
\mathbf{P}\{ \xi \geq 11 \} = \sum_{i=11}^{\infty} \mathbf{P}\{ i \leq \xi < i+1 \} \leq 1, \quad \text{и, по замечанию 12, } \sum_{i=n}^{\infty} \mathbf{P}\{ i \leq \xi < i+1 \} \to 0.
\]

Но \( \sum_{i=n}^{\infty} \mathbf{P}\{ i \leq \xi < i+1 \} = \mathbf{P}\{ \xi \geq n \} = 1 - F_\xi(n) \), и сходимость \( F_\xi(x) \) к единице при \( x \to \infty \) доказана. □
