Обзор основных математических понятий

5(x - 4) = 40,
раскройте скобки в левой части:
    5x - 20 = 40.
Затем выделите x, прибавив 20 к обеим частям:
    5x = 60.
Затем поделите обе части на 5:
    x = 12.
Для проверки решения нужно подставить 12 в исходное уравнение:
    5(12 - 4) = 5(8) = 40, что является верным.

Для более сложных задач нужно группировать подобные члены следующим образом:

2(3x + 1) = 5(x + 2)
6x + 2 = 5x + 10        Раскрываем скобки с обеих сторон.
x + 2 = 10              Вычитаем 5x из обеих сторон.
x = 8                   Вычитаем 2 из обеих сторон.
2(24 + 1) = 5(8 + 2) = 50   Проверка: подставляем 8 вместо x в исходное уравнение.

Логарифмы полезны для решения уравнений, в которые входят возведенные в степень значения; вы можете прологарифмировать обе стороны, а затем использовать свойства логарифмов для нахождения неизвестного. Например, используя основание 10:

5^x = 3
log 5^x = log 3         Логарифмируем обе стороны.
x log 5 = log 3         Используем правило для логарифмов степенных выражений.
x = log 3 / log 5 = 0.683   Делим обе части на log 5.
5^{0.683} = 3           Проверка: подставляем 0.683 вместо x в исходное уравнение.

Системы уравнений

Система уравнений, также называемая системой совместных уравнений, — это набор алгебраических уравнений с общими переменными. Решение системы уравнений означает нахождение общего решения, значений переменных, которые будут верны для всех уравнений в системе. Если существует общее решение (это справедливо для всех представленных здесь случаев), система называется непротиворечивой; если нет, система называется противоречивой. Системы уравнений можно решать графически (прочертив соответствующие уравнениям линии; решение — это точка пересечения) или алгебраически. Ниже мы продемонстрируем второй метод.
