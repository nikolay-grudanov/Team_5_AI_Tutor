Четыре ячейки с данными часто обозначают следующим образом:

+   -
+   a   b
-   c   d

Ячейки \(a\) и \(d\) обозначают согласия (в \(a\) — случаи, когда оба теста дали положительный результат, то есть наличие заболевания, а в \(d\) — случаи, когда оба теста дали отрицательный результат), тогда как \(b\) и \(c\) обозначают несогласия.

Формула для каппы выглядит следующим образом:

\[
K = \frac{P_o - P_e}{1 - P_e}
\]

где \(P_o =\) наблюдаемые согласия, а \(P_e =\) наблюдаемые несогласия.

\[
P_o = (a + d)/(a + b + c + d),
\]

то есть число случаев согласия, поделенное на общее число наблюдений. В данном случае

\[
P_o = 80/100 = 0,80;
\]
\[
P_e = [(a + c)(a + b)]/(a + b + c + d)^2 + [(b + d)(c + d)]/(a + b + c + d)^2,
\]

а это число случаев согласия, ожидаемых случайно. Ожидаемое согласие в данном случае составляет следующее число:

\[(60*60)/(100*100) + (40*40)/(100*100) = 0,36 + 0,16 = 0,52.\]

В данном случае каппа рассчитывается таким образом:

\[
K = \frac{0.80 - 0.52}{1 - 0.52} = 0.58.
\]

Каппа может принимать значения от \(-1\) до \(+1\); значение \(0\) она примет, если число наблюдаемых совпадений равно числу ожидаемых случайных, а \(1\) — если все наблюдения согласованы.
