Решение
Для расчета вероятности подставьте значения для женского пола и GPA в уравнение логистической регрессии и затем пересчитайте результат по следующей формуле, чтобы получить вероятность быть исключенным:

\[
\text{Предсказанная вероятность} = \frac{e^{\text{уравнение логистической регрессии}}}{(1 + e^{\text{уравнение логистической регрессии}})}.
\]

Предсказанный логит равен:
\[
\text{Логит}(p) = 4,983 + 1,876(0) - 2,014(3.0) = -1,059.
\]
Предсказанная вероятность быть исключенным равна:
\[
\text{Предсказанная вероятность} = e^{1.059}/(1 + e^{1.059}) = 0,258 = 25,8\%.
\]
