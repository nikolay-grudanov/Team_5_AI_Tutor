Увеличение заболеваемости при появлении воздействия составляет 0,02, или 20 человек на 1000. С воздействием связано 40% заболеваний, и нужно будет устранить воздействие на 50 человек, чтобы стало одним случаем заболевания меньше.

Задача
Вычислите размер выборки, достаточный для оценки процентного соотношения с 95%-ным доверительным интервалом шириной в 10%, если предполагаемое значение равно 0,7.

Решение
Используйте формулу для нахождения размера выборки для процентных соотношений и подставьте в нее нужные числа:

\[
Z_{1-\alpha/2} = 1,96; \quad \omega = 0,10; \quad \pi = 0,70.
\]

Вычисления (рис. 15.49) показали, что выборка должен состоять из 81 объекта.

\[
n = \frac{\left(Z_{1-\alpha/2}\right)^2}{\omega} [\pi(1-\pi)] = \left(\frac{1.96}{0.10}\right)^2 [0.70(0.30)] = 80.7
\]

Задача
Вычислите размер выборки, необходимый для теста на различие средних, при помощи одностороннего теста Стьюдента для независимых выборок с мощностью 90% и величиной эффекта 0,4.

Решение
Подставьте в нужную формулу указанные числа:

\[
Z_\alpha = 1,645; \quad Z_{1-\beta} = 1,28; \quad \delta = 0,4
\]

Вычисления (рис. 15.50) показали, что в каждую группу должно войти 107 объектов.

\[
n = \frac{2\left(Z_\alpha + Z_{1-\beta}\right)^2}{\delta^2} = \frac{2(1.645 + 1.28)^2}{0.16} = 106.9
\]
