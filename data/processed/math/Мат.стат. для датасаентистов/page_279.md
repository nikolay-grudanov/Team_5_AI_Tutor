Рис. 7.7. Дендрограмма акций

Агломеративный алгоритм

Главным алгоритмом иерархической кластеризации является агломеративный алгоритм, который итеративно объединяет схожие кластеры. Агломеративный алгоритм начинает свою работу с того, что делает каждую запись ее собственным одноэлементным кластером, затем достраивает кластеры все больших и больших размеров. Первый шаг состоит в вычислении расстояния между всеми парами записей.

Для каждой пары записей \((x_1, x_2, ..., x_p)\) и \((y_1, y_2, ..., y_p)\) мы измеряем расстояние между двумя записями, \(d_{x,y}\), используя метрический показатель расстояния (см. разд. "Метрические показатели расстояния" главы 6). Например, мы можем использовать евклидово расстояние:

\[
d(x, y) = \sqrt{(x_1 - y_1)^2 + (x_2 - y_2)^2 + ... + (x_p - y_p)^2}.
\]

Теперь мы переходим к межкластерному расстоянию. Рассмотрим два кластера \(A\) и \(B\), каждый с характерным набором записей, \(A = (a_1, a_2, ..., a_m)\) и \(B = (b_1, b_2, ..., b_m)\). Мы можем измерить различие между кластерами \(D(A, B)\) при помощи расстояний между членами \(A\) и членами \(B\).

Одной из мер различия является метод полной связи, т. е. максимальное расстояние по всем парам записей между \(A\) и \(B\):

\[
D(A, B) = \max d(a_i, b_j) \text{ для всех пар } i, j.
\]
