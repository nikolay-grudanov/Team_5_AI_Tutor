7. Примеры

\[
T_{кр} = -t_{n-1; \alpha} = -t_{40; 0,05}.
\]

В Excel: \(-t_{40; 0,05} = -\) СТЫЮДЕНТ.ОБР(0,95; 40) = -1,68

В MatCalc: \(-t_{40; 0,05} =\) tlaw(40).q(0.95) = -1,68

На языке R: \(-t_{40; 0,05} = -qt(0.05, 40, lower.tail=FALSE) = -1,68\)
На языке Python: student = sts.t(40)
    -student.isf(0.05) = -1,68

Так как \(T_{набл} = T_{кр}\), то гипотеза \(H_0\) принимается; то есть оснований для опровержения заявления нет.

б) Ищем P-значение критерия.
(P-значение критерия: минимальная граница значений уровня значимости \(\alpha\), при которых нулевая гипотеза отклоняется).

\[
\begin{array}{c}
\alpha < \text{P-value} \\
H_0 \text{ принимается}
\end{array}
\]
\[
\begin{array}{c}
\alpha > \text{P-value} \\
H_0 \text{ отклоняется}
\end{array}
\]
0   P-value   1

Для левосторонней критической области P-значение вычисляется по формуле: \(P - value = P_{H_0}(T < T_{набл})\).

![График распределения с выделенной областью P-value](https://i.imgur.com/3Q5z5QG.png)

При условии справедливости гипотезы \(H_0\), статистика \(T\) имеет распределение Стьюдента с \(n - 1 = 40\) степенями свободы.
Искомую вероятность можно получить
в Excel: СТЫЮД.РАСП(T_{набл}; n-1; 1) = СТЫЮД.РАСП(-1,32; 40; 1) = 0,097;
в MatCalc: tlaw(40).pl(-1.32)
на языке R: pt(-1.32, 40) = 0.097
