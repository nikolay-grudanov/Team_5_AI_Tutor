Так как поиск контрпримера для E1, ..., ..., Ek → F1, ..., Ft проводится по всем конкретным путям секвенциального дерева, то мы отмечаем на каждом шаге, какие из термов u0, ..., up, a0, a1, a2, ... уже были активированы. Для этого мы можем употреблять в качестве перегородки между переменными вертикальную черту. Первоначально мы проводим перегородку справа от up:

u0, ..., up | a0, a1, a2, ...

Для указания того, что u0, ..., up активированы в исходной ситуации, если список u0, ..., up непуст. Если же список u0, ..., ..., up пуст, то первоначально проводим перегородку так:

a0 | a1, a2, ...

Для указания того, что переменная a0 в исходной ситуации активирована. На каждом шаге применения правил →∀ или →∃ мы используем в качестве b новую переменную. Предположим, что u0, ..., up, a0, ..., ai-1, или, короче, t0, ..., tq (q = p + i - 1), уже активированы, так что список термов имеет вид

u0, ..., up, a0, ..., ai-1 | ai, ai+1, ai+2.

Теперь в качестве b для правил →∀ или →∃ мы используем ai и одновременно передвинем перегородку для указания того, что новая переменная добавлена к списку уже активированных. Шаги по любому конкретному пути дерева разобьем на циклы.

Предположим, что в начале некоторого цикла, пусть это будет цикл d, мы рассматриваем секвенцию Δ → Λ, или, точнее, A1, ..., Am → B1, ..., Bn. (При d = 0 рассматриваем E1, ..., ..., Ek → F1, ..., Ft; при d > 0 рассматриваем секвенцию, полученную в конце цикла d - 1.)

Если формула, выбранная из списка A1, ..., Am, B1, ..., Bn, является атомом, шаг не выполняется.

В случае когда вхождение формулы имеет один из типов →∃, ..., ~→, выполняется один шаг, полностью определяемый формулой и соответствующим правилом.

В случае когда это вхождение имеет тип →∀, →∃, выполняется один шаг цикла по соответствующему правилу с ai, первой из еще не активированных переменных, в качестве b (как объяснено выше).

В случае когда вхождение имеет тип ∀→, →∃, шаги цикла выполняются по соответствующим правилам с использованием в качестве r каждого из уже активированных термов t0, ..., tq, которые не использовались ранее в качестве r для применяемого правила с той же главной формулой (более точно, с главной формулой, являющейся образом-потомком рассматриваемого вхождения). Таким образом, выполняются шаги цикла с 0 по q + 1.

Конкретный путь оборвется и будет закрыт, когда мы впервые получим секвенцию вида (×) с атомарной формулой C, не обращая, как и раньше, внимания на порядок формул в антecedente и сукцеденте.

Путь оборвется, не будучи закрыт, в том случае, когда получена секвенция, исходя из которой невозможно сделать ни один из описанных шагов. Это происходит, если данная секвенция (при ранее активированных термах t0, ..., tq) содержит только атомы и образы-потомки ее ∀→ и →∃-формул уже использовались как главные формулы с каждым из термов t0, ..., tq в качестве r. (Последнее может случиться только в конце такого цикла, для которого следующего цикла не существует.)
