434 гл. VI. ИСЧИСЛЕНИЕ ПРЕДИКАТОВ (ДОПОЛНИТЕЛЬНЫЕ РАЗДЕЛЫ).

поймет, какие нужны изменения, когда один или оба списка конечны или даже пусты. Мы предполагаем, что q и все указанные параметры различны. В рамках настоящего рассмотрения каждый параметр из обоих списков (которые вместе с q должны включать все параметры, входящие в замкнутые аксиомы E_0, E_1, E_2, ...) будет (пропозициональным или) предикатным или индивидным символом, q будет n-местным (n \geq 0) предикатным символом Q, а логика будет исчислением предикатов без равенства.

Метод Падоа эквивалентен (в силу контрапозиции и т. д.) следующей импликации: {Q определим через p_0, p_1, p_2, ... в теории, основанной на аксиомах E_0, E_1, E_2, ...} \rightarrow {Q интерпретируется одинаково при любых двух интерпретациях параметров Q, p_0, p_1, p_2, ..., r_0, r_1, r_2, ..., при которых E_0, E_1, E_2, ... истинны, а p_0, p_1, p_2, ... интерпретируются одинаково} или, короче, Dfb_E \rightarrow Dfd_1.

Сначала разберем Dfb_E. Мы понимаем его как утверждение о возможности дать подходящее определяющее выражение для Q(x_1, ..., x_n) (определяемого). Такое выражение будет записано в языке рассматриваемой теории, т. е. исчисления предикатов с указанными параметрами. Оно будет содержать свободно лишь переменные x_1, ..., x_n, а в качестве параметров — только члены списка p_0, p_1, p_2, ..., причём их оно может содержать лишь конечное число, скажем p_0, ..., p_s. Обозначим это определяющее выражение через R(x_1, ..., x_n, p_0, ..., p_s). Так как логикой является исчисление предикатов, мы должны иметь s \geq 0, причем среди p_0, ..., p_s должен быть хотя бы один предикатный параметр, так как определяющее выражение нельзя построить без использования хотя бы одного предикатного параметра. Наконец, мы должны выразить утверждение о том, что R(x_1, ..., x_n, p_0, ..., p_s) — определяющее выражение для Q(x_1, ..., x_n) в теории, основанной на E_0, E_1, E_2, ... . Это значит, что для любой области D и любого распределения значений, выполняющего E_0, E_1, E_2, ..., формула R(x_1, ..., x_n, p_0, ..., p_s) должна иметь то же истинностное значение, что и Q(x_1, ..., x_n), для всех значений x_1, ..., x_n в D. Это приводит нас к следующей формулировке Dfb_E^1).

Dfb_E^M: Для некоторой формулы R(x_1, ..., x_n, p_0, ..., p_s), содержащей лишь параметры, указанные явно,
\[
E_0, E_1, E_2, ... \models Q(x_1, ..., x_n) \sim R(x_1, ..., x_n, p_0, ..., p_s).
\]

Мы говорим в этом случае, что (в смысле теории моделей) Q явно определимо через p_0, p_1, p_2 ... в теории, основанной на замкнутых аксиомах E_0, E_1, E_2, ..., или что E_0, E_1, E_2, ... делают Q определимым. Далее, для такого R(x_1, ..., x_n, p_0, ..., p_s) мы говорим, что (в смысле теории моделей) E_0, E_1, E_2, ... явно определяют Q через p_0, p_1, ..., p_s как R(x_1, ..., x_n, p_0, ..., p_s).

Мы можем, эквивалентным образом, заменить «E_0, E_1, E_2, ... \models» на «для некоторого d E_0, E_1, ..., E_d \models» (ввиду полноты и непротиворечивости исчисления предикатов) и далее на «для некоторого d E_0 & ... & E_d \models» (по \models-введ. и т. д.). В E_0 & ... & E_d может входить лишь конечное число параметров. Мы можем, если нужно, увеличить s в R(x_1, ..., x_n, p_0, ..., p_s) и добиться того, чтобы в E_0 & ... & E_d входили только p_0, ..., p_s, r_0, ..., r_t. После того, как это проделано, Dfb_E^M эквивалентно следующему.

Dfb_E^P: Для некоторой конъюнкции E(Q, p_0, ..., p_s, r_0, ..., r_t) формул E_0, E_1, E_2, ... и для некоторой формулы R(x_1, ..., x_n, p_0, ..., p_s), содержащих лишь указанные явно параметры,
\[
\models E(Q, p_0, ..., p_s, r_0, ..., r_t) \models [Q(x_1, ..., x_n) \sim R(x_1, ..., x_n, p_0, ..., p_s)].
\]

Мы говорим тогда, что (в смысле теории доказательств) Q явно определимо через \{p_0, p_1, p_2, ... \} в теории, основанной на \{E_0, E_1, E_2, ... \{E(Q, p_0, ..., p_s, r_0, ..., r_t)\}, или что \{E_0, E_1, E_2, ... \{E(Q, p_0, ..., p_s, r_0, ..., r_t)\} делает Q определимым таким образом. Далее (в смысле теории доказательств), \{E_0, E_1, E_2, ... явно определяют \{E(Q, p_0, ..., p_s, r_0, ..., r_t) явно определяет\} Q через p_0, ..., p_s как R(x_1, ..., x_n, p_0, ..., p_s). Вариант, соответствующий нижним строкам, может использоваться для любой формулы E(Q, p_0, ..., p_s, r_0, ..., r_t), содержащей лишь параметры, указанные явно, независимо от того, является ли она конечной конъюнкцией формул E_0, E_1, E_2, ...^1).

Теперь разберем Dfd_1. Пусть Q', r_0', r_1', r_2', ... — новые параметры соответствующих типов (предикатные или индивидные) и с тем же числом аргументных мест, что и Q, r_0, r_1, r_2, ...

1) Аналогичным образом мы могли бы ввести E(Q, p_0, ..., p_s, r_0, ..., r_t) в теоретико-модельную формулировку Dfb_E^M (а также в Dfd_1^M, см. ниже). Если для начала имеется лишь конечное множество аксиом, то E(Q, p_0, ..., p_s, r_0, ..., r_t) может быть просто их конъюнкцией.

1) Так как E_0, E_1, E_2, ... замкнуты, то «E_0, E_1, E_2, ... \models Q(x_1, ..., x_n) \sim R(x_1, ..., x_n, p_0, ..., p_s)» эквивалентно «E_0, E_1, E_2, ... \models \forall x_1 ... \forall x_n [Q(x_1, ..., x_n) \sim R(x_1, ..., x_n, p_0; ..., p_s)]» и аналогично с «\models» вместо «\models».
