(1) \( \forall x (M(x) \supset P(x)), \forall x (S(x) \supset M(x)) \therefore \forall x (S(x) \supset P(x)) \),
иначе говоря,
(2) \( \forall x (M(x) \supset P(x)), \forall x (S(x) \supset M(x)) \models \forall x (S(x) \supset P(x)) \),

(3a) \( M(x) \supset P(x), S(x) \supset M(x) \vdash S(x) \supset P(x) \),

(3b) \( \forall x (M(x) \supset P(x)), \forall x (S(x) \supset M(x)) \vdash \forall x (S(x) \supset P(x)) \)

Пример 24. «Отношение x < y транзитивно и антитефлексивно. Значит, оно несимметрично». Символически:
(1)
\[
\forall x \forall y \forall z (x < y \& y < z \supset x < z), \\
\forall x \neg x < x \therefore \forall x \forall y (x < y \supset \neg y < x),
\]
что можно переписать в эквивалентной формулировке со свободными переменными, как поступают в учебниках математики, но все же с указанием (с помощью верхних индексов) того, что переменные x, y, z имеют интерпретацию всеобщности:
(1') \( x < y \& y < z \supset x < z, \neg x < x \therefore xyzx < y \supset \neg y < x \).
Для того чтобы установить (1) и (1'), достаточно (в силу теоремы 12_рд и начала § 13) установить, что
(3') \( x < y \& y < z \supset x < z, \neg x < x \vdash xyzx < y \supset \neg y < x \)
(где «x < y & y < z \supset x < z, \neg x < x \vdash xyz» означает «\( \forall x \forall y \forall z (x < y \& y < z \supset x < z), \forall x \neg x < x \vdash »). Поступим по схеме (B_1) § 13, 25. Пусть \( \forall x \forall y \forall z (x < y \& y < z \supset x < z), \forall x \neg x < x \). По \( \forall \)-удал. \( x < y \& y < x \supset x < x \) и \( \neg x < x \), откуда в силу исчисления высказываний \( x < y \supset \neg y < x \). Здесь мы пользуемся удобным и кратким обозначением «<».
