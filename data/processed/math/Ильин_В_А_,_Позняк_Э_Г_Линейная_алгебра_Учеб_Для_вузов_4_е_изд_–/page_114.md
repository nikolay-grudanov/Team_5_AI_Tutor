Пусть

\[
\dim (\ker B) = q.
\] (5.9)

Согласно теореме 5.3 \(\dim (\ker AB) \geq q\). Поэтому справедливо соотношение

\[
\dim (\ker AB) = p + q, \quad \text{где } p \geq 0. \tag{5.10}
\]

Так как \(\ker B \subseteq \ker AB\), то в подпространстве \(\ker AB\) можно выбрать базис \(x_1, x_2, \ldots, x_{p+q}\) так, что элементы \(x_{p+1}, \ldots, x_{p+q}\) образуют базис в \(\ker B\). При таком выборе \(x_1, x_2, \ldots, x_{p+q}\) элементы \(Bx_1, Bx_2, \ldots, Bx_p\) линейно независимы (если линейная комбинация
\[
\sum_{k=1}^p \lambda_k Bx_k = 0, \text{ то } B \left( \sum_{k=1}^p \lambda_k x_k \right) = 0, \text{ т. е. } \sum_{k=1}^p \lambda_k x_k \in \ker B,
\]
может быть, в силу выбора \(x_1, x_2, \ldots, x_p\), лишь при \(\lambda_k = 0, k = 1, 2, \ldots, p\)). Поэтому элементы \(Bx_1, Bx_2, \ldots, Bx_p\) принадлежат \(\ker A\), т. е. \(p \leq \dim (\ker A)\). Из этого неравенства и соотношений (5.9) и (5.10) вытекает требуемое неравенство (5.8). Теорема доказана.

§ 2. Матричная запись линейных операторов

1. Матрицы линейных операторов в заданном базисе линейного пространства \(V\). Фиксируем в линейном пространстве \(V\) базис \(e_1, e_2, \ldots, e_n\). Пусть \(x\) — произвольный элемент \(V\) и

\[
x = \sum_{k=1}^n x^k e_k \tag{5.11}
\]
разложение \(x\) по данному базису.

Пусть \(A\) — линейный оператор из \(L(V, V)\). Тогда из (5.11) получаем

\[
Ax = \sum_{k=1}^n x^k Ae_k. \tag{5.12}
\]
Полагая

\[
Ae_k = \sum_{j=1}^n a_k^j e_j, \tag{5.13}
\]
перепишем (5.12) в следующей форме:

\[
Ax = \sum_{k=1}^n x^k \sum_{j=1}^n a_k^j e_j = \sum_{j=1}^n \left( \sum_{k=1}^n a_k^j x^k \right) e_j.
\]

Таким образом, если \(y = Ax\) и элемент \(y\) имеет координаты \(y^1, y^2, \ldots, y^n\), то

\[
y^j = \sum_{k=1}^n a_k^j x^k, \quad j = 1, 2, \ldots, n. \tag{5.14}
\]

Рассмотрим квадратную матрицу \(A\) с элементами \(a_k^j\):

\[
A = (a_k^j).
\]

Эта матрица называется *матрицей линейного оператора* в заданном базисе \(e_1, e_2, \ldots, e_n\).

Наряду с ранее указанным способом записи линейного оператора используется при заданном базисе \(e_1, e_2, \ldots, e_n\) матричная форма записи: \(y = Ax\), причем, если \(x = (x^1, x^2, \ldots, x^n)\), то \(y = (y^1, y^2, \ldots, y^n)\), где \(y^j, j = 1, 2, \ldots, n\), определяются с помощью соотношений (5.14), а элементы \(a_k^j\) матрицы \(A\) вычисляются по формулам (5.13).

Замечание 1. Если оператор \(A\) нулевой, то все элементы матрицы \(A\) этого оператора равны нулю в любом базисе, т. е. \(A\) — нулевая матрица.

Замечание 2. Если оператор \(A\) единичный, т. е. \(A = I\), то матрица этого оператора будет единичной в любом базисе. Иными словами в этом случае \(A = E\), где \(E\) — единичная матрица. В дальнейшем единичную матрицу мы будем обозначать также символом \(I\).

Мы выяснили, что каждому линейному оператору \(A\) из \(L(V, V)\) при заданном базисе линейного пространства \(V\) отвечает матрица \(A\) этого оператора. Естественно возникает обратный вопрос — каждой ли данной матрице \(A\) при заданном базисе в \(V\) можно поставить в соответствие линейный оператор \(A\), матрицей которого будет данная матрица. Важно также выяснить вопрос о единственности матрицы линейного оператора в заданном базисе.
