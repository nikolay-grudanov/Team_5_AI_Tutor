* 501. Не производя умножения, доказать тождество Коши

\[
(a_1 c_1 + a_2 c_2 + \cdots + a_n c_n)(b_1 d_1 + b_2 d_2 + \cdots + b_n d_n) -
- (a_1 d_1 + a_2 d_2 + \cdots + a_n d_n)(b_1 c_1 + b_2 c_2 + \cdots + b_n c_n) =
= \sum_{1 \leq i < k \leq n} (a_i b_k - a_k b_i)(c_i d_k - c_k d_i) \quad (n > 1).
\]

* 502. Не производя умножения, доказать тождество Лагранжа

\[
\left( \sum_{i=1}^n a_i^2 \right) \left( \sum_{i=1}^n b_i^2 \right) - \left( \sum_{i=1}^n a_i b_i \right)^2 = \sum_{1 \leq i < k \leq n} (a_i b_k - a_k b_i)^2.
\]

* 503. Доказать, что для любых действительных чисел \( a_1, a_2, \ldots, a_n \) и \( b_1, b_2, \ldots, b_n \) справедливо неравенство

\[
(a_1^2 + a_2^2 + \cdots + a_n^2)(b_1^2 + b_2^2 + \cdots + b_n^2) \geq (a_1 b_1 + a_2 b_2 + \cdots + a_n b_n)^2,
\]

причем знак равенства имеет место тогда и только тогда, когда одна из систем чисел отличается от другой лишь числовым множителем (быть может, равным нулю). (Неравенство Коши–Буняковского.)

* 504. Доказать, что для любых комплексных чисел \( a_1, a_2, \ldots, a_n \) и \( b_1, b_2, \ldots, b_n \) имеет место равенство

\[
\left( \sum_{k=1}^n a_k \overline{a}_k \right) \left( \sum_{k=1}^n b_k \overline{b}_k \right) - \left( \sum_{k=1}^n a_k \overline{b}_k \right) \left( \sum_{k=1}^n \overline{a}_k b_k \right) =
= \sum_{1 \leq j < k \leq n} (a_j b_k - a_k b_j)(\overline{a}_j \overline{b}_k - \overline{a}_k \overline{b}_j).
\]
