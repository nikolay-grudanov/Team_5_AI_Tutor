*763. При каких условиях система линейных уравнений с вещественными коэффициентами

\[
\left\{
\begin{array}{l}
\lambda x + ay + bz + ct = 0, \\
-ax + \lambda y + hz - gt = 0, \\
-bx - hy + \lambda z + ft = 0, \\
-cx + gy - fz + \lambda t = 0
\end{array}
\right.
\]

имеет ненулевое решение?
