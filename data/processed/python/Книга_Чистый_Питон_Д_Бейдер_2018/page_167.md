bytes — неизменяемые массивы одиночных байтов

Объекты bytes представляют собой неизменяемые последовательности одиночных байтов (целых чисел в диапазоне 0 \( x \) 255)\(^1\). В концептуальном плане они подобны объектам str и их также можно представить как неизменяемые массивы байтов.

Аналогично строковому типу, тип bytes имеет свой собственный литературный синтаксис, предназначенный для создания объектов, и объекты этого типа пространственно эффективны. Объекты bytes не могут изменяться, но, в отличие от строковых объектов, для «изменяемых массивов байтов» есть специальный тип данных, который называется bytearray, или байтовый массив, в который они могут быть распакованы. Вы узнаете о нем подробнее в следующем подразделе.

```python
>>> arr = bytes((0, 1, 2, 3))
>>> arr[1]
1

# Байтовые литералы имеют свой собственный синтаксис:
>>> arr
b'x00x01x02x03'
>>> arr = b'x00x01x02x03'

# Разрешены только допустимые "байты":
>>> bytes((0, 300))
    : "bytes must be in range(0, 256)"

# Байты неизменяемы:
>>> arr[1] = 23
TypeError:
"'bytes' object does not support item assignment"

>>> del arr[1]
TypeError:
"'bytes' object doesn't support item deletion"
```

1 См. документацию Python «bytes»: https://docs.python.org/3/library/stdtypes.html#bytes
