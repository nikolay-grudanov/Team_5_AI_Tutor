Для проверки того, что две ссылки ведут на один и тот же объект, служит оператор is. Оператор is not тоже существует и позволяет проверить, что два объекта различаются.

In [40]: a = [1, 2, 3]
In [41]: b = a
In [42]: c = list(a)
In [43]: a is b
Out[43]: True
In [44]: a is not c
Out[44]: True
