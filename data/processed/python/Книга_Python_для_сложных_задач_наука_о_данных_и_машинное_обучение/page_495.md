Использование метода РСА для фильтрации шума

Метод РСА можно применять для фильтрации зашумленных данных. Основная идея состоит в следующем: шум должен довольно мало влиять на компоненты с дисперсией, значительно превышающей его уровень. Восстановление данных с помощью лишь самого крупного подмножества главных компонент должно приводить к относительному сохранению сигнала и отбрасыванию шума.
