Код программы `calc1.py` создает простое приложение калькулятора на языке Python с использованием библиотеки Tkinter. В коде приведены комментарии, объясняющие основные моменты:

1. **Определение функций `frame` и `button`**:
   - Функция `frame` создает фрейм с заданными параметрами.
   - Функция `button` создает кнопку с текстом и командой.

2. **Класс `Calculator`**:
   - Класс наследуется от `Frame`.
   - Метод `__init__` инициализирует окно, устанавливает заголовок и иконку.
   - Создается переменная `display` типа `StringVar` для отображения вычислений.
   - В цикле создаются фреймы для цифровых и операционных кнопок.
   - Для каждой кнопки создается обработчик событий, который добавляет текст в `display`.

3. **Обработка операций**:
   - Для операций `+`, `-`, `*`, `/` создается обработчик событий, который вызывает метод `calc` с текущим значением `display`.

4. **Метод `calc`**:
   - Метод `calc` принимает `display` и пытается вычислить значение с помощью `eval`.
   - Если возникает ошибка `ValueError`, выводится сообщение "ERROR".

5. **Основная программа**:
   - Если модуль запускается напрямую (не импортирован), создается объект `Calculator` и запускается главный цикл событий.
