Object Storage

Object Storage — сервис для хранения данных любого типа и объема. В хранилище можно помещать медиафайлы, электронные письма, резервные копии, образы виртуальных машин и BigData. Данные хранятся в исходном формате и размещаются в виде объектов в плоской адресной структуре.

• Создание Telegram-бота без написания кода с помощью n8n на основе Container Apps или Notebooks
• Подключение ИИ из Foundation Models к no-code Telegram-боту на основе Container Apps или Notebooks
• Подключение Trino к S3
• Создание django-приложения для раздачи фотографий
• Создание AI-агента с МСР-сервером Managed RAG
