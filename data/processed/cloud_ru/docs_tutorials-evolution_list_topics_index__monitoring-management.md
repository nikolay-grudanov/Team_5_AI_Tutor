Мониторинг и управление

Мониторинг — сервис сбора и хранения метрик облачных ресурсов. С его помощью можно отслеживать ключевые показатели производительности и поддерживать стабильную работу приложений.

Клиентское логирование — сервис управления логами и алертами без входа в интерфейс подключенного продукта. Сервис читает и фиксирует логи подключенных продуктов, объединяя все сообщения в лог-группы.

Аудит-логирование — сервис, позволяющий собирать логи и просматривать историю событий, а также экспортировать журналы логов во внешнюю клиентскую систему SIEM (разработка находится на стадии Preview).

• Мониторинг виртуальной машины с помощью vmagent
• Просмотр архивированных метрик в Grafana
• Передача аудит-логов с виртуальной машины с помощью Fluent Bit
• Экспорт аудит-логов в SIEM с использованием защищенного протокола TLS
• Архивирование аудит-логов личного кабинета в бакете Object Storage
• Моментальное групповое email-уведомление о событии аудита
• Передача логов с виртуальной машины с помощью Docker-контейнера плагина Fluent Bit
• Передача логов с виртуальной машины с помощью Fluent Bit loggaas plugin
• Передача логов с виртуальной машины с помощью Fluent Bit и Lua-скрипта
• Передача логов с кластера Managed Kubernetes
