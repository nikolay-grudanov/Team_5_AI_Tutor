Managed PostgreSQL®

Managed PostgreSQL® — это сервис для развертывания и управления кластерами Managed PostgreSQL® в инфраструктуре платформы Evolution.

• Настройка взаимодействия приложения на виртуальных машинах с сервисом Managed PostgreSQL®
• Развертывание Identity Provider Keycloak на виртуальной машине и Managed PostgreSQL®
• Резервное копирование и восстановление базы данных
• Развертывание сервиса статического мониторинга кода и безопасности SonarQube
• Подключение кластера Managed PostgreSQL® к локальной IDE

Managed Redis®

Managed Redis® — полностью управляемая in-memory база данных для высокопроизводительного кеширования данных, управления очередями и работы с данными в реальном времени.

• Оптимизация производительности Web-приложения с Managed Redis®
• Использование Managed Redis® как брокера сообщений
• Подключение Managed Redis® к сервисам в кластере Managed Kubernetes
