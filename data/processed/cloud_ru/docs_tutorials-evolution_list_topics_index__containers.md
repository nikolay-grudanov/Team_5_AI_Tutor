Container Apps и Artifact Registry

Container Apps — это облачный сервис платформы Cloud.ru Evolution для создания и запуска контейнерных приложений без знаний Kubernetes и настройки виртуальных машин. Контейнеры запускаются из Docker-образов, которые хранятся в репозитории Artifact Registry.

В руководствах для развертывания контейнеров используются готовые демонстрационные образы. Вы можете использовать их или образы своих приложений.
