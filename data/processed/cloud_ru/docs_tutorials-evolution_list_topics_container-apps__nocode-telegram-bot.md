<h2>Создание Telegram-бота без написания кода с помощью n8n на основе Container Apps или Notebooks</h2>
<p>С помощью этого руководства вы развернете платформу для создания рабочих процессов без написания кода и создадите Telegram-бота, который будет повторять сообщения пользователя.</p>
<p>Вы можете развернуть платформу на основе сервиса Container Apps или Notebooks.</p>
<p>Вы будете использовать следующие сервисы:</p>
<ul>
<li><b>Container Apps</b> — сервис для запуска контейнерных приложений в облаке. Не требует знания Kubernetes и создание виртуальных машин.</li>
<li><b>Notebooks</b> — сервис для запуска сред ML и работы DS-специалистов в ноутбуках на платформе Evolution.</li>
<li><b>Object Storage</b> — объективное S3-хранилище с <b>бесплатным хранением файлов</b>, объемом до 15 ГБ. Будет использоваться в качестве постоянного хранилища для запущенного контейнера.</li>
<li><b>n8n</b> — платформа с открытым кодом для автоматизации рабочих процессов и интеграции сервисов. Подходит для экспериментов и пет-проектов.</li>
</ul>
<h3>Шаги:</h3>
<ol>
<li><b>Создайте бакет в сервисе Object Storage.</b></li>
<li><b>Создайте контейнер / ноутбук с образом n8n.</b></li>
<li><b>Зарегистрируйте бота в Telegram.</b></li>
<li><b>Запустите n8n.</b></li>
<li><b>Настройте параметры подключения к Telegram.</b></li>
<li><b>Создайте триггер Telegram в n8n.</b></li>
<li><b>Добавьте в чат статус «... печатает / ... typing».</b></li>
<li><b>Настройте ответ пользователю от бота.</b></li>
<li><b>Убедитесь, что бот работает.</b></li>
</ol>
<h3>Перед началом работы</h3>
<p>Зарегистрируйтесь в личном кабинете Cloud.ru.</p>
<p>Если вы уже зарегистрированы, войдите под своей учетной записью.</p>
<h3>1. Создайте бакет в сервисе Object Storage</h3>
<p>В сервисе Object Storage создайте новый бакет со следующими параметрами:</p>
<ul>
<li><b>Название</b> — <code>&lt;your_name&gt;-n8n</code>, например name-n8n .</li>
<li><b>Доменное имя</b> — <code>&lt;your_name&gt;-n8n</code>, например name-n8n .</li>
<li><b>Класс хранения по умолчанию</b> — Стандартный .</li>
<li><b>Максимальный размер</b> — отключите или укажите на свое усмотрение.</li>
</ul>
<p>Убедитесь, что в личном кабинете на странице сервиса Object Storage:</p>
<ul>
<li>в списке бакетов отображается созданный вами бакет;</li>
<li>класс хранения — Стандартный .</li>
</ul>
<h3>2. Создайте контейнер / ноутбук с образом n8n</h3>
<p>В личном кабинете создайте контейнер n8n из готового образа с помощью Container Apps:</p>
<p>Внимание<br>Тестовый образ n8n создан в версии n8n@1.116.2. Если вы создаете и развиваете кастомный образ, рекомендуется использовать версию n8n 1.116.2 для стабильной работы образа с Container Apps.</p>
<ol>
<li>Перейдите на страницу сервиса Container Apps, выберите Container Services и нажмите Создать.</li>
<li>Укажите Название создаваемого Container Service, например container-app-n8n-name .</li>
<li>Включите Тестовый образ и выберите Воркфлоу-приложение на n8n . На вкладке Общие параметры отображаются параметры тестового образа:
<ul>
<li>Конфигурация — 0.2 vCPU - 512 RAM .</li>
<li>Название контейнера, например n8n-container .</li>
<li>Порт контейнера — 5678 .</li>
</ul>
</li>
<li>В поле Образ выберите Воркфлоу-приложение на n8n .</li>
<li>На вкладке Тома создайте новый том со следующими параметрами:
<ul>
<li>Тип — Постоянный .</li>
<li>Название — например n8n-volume .</li>
<li>Бакет из Object Storage — выберите бакет сервиса Object Storage, который вы создали на шаге 1 .</li>
<li>Путь (path) — /synced/n8n .</li>
</ul>
</li>
<li>Нажмите Создать.</li>
<li>Убедитесь, что в личном кабинете на странице созданного Container Service:
<ul>
<li>отображается одна ревизия;</li>
<li>статус ревизии — «Создается».</li>
</ul>
</li>
<li>Дождитесь, когда статус ревизии изменится на «Выполняется».</li>
</ol>
<h3>3. Зарегистрируйте бота в Telegram</h3>
<p>Для работы вам потребуется зарегистрировать нового бота в Telegram и получить токен для него.</p>
<ol>
<li>В Telegram найдите бота BotFather.</li>
<li>Выполните команду /newbot .</li>
<li>Задайте для бота имя (name) и имя пользователя (username).</li>
<li>Имя пользователя должно оканчиваться на Bot или _bot .</li>
<li>В нашем случае:
<ul>
<li>name: new-bot</li>
<li>username: nocodelabbot</li>
</ul>
</li>
</ol>
<p>В результате вы получите токен. Сохраните его — он потребуется на следующих этапах.</p>
<p>Внимание<br>Токен является секретом. Не публикуйте его и не передавайте третьим лицам.</p>
<h3>4. Запустите n8n</h3>
<p>Со страницы созданного на шаге 2 Container Service:</p>
<ol>
<li>Нажмите на публичный URL.<br>Откроется интерфейс сервиса n8n с формой регистрации нового пользователя.</li>
<li>Заполните поля формы регистрации и нажмите Next .</li>
<li>В следующих окнах нажмите Get Started и Skip .<br>После регистрации в n8n вы будете перенаправлены в веб-интерфейс n8n.</li>
</ol>
<h3>5. Настройте параметры подключения к Telegram</h3>
<p>В личном кабинете n8n создайте и настройте учетные данные для подключения к Telegram:</p>
<ol>
<li>В правом верхнем углу личного кабинета раскройте меню Create Workflow и выберите Create Credentials .</li>
<li>В следующем окне в качестве типа создаваемой учетной записи выберите Telegram API и нажмите Continue .<br>Откроется диалоговое окно создания учетной записи.</li>
<li>В поле Access Token вставьте токен бота, полученный на шаге 3 .</li>
<li>В правом верхнем углу диалогового окна нажмите Save .</li>
<li>Дождитесь, когда вверху окна появится подтверждение об успешном тестовом подключении, и закройте диалоговое окно.</li>
</ol>
<p>Внимание<br>При ошибках в работе n8n обращайтесь к документации разработчика n8n .</p>
<h3>6. Создайте триггер Telegram в n8n</h3>
<p>В личном кабинете n8n создайте триггер для Telegram-бота:</p>
<ol>
<li>В правом верхнем углу личного кабинета нажмите Create Workflow .</li>
<li>В центре рабочей области My workflow нажмите Add first step .</li>
<li>На вкладке выбора триггеров в поле поиска введите telegram и выберите Telegram в результатах поиска.</li>
<li>В появившемся списке выберите On message .</li>
<li>В открывшемся диалоговом окне убедитесь, что в поле Credential to connect with выбрана учетная запись, созданная на шаге 5 .</li>
<li>В левой области окна отображены параметры, которые вы получили на шаге 6 для стартового триггера, после того как нажали Execute step .</li>
<li>Чтобы закрыть диалоговое окно, в левом верхнем углу интерфейса нажмите Back to canvas .<br>В центре рабочей области появится блок Telegram Trigger с новым триггером.</li>
<li>Нажмите дважды на добавленный триггер.</li>
<li>В открывшемся окне свойств триггера нажмите Execute step или Test step , в зависимости от версии n8n .</li>
<li>Нажмите Execute step .<br>Слева появится всплывающее уведомление о том, что n8n отслеживает сообщения, отправленные в Telegram-бота.</li>
</ol>
<p>10. Отправьте любое сообщение в Telegram-бота.</p>
<p>После этого на странице триггера в секции Output появятся данные отправленного сообщения.</p>
<h3>7. Добавьте в чат статус «... печатает / ... typing»</h3>
<ol>
<li>Нажмите Back to canvas .</li>
<li>Нажмите справа от стартового триггера.</li>
<li>На вкладке выбора триггеров введите в поле поиска telegram и выберите Telegram в результатах поиска.</li>
<li>В появившемся списке выберите Send a chat action .</li>
<li>В открывшемся диалоговом окне убедитесь, что:
<ul>
<li>в поле Credential to connect with выбрана учетная запись, которую вы создали на шаге 5 ;</li>
<li>в левой области окна отображены параметры, которые вы получили на шаге 6 для стартового триггера, после того как нажали Execute step .</li>
</ul>
</li>
<li>Перетащите параметр message|chat|id в поле Chat ID .</li>
<li>Нажмите Execute step .<br>В правой части окна отобразится результат выполнения true .</li>
</ol>
<h3>8. Настройте ответ пользователю от бота</h3>
<p>Настройте бота, чтобы он отправлял текст сообщения пользователю обратно:</p>
<ol>
<li>Нажмите Back to canvas .</li>
<li>Нажмите справа от созданного на предыдущем шаге действия Send a chat action .</li>
<li>В открывшейся справа вкладке вновь найдите и выберите Telegram .</li>
<li>В списке Actions выберите Send a text message .</li>
<li>В открывшемся диалоговом окне введите параметры:
<ul>
<li>Chat ID — укажите {{ $node["Telegram trigger"].json["message"]["chat"]["id"] }} ;</li>
<li>Text — укажите {{ $node["Telegram trigger"].json["message"]["text"] }} .</li>
</ul>
</li>
<li>Чтобы вернуться к рабочей области, в левом верхнем углу нажмите Back to canvas .</li>
<li>В верхней строке нажмите Save .</li>
<li>В верхней строке активируйте бот.</li>
</ol>
<p>Созданный вами Telegram-бот активирован.</p>
<h3>9. Убедитесь, что бот работает</h3>
<p>Отправьте в Telegram любое сообщение боту. Бот должен прислать обратно ваше сообщение.</p>
