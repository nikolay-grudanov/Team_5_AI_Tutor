Сворачивающийся текст

Еще один остроумный прием, использующий события, — создание сворачивающихся страниц. Суть сворачивающейся страницы такова: если у вас есть большой объем информации, приготовленной для посетителей, но вы не хотите утомлять их длинной страницей, вы скрываете (сворачиваете) фрагменты текста, пряча их за заголовками, которые гости могут щелкнуть кнопкой мыши, когда захотят увидеть свернутый текст (рис. 15.9).
