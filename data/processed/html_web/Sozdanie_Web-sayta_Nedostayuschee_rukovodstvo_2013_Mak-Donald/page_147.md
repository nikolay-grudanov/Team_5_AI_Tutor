Маркированные списки

Маркированные списки похожи на нумерованные только без последовательной нумерации пунктов. Элемент-контейнер — <ul>, в него вы вставляете пункт списка, заключенный в элемент <li>. Браузер формирует отступ для каждого пункта списка и автоматически отображает маркеры.

Самый интересный компонент маркированных списков — атрибут type, позволяющий изменять значок маркера. Вы можете использовать кружок (черный кружок, выбираемый автоматически), окружность (не закрашенный кружок) или квадратик (закрашенный квадратик). На рис. 5.11 показаны списки с разными стилями.
