Имейте в виду, что преимущество первоначального получения объекта заключается в том, что вы можете изменить несколько свойств по очереди, и при этом не требуется каждый раз искать HTML-объект с помощью метода getElementById().

Таблица 15.1. Общие свойства HTML-объектов

<table>
  <tr>
    <th>Свойство</th>
    <th>Описание</th>
  </tr>
  <tr>
    <td>className</td>
    <td>Позволяет извлечь или задать атрибут class (см. разд. "Селекторы классов" главы 6). Другими словами, это свойство определяет, какой стиль (если есть) использует этот элемент. Конечно, определить стиль вы должны во встроенной или связанной таблице стилей или все закончится уродливым стандартным форматированием</td>
  </tr>
  <tr>
    <td>innerHTML</td>
    <td>Позволяет считать или изменить HTML-код внутри элемента. Это свойство безумно полезно, но у него есть две хитрости. Во-первых, его можно применять к любому HTML-содержимому, включая текст и теги. Поэтому, если вы хотите поместить внутрь абзаца жирный текст, можно задать innerHTML для <b>Hi</b>. Специальные символы не приветствуются — их нужно заменить символьными последовательностями, описанными в разд. "Специальные символы" главы 5. Во-вторых, когда вы задаете innerHTML, в указанном элементе заменяется все содержимое, включая другие HTML-элементы. Поэтому, если вы задаете innerHTML для элемента <div>, содержащего несколько абзацев и изображений, все эти элементы заменяются новым содержимым. Если вы хотите изменить фрагмент абзаца, заключите его в элемент <span></span></td>
  </tr>
  <tr>
    <td>parentElement</td>
    <td>Предоставляет HTML-объект для элемента, содержащего данный элемент. Например, если текущий элемент в абзаце — элемент <b>, это свойство возвращает объект для элемента <p>. Когда вы получите этот объект, можно модифицировать абзац. С помощью данного приема (и аналогичных приемов в Dynamic HTML) можно переходить от одного элемента к другому</td>
  </tr>
  <tr>
    <td>style</td>
    <td>Собирает вместе все атрибуты CSS, определяющие внешний вид HTML-элемента. Строго говоря, свойство style возвращает объект с полноценным стилем, и вам нужно вставить еще одну точку (.) и имя атрибута стиля, который хотите изменить, например, myObject.style.fontSize. Вы можете применять объект стиля для задания цветов, границ, шрифтов и даже позиционирования</td>
  </tr>
  <tr>
    <td>tagName</td>
    <td>Предоставляет имя HTML-элемента для данного объекта, без угловых скобок. Например, если текущий объект представляет элемент <img>, это свойство вернет текст img</td>
  </tr>
</table>
