Рис. 17.12. Встраивание позволяет просматривать ваше видео на странице собственного производства

Внимательный читатель заметит, что это видеоокно состоит из элемента <embed>, заключенного в элемент <object>. Эта запутанная разметка отлично подходит для обеспечения совместимости браузеров, потому что браузеры, поддерживающие элемент <object>, проигнорируют внутреннее содержимое, а браузеры, не знакомые с элементом <object>, воспользуются элементом <embed>.

СОВЕТ
