![Таблица с названием "Largest Cities in the Year 100", содержащая столбцы Rank, Name и Population](../images/ch9_12.png)

Рис. 9.12. В данном примере таблица стилей вызывается для таблицы шириной 1 пиксел. Но браузер не сжимает таблицу до такой степени и виной тому контент, влияющий на минимальный размер таблицы. В этой таблице название города Anuradhapura (Анурадхапура) — самое длинное неделимое значение, поэтому браузер использует это название для определения ширины столбца. Если вы хотите постепенно уменьшить размер столбца, попробуйте сжать текст, применив меньший размер шрифта

Задание размера таблицы

Чаще всего хочется явно задать ширину таблицы и отдельных столбцов. Если сделать это, браузер учтет заданные размеры и выполнит перенос текста в столбцах для соблюдения указанной ширины.

При задании размера таблицы можно указывать ширину в пикселах или процентах. Например, в следующем правиле ширина таблицы ограничена половиной ширины текущего контейнера (которая на стандартной странице равна половине ширины страницы).

table.Cities {
    width: 50%;
}
