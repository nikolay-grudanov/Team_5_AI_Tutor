■ Шаблон — это обычная HTML-страница, используемая как основа для всех остальных страниц вашего сайта. Контент, который вставляется с помощью шаблона, становится фиксированным контентом: ваш редактор Web-страниц переносит его на каждую страницу, создаваемую вами с помощью данного шаблона. Если вы помещаете в шаблон панель меню, эту панель получат все страницы. Изменять фиксированный контент можно только в самом шаблоне; на страницах, созданных с помощью этого шаблона, отредактировать фиксированный контент не удастся.

■ Помимо фиксированного, неизменного контента шаблон страницы содержит редактируемые области, куда вставляется уникальный контент каждой страницы. Для создания такой области применяются особые HTML-комментарии. Хотя эти комментарии в Dreamweaver и Expression Web выглядят одинаково, они немного отличаются, поэтому шаблон страницы из одной программы нельзя использовать в другой.

■ После того как шаблон создан, можно формировать отдельные страницы сайта, которые его используют. Эти страницы включают в себя весь фиксированный контент из шаблона и предоставляют новый контент для всех редактируемых областей страницы. Специальные комментарии всегда остаются на своем месте. Они позволяют редактору Web-страниц обновлять страницу при изменении шаблона.
