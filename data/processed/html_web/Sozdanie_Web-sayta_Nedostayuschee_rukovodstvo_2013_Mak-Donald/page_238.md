вяжчивое изображение. Для вставки на страницу водяного знака применяется то же самое свойство background-image, о котором вы только что узнали. Но в набор придется добавить несколько дополнительных стилевых свойств (табл. 7.2). Сначала нужно применить свойство background-repeat, чтобы отказаться от формирования мозаики. В то же время имеет смысл использовать свойство background-position, чтобы выровнять вашу картинку относительно бокового края страницы или ее центра.

Таблица 7.2. Свойства фонового изображения

<table>
  <tr>
    <th>Свойство</th>
    <th>Описание</th>
    <th>Стандартные значения</th>
    <th>Может наследоваться?</th>
  </tr>
  <tr>
    <td>background-image</td>
    <td>Файл с изображением, которое вы хотите использовать как фон вашей страницы</td>
    <td>URL, указывающий на файл с изображением в виде url('mypig.jpg')</td>
    <td>Нет<sup>1</sup></td>
  </tr>
  <tr>
    <td>background-repeat</td>
    <td>Применяете вы или нет мозаичное расположение изображения для заполнения страницы, мозаику можно отключить полностью или в одном направлении (например, чтобы изображение повторялось по вертикали, но не по горизонтали)</td>
    <td>repeat, repeat-x, repeat-y, no-repeat</td>
    <td>Нет</td>
  </tr>
  <tr>
    <td>background-position</td>
    <td>Место, куда вы хотите поместить изображение. Применяйте, только если не используете мозаику</td>
    <td>top left, top center, top right, center left, center, center right, bottom left, bottom center, bottom right</td>
    <td>Нет</td>
  </tr>
  <tr>
    <td>background-attachment</td>
    <td>Если хотите зафиксировать изображение (или мозаику) на прежнем месте при прокрутке страницы посетителем</td>
    <td>scroll, fixed</td>
    <td>Нет</td>
  </tr>

1 Фоновые изображения не наследуются (см. разд. "Наследование" главы 6). Но если явно не задать для элемента цвет фона, у него будет прозрачный фон, т. е. будет виден фон содержащего изображение элемента.

Далее приведен пример, в котором изображение помещается в центр Web-страницы.

body {
  background-image: url('smiley.jpg');
  background-repeat: no-repeat;
  background-position: center;
}
