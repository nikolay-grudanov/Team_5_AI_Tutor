Таблицы стилей предлагают другой вариант: можно поместить элемент в фиксированную неменяющуюся позицию. Для этого применяют свойства позиционирования top, left, bottom, right в сочетании со свойством position (в данном случае со значением absolute). Далее приведен пример.

.FixedBox {
    position: absolute;
    top: 20px;
    left: 0px;
    width: 150px;
    background-color: orange;
    border-width: 2px;
    border-style: solid;
    border-color: black;
    padding: 10px;
    margin: 8px;
    font-weight: bold;
    color: white;
}
