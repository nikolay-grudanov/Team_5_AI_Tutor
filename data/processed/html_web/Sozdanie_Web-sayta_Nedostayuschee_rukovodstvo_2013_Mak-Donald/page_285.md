Построение макета с фиксированной шириной

Для создания макета с фиксированной шириной нужно просто определить ширину каждого элемента <div> в пикселах. Например, можно сделать левую и правую полосы шириной 100 пикселов, а среднюю полосу шириной 450 пикселов. В сумме (с учетом небольших полей между ними) это будет равно примерно 700 пикселам.

СОВЕТ
Если вы остановились на макете с фиксированной шириной, придерживайтесь 1 000-пиксельной нормы и в пределах этой ширины удобно размещайте контент, чтобы не возникло необходимости в горизонтальной прокрутке страницы. Статья из New York Times (см. рис. 9.1) может служить хорошим примером. Пример ширины в 700 пикселов из текущего раздела соответствует этим рекомендациям, но немного маловат.

Теперь, когда пространство распределено, нужно просто создать три стилевых правила, по одному для каждой колонки. Как и в приведенных ранее примерах, вы управляете местоположением колонки с помощью задания ее левой координаты и позволяете контенту определить ее общую высоту.

Следующее правило определяет полосу шириной 100 пикселов, расположенную у левого края страницы.

.LeftPanel {
    position: absolute;
    left: 0px;
    width: 100px;
}

Средняя полоса начинается с отметки 120 пикселов и заполняет следующие 450 пикселов. Таким образом, между правым краем левой полосы и левым краем средней полосы остается пустое пространство шириной 20 пикселов. Кроме того, средняя полоса отделяется от боковых с помощью свойств, задающих границы (см. табл. 6.5).

.MiddlePanel {
    position: absolute;
    left: 120px;
    width: 450px;
    border-left-width: 1px;
    border-right-width: 1px;
    border-top-width: 0px;
    border-bottom-width: 0px;
    border-style: solid;
    border-color: blue;
    padding-top: 0px;
}
