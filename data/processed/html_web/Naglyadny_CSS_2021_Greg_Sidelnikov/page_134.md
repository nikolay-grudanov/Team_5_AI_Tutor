19.7. Свойство flex-basis

Свойство flex-base работает аналогично другому свойству CSS: min-width — и увеличит размер элемента в зависимости от размера внутреннего содержимого. В противном случае по умолчанию будет использоваться базовое значение.

flex-basis: auto

a CSS Is Awesome b c

flex-basis: 50px

a CSS Is Awesome b c
