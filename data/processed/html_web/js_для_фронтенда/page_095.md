Тем времени, где-то на сервере, возможно, есть следующий код, занимающийся обработкой события LOG:

// «Прослушиваем» событие LOG, как только оно возникнет,
// будет вызвана функция logIt
eventHub.listen('LOG', logIt);
// Функция-обработчик события
function logIt(what) {
    // делаем что-то интересное
    console.log(what.severity + ': ' + what.message);
}

Концентратор событий соединяет эти два несвязанных фрагмента кода. Сообщения передаются между клиентами концентратора события независимо от их расположения. Рис. 3.2 иллюстрирует отношения между концентратором и всеми его подключенными модулями.
