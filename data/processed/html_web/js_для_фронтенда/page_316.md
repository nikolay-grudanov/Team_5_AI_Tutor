Рис. 8.24. Полностью настроенная домашняя страница проекта Jenkins

Работаем без интерфейса пользователя

Вы можете практически полностью управлять Jenkins через командную строку или удаленный API. При этом существует два метода управления Jenkins из командной строки: через SSH или через cli.jar. Jenkins запускает свой собственный SSHD-сервер, по умолчанию работающий на случайном порту.

Чтобы определить номер порта, сделайте запрос HEAD к вашему базовому URL Jenkins и найдите ключ заголовка x-SSHEndpoint. Значение этого ключа и есть тот самый случайный номер порта, который прослушивает SSHD-сервер Jenkins. В качестве альтернативы, вы можете настроить Jenkins на использование фиксированного порта в разделе Configure System.

Мой случайный номер SSH-порта — 64812, поэтому команда подключения к SSHD-серверу будет такой:

% ssh -p 64812 localhost build MyJavaScriptProject -s
